Class vs ID - asosiy farqlar

Xususiyat Class (.) ID (#)
Sintaksis .classname #idname
Qayta ishlatish Bir sahifada ko'p marta Bir sahifada faqat BIR marta
Specificity 10 ball 100 ball (juda yuqori)
CSS da ishlatish Tavsiya etiladi ✅ Tavsiya etilmaydi ❌
Asosiy ishlatilishi Stillash uchun JavaScript va anchor havolalar

Qachon Class, qachon ID ishlatish kerak?

Class ishlatiladigan holatlar

  • CSS da stillash uchun (har doim)
  • Bir xil ko'rinishdagi ko'p elementlar uchun
  • Qayta ishlatiladigan komponentlar uchun
  • Modulli CSS yozish uchun

ID ishlatiladigan holatlar

  • JavaScript da element tanlash uchun
  • Sahifa ichidagi havolalar (anchor) uchun
  • Forma elementlari uchun (label for="")
  • CSS da - ILOJI BO'LSA ISHLATMANG!

!important nima?

!important - bu maxsus CSS kalit so'zi bo'lib, u barcha specificity qoidalarini bekor qiladi va o'z qoidasini eng kuchli qiladi.

CSS 
p { color: blue; }
p { color: red !important; } /* Bu g'alaba qiladi! */
!important specificity zanjirini buzadi! Agar siz !important ishlatsangiz, keyinchalik bu stilni o'zgartirish uchun yana !important ishlatishingiz kerak bo'ladi.

!important qachon ishlatiladi?

1
Uchinchi tomon kutubxonalarini bekor qilishda

Bootstrap, Tailwind kabi kutubxonalarning stillarini o'zgartirish kerak bo'lganda.

2
Foydalanuvchi stillarini bekor qilishda

Brauzer yoki foydalanuvchi tomonidan o'rnatilgan stillarni o'zgartirishda.

3
Utility klasslar uchun

.hidden { display: none !important; } - bu har doim ishlashi kerak bo'lgan klasslar.

Amaliy mashq

Vazifa: Class va ID farqini tushunganingizni tekshiring.

  1. CSS da stillash uchun faqat class lardan foydalaning
  2. JavaScript da ishlatish uchun bitta ID yarating
  3. .hidden { display: none !important; } utility klassini yarating

Namunaviy yechim:

<div class="card">
    <h2 class="card-title">Sarlavha</h2>
    <p class="card-text">Karta ichidagi matn</p>
</div>

<div id="message-box" class="hidden">Yashirin xabar</div>

<style>
.card {
    background: white;
    padding: 20px;
    border-radius: 12px;
}

.hidden {
    display: none !important;
}
</style>

Dars xulosasi

  • Class - CSS stillash uchun, qayta ishlatiladi (10 ball specificity)
  • ID - JavaScript uchun, bir marta ishlatiladi (100 ball specificity)
  • !important - barcha specificity ni bekor qiladi, faqat zarur holatlarda ishlating
