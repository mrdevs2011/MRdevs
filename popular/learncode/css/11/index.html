Mavjud loyihani tahlil qilish

Dars davomida biz oldindan tayyorlangan HTML strukturaga ega bo'lgan "Natour" loyihasini CSS yordamida jonlantiramiz. Birinchi navbatda HTML struktura va mavjud klasslarni tahlil qilamiz.

HTML strukturasi (qisqartirilgan)
<header class="header">
    <div class="header__logo-box">...</div>
    <div class="header__text-box">
        <h1 class="heading-primary">...</h1>
        <a href="#" class="btn btn--white btn--animated">...</a>
    </div>
</header>

<main>
    <section class="section-about">...</section>
    <section class="section-features">...</section>
    <section class="section-tours">...</section>
    <section class="section-stories">...</section>
    <section class="section-book">...</section>
</main>

Bu yerda BEM metodologiyasiga o'xshash klass nomlash ishlatilgan: .header__logo-box (Element), .btn--white (Modifier). Bu CSS yozishni ancha tartibli qiladi.

Reset va global stillar

Har qanday loyiha universal reset va global stillardan boshlanadi. Bu barcha brauzerlarda bir xil ko'rinishni ta'minlaydi.

CSS
/* Reset */
*,
*::after,
*::before {
    margin: 0;
    padding: 0;
    box-sizing: inherit;
}

html {
    font-size: 62.5%; /* 1rem = 10px */
}

body {
    box-sizing: border-box;
    font-family: "Lato", sans-serif;
    font-weight: 400;
    line-height: 1.7;
    color: #777;
    padding: 3rem;
}
62.5% hiylasi: font-size: 62.5% qilish orqali 1rem = 10px bo'ladi. Bu rem birligida ishlashni juda osonlashtiradi (masalan, 1.6rem = 16px).

Header komponentini stillash

Header qismi uchun orqa fon rasmi, clip-path bilan kesilgan shakl, logotip va sarlavha stillarini yozamiz.

CSS
.header {
    height: 95vh;
    background-image: linear-gradient(
        to right bottom,
        rgba(126, 213, 111, 0.8),
        rgba(40, 180, 133, 0.8)),
    url(../img/hero.jpg);
    background-size: cover;
    background-position: top;
    clip-path: polygon(0 0, 100% 0, 100% 75vh, 0 100%);
    position: relative;
}

.header__logo-box {
    position: absolute;
    top: 4rem;
    left: 4rem;
}

.header__text-box {
    position: absolute;
    top: 40%;
    left: 50%;
    transform: translate(-50%, -50%);
    text-align: center;
}

Tugmalar va animatsiyalar

Qayta ishlatiladigan tugma komponentini yaratamiz va unga oddiy animatsiya qo'shamiz.

CSS
.btn:link,
.btn:visited {
    text-transform: uppercase;
    text-decoration: none;
    padding: 1.5rem 4rem;
    display: inline-block;
    border-radius: 10rem;
    transition: all .2s;
    position: relative;
    font-size: 1.6rem;
}

.btn--white {
    background-color: #fff;
    color: #777;
}

.btn--animated {
    animation: moveInBottom .5s ease-out .75s;
    animation-fill-mode: backwards;
}

@keyframes moveInBottom {
    0% {
        opacity: 0;
        transform: translateY(3rem);
    }
    100% {
        opacity: 1;
        transform: translate(0);
    }
}

Bilimingizni tekshiring

11-dars testi
1rem birligini oson hisoblash uchun html ga qanday font-size beriladi?
2clip-path xususiyati nima uchun ishlatiladi?
3BEM metodologiyasida Element qanday belgilanadi?
4linear-gradient() ichida rgba ishlatishdan maqsad nima?

Amaliy mashq

Vazifa: O'zingizning loyihangizda CSS asoslarini qo'llang.

  1. HTML faylingizga reset va global stillar qo'shing
  2. rem birligini qulay hisoblash uchun html { font-size: 62.5%; } dan foydalaning
  3. Header qismiga orqa fon rasmi va gradient overlay qo'shing
  4. Tugma komponenti yarating va unga hover animatsiyasi qo'shing
  5. BEM yoki shunga o'xshash klass nomlashdan foydalaning

Namunaviy yechim:

/* Reset */
* { margin: 0; padding: 0; box-sizing: border-box; }

html { font-size: 62.5%; }
body { font-family: 'Segoe UI', sans-serif; line-height: 1.6; }

/* Header */
.header {
    height: 90vh;
    background-image: linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)), url('bg.jpg');
    background-size: cover;
    background-position: center;
    display: flex;
    align-items: center;
    justify-content: center;
}

/* Tugma */
.btn {
    display: inline-block;
    padding: 1.5rem 3rem;
    font-size: 1.6rem;
    text-decoration: none;
    border-radius: 5rem;
    transition: all 0.3s;
}
.btn-primary { background: #3b82f6; color: white; }
.btn-primary:hover { transform: translateY(-3px); box-shadow: 0 10px 20px rgba(0,0,0,0.2); }
