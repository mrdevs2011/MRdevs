Font xossalari

23-fevral, 2021 Fontlar va Transforms O'rta daraja
Bu darsda nimalarni o'rganasiz?

CSS da shriftlar bilan ishlashning ilg'or xususiyatlarini o'rganasiz. font-variant, font-stretch, font-kerning, font-feature-settings va OpenType xususiyatlarini qanday ishlatishni bilib olasiz.

Bu darsda nimalarni o'rganasiz?

  • font-variant - small-caps va boshqa variantlar
  • font-stretch - shrift kengligini sozlash
  • font-kerning - harflar orasidagi maxsus masofa
  • font-feature-settings - OpenType xususiyatlari
  • font-variant-numeric - raqamlar ko'rinishi
  • font-variant-ligatures - ligature sozlamalari

Bu dars uchun Senior maslahatlar

font-variant: small-caps; - bosh harflar kichikroq ko'rinishda. Sarlavhalar uchun ajoyib. Senior yodlaydi
font-feature-settings: OpenType shriftlarning maxsus xususiyatlarini yoqish/o'chirish. "smcp", "liga", "dlig". Professional
font-variant-numeric: lining-nums, oldstyle-nums, tabular-nums - raqamlar uchun. Typography
Brauzer qo'llab-quvvatlashi: Bu xususiyatlar barcha shriftlarda ishlamaydi. OpenType shriftlar kerak. Muhim

font-variant

font-variant xususiyati shrift variantini belgilaydi. Eng ko'p ishlatiladigan qiymat small-caps.

CSS
/* font-variant qiymatlari */
.normal { font-variant: normal; }
.small-caps { font-variant: small-caps; }  /* Barcha harflar kichik bosh harf */

/* Small-caps - sarlavhalar uchun ajoyib */
h2 { font-variant: small-caps; }

/* Shorthand orqali */
.text { font: small-caps 700 2rem/1.5 'Lato', sans-serif; }

font-stretch

font-stretch shriftning kengligini (condensed yoki expanded) belgilaydi. Shrift oilasi bu variantlarni qo'llab-quvvatlashi kerak.

CSS
/* font-stretch qiymatlari */
.ultra-condensed { font-stretch: ultra-condensed; }  /* 50% */
.extra-condensed { font-stretch: extra-condensed; }  /* 62.5% */
.condensed { font-stretch: condensed; }              /* 75% */
.semi-condensed { font-stretch: semi-condensed; }    /* 87.5% */
.normal-stretch { font-stretch: normal; }            /* 100% */
.semi-expanded { font-stretch: semi-expanded; }      /* 112.5% */
.expanded { font-stretch: expanded; }                /* 125% */
.extra-expanded { font-stretch: extra-expanded; }    /* 150% */
.ultra-expanded { font-stretch: ultra-expanded; }    /* 200% */

font-kerning

font-kerning harflar orasidagi maxsus kerning ma'lumotlarini yoqish yoki o'chirish uchun.

CSS
/* font-kerning qiymatlari */
.auto-kerning { font-kerning: auto; }      /* Brauzer qaror qiladi (default) */
.normal-kerning { font-kerning: normal; }  /* Kerning yoqilgan */
.no-kerning { font-kerning: none; }        /* Kerning o'chirilgan */

/* Kerning - harflar orasidagi masofani avtomatik sozlaydi */
/* Masalan: "AV" harflari bir-biriga yaqinroq joylashadi */

font-feature-settings

font-feature-settings OpenType shriftlarning maxsus xususiyatlarini boshqarish uchun.

CSS
/* OpenType feature taglari */
.text {
    /* Small capitals */
    font-feature-settings: "smcp" on;
    
    /* Standard ligatures */
    font-feature-settings: "liga" on;
    
    /* Discretionary ligatures */
    font-feature-settings: "dlig" on;
    
    /* Bir nechta xususiyat */
    font-feature-settings: "smcp" on, "liga" on, "dlig" off;
}

/* Zamonaviy alternativ - font-variant-* */
.modern {
    font-variant-ligatures: common-ligatures discretionary-ligatures;
    font-variant-caps: small-caps;
    font-variant-numeric: lining-nums tabular-nums;
}
Senior biladi: font-variant-* xususiyatlari font-feature-settings dan afzal, chunki ular kaskadlanadi va meros bo'ladi.

font-variant-numeric

Raqamlarning ko'rinishini boshqarish uchun maxsus xususiyatlar.

CSS
/* Raqamlar ko'rinishi */
.numbers {
    /* Lining numbers - bir xil balandlikda */
    font-variant-numeric: lining-nums;
    
    /* Oldstyle numbers - pastga tushib turadigan */
    font-variant-numeric: oldstyle-nums;
    
    /* Tabular numbers - bir xil kenglikda (jadval uchun) */
    font-variant-numeric: tabular-nums;
    
    /* Proportional numbers - har xil kenglikda */
    font-variant-numeric: proportional-nums;
    
    /* Diagonal fractions */
    font-variant-numeric: diagonal-fractions;
    
    /* Bir nechtasi birgalikda */
    font-variant-numeric: lining-nums tabular-nums;
}

Amaliy mashq (Senior daraja)

Vazifa: Font xossalaridan foydalanib professional typography yarating.

  1. Sarlavhalar uchun font-variant: small-caps qo'llang
  2. Raqamlar uchun font-variant-numeric: tabular-nums ishlating
  3. Ligaturelarni yoqish uchun font-variant-ligatures qo'llang
  4. font-kerning ni yoqing
  5. OpenType shrift ishlatganingizga ishonch hosil qiling

Senior darajadagi yechim:

body { font-family: 'Lato', 'Inter', sans-serif; font-kerning: normal; }

h1, h2, h3 { font-variant: small-caps; letter-spacing: 0.5px; }

table { font-variant-numeric: tabular-nums lining-nums; }

.article { font-variant-ligatures: common-ligatures discretionary-ligatures; }

.price { font-variant-numeric: lining-nums proportional-nums; }

.fraction { font-variant-numeric: diagonal-fractions; }

Dars xulosasi (Senior bilishi kerak)

  • font-variant: small-caps - kichik bosh harflar, sarlavhalar uchun
  • font-stretch - shrift kengligi (condensed/expanded)
  • font-kerning - harflar orasidagi kerning
  • font-feature-settings - OpenType xususiyatlari
  • font-variant-numeric - raqamlar ko'rinishi (tabular-nums jadval uchun)
  • font-variant-ligatures - ligature sozlamalari
  • Keyingi darsda: Fonts yordamida loyihani yaxshilab olish
