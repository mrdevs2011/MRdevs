Fonts yordamida loyihani yaxshilab olish

24-fevral, 2021 Amaliy loyiha O'rta daraja
Bu darsda nimalarni o'rganasiz?

Bu darsda 44-47 darslarda o'rgangan shrift mavzularini amaliyotda qo'llab, Natour loyihasining typography tizimini professional darajada yaxshilaymiz. Google Fonts, lokal shriftlar va zamonaviy font xossalaridan foydalanamiz.

Bu darsda nimalarni qilamiz?

  • Loyihaga Google Fonts orqali shrift qo'shish
  • @font-face orqali lokal shriftlarni ulash
  • Sarlavhalar uchun font-variant: small-caps qo'llash
  • rem va em orqali responsive typography
  • letter-spacing va word-spacing orqali matnni yaxshilash

Bu dars uchun Senior maslahatlar

Typography hierarchy: Sarlavhalar, matn, kichik matn uchun aniq o'lchamlar tizimi yarating. Senior yodlaydi
Fallback fontlar: Har doim sans-serif yoki serif zaxira shriftlarni ko'rsating. Professional
Performance: Google Fonts da faqat kerakli qalinliklarni yuklang. &display=swap qo'shing. Optimizatsiya
Responsive typography: @media querylar orqali mobil qurilmalarda shrift o'lchamlarini kichiklashtiring. Mobile-first

Google Fonts qo'shish

HTML
<head>
    <!-- Lato shrifti - 100, 300, 400, 700, 900 qalinliklari -->
    <link href="https://fonts.googleapis.com/css2?family=Lato:wght@100;300;400;700;900&display=swap" rel="stylesheet">
</head>

Typography tizimi

CSS
/* Typography tizimi */
body {
    font-family: "Lato", sans-serif;
    font-weight: 400;
    line-height: 1.7;
    color: #777;
    font-size: 1.6rem;
}

.heading-primary {
    color: #fff;
    text-transform: uppercase;
    backface-visibility: hidden;
    margin-bottom: 6rem;
}

.heading-primary--main {
    display: block;
    font-size: 6rem;
    font-weight: 400;
    letter-spacing: 3.5rem;
}

.heading-primary--sub {
    display: block;
    font-size: 2rem;
    font-weight: 700;
    letter-spacing: 1.75rem;
}

.heading-secondary {
    font-size: 3.5rem;
    text-transform: uppercase;
    font-weight: 700;
    letter-spacing: 0.2rem;
    font-variant: small-caps;
}

.heading-tertiary {
    font-size: 1.6rem;
    font-weight: 700;
    text-transform: uppercase;
}

Responsive Typography

CSS
/* Planshet */
@media (max-width: 56.25em) {
    html { font-size: 56.25%; }
    .heading-primary--main { font-size: 5rem; letter-spacing: 2.5rem; }
    .heading-primary--sub { font-size: 1.8rem; letter-spacing: 1.2rem; }
    .heading-secondary { font-size: 3rem; }
}

/* Mobil */
@media (max-width: 37.5em) {
    html { font-size: 50%; }
    .heading-primary--main { font-size: 4rem; letter-spacing: 1.5rem; }
    .heading-primary--sub { font-size: 1.5rem; letter-spacing: 0.8rem; }
    .heading-secondary { font-size: 2.5rem; }
}

Lokal shrift qo'shish

CSS
/* Lokal shriftlar uchun @font-face */
@font-face {
    font-family: 'Honey';
    src: url('../fonts/HoneyScript-Light.woff2') format('woff2'),
         url('../fonts/HoneyScript-Light.woff') format('woff');
    font-weight: 300;
    font-style: normal;
    font-display: swap;
}

/* Ishlatish */
.logo-text {
    font-family: 'Honey', cursive;
    font-size: 4rem;
}

Amaliy mashq (Senior daraja)

Vazifa: O'z loyihangiz uchun professional typography tizimi yarating.

  1. Google Fonts dan 2 ta shrift tanlang (sarlavha va body uchun)
  2. rem birligidan foydalanib shrift o'lchamlarini belgilang
  3. Sarlavhalar uchun font-variant: small-caps qo'llang
  4. Media querylar orqali responsive typography yarating

Senior darajadagi yechim:

html { font-size: 62.5%; }
body { font-family: 'Lato', sans-serif; font-size: 1.6rem; line-height: 1.6; }
h1 { font-family: 'Playfair Display', serif; font-size: 4.8rem; font-variant: small-caps; }
h2 { font-size: 3.6rem; font-variant: small-caps; }
h3 { font-size: 2.4rem; }

@media (max-width: 37.5em) {
    html { font-size: 50%; }
    h1 { font-size: 3.6rem; }
    h2 { font-size: 2.8rem; }
}

Dars xulosasi (Senior bilishi kerak)

  • Google Fonts: &display=swap bilan ulash, faqat kerakli qalinliklar
  • Typography hierarchy: sarlavhalar, matn, kichik matn uchun tizim
  • rem birligi: shrift o'lchamlari uchun, responsive
  • font-variant: small-caps: sarlavhalar uchun professional ko'rinish
  • Responsive typography: @media da html { font-size } o'zgartirish
  • Keyingi darsda: 2D transforms
