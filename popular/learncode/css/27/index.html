Styling images - Rasmlarni stillash

3-fevral, 2021 Background va Rasmlar O'rta daraja
Bu darsda nimalarni o'rganasiz?

CSS da <img> elementlarini qanday qilib professional darajada stillashni o'rganasiz. Rasmlarni moslashuvchan (responsive) qilish, shakl berish, filterlar va hover effektlar qo'shish - bularning barchasi zamonaviy web dizaynning ajralmas qismi.

Bu darsda nimalarni o'rganasiz?

  • width va height - rasm o'lchamlarini boshqarish
  • object-fit - rasmni konteynerga moslash (cover, contain, fill)
  • border-radius - yumaloq va doira shaklidagi rasmlar
  • box-shadow - rasmga soya qo'shish
  • filter - rasmga effektlar berish (blur, brightness, grayscale)
  • transition + :hover - hover animatsiyalar
  • Responsive rasmlar - max-width: 100%

Bu dars uchun Senior maslahatlar

object-fit vs background-size: object-fit <img> uchun, background-size fon rasmlari uchun. Ikkalasini aralashtirmang! Senior yodlaydi
Performance: filter va box-shadow ko'p ishlatilsa, sahifa sekinlashadi. Muhim elementlardagina ishlating. Optimizatsiya
aspect-ratio: Zamonaviy CSS da aspect-ratio xususiyati rasm nisbatini saqlash uchun juda qulay. Yangi xususiyat
srcset va picture: Responsive rasmlar uchun HTML da srcset va <picture> elementlaridan foydalaning. Senior yondashuv

Rasm o'lchamlari

Rasmlarning width va height xususiyatlari orqali o'lchamlarini boshqarish. Responsive dizayn uchun foizlardan foydalanish tavsiya etiladi.

CSS 
/* Rasm o'lchamlari */
img {
    width: 100%;           /* Ota element kengligicha */
    height: auto;          /* Nisbat saqlanadi */
    max-width: 100%;       /* Asl o'lchamidan katta bo'lmaydi */
}

/* Belgilangan o'lcham */
.fixed-img {
    width: 300px;
    height: 200px;
}

/* Foizda - responsive */
.responsive-img {
    width: 100%;
    height: auto;
}

/* Max-width bilan */
.max-width-img {
    max-width: 100%;
    height: auto;
}
Senior biladi: width: 100% va max-width: 100% farqi: width: 100% rasmni har doim ota element kengligicha cho'zadi (sifati buzilishi mumkin). max-width: 100% esa rasm asl o'lchamidan katta bo'lmaydi, faqat kichik ekranlarda kichiklashadi.

object-fit - Rasmni moslash

object-fit xususiyati <img> yoki <video> elementining konteynerga qanday moslashishini belgilaydi. Bu background-size ning rasm versiyasi.

CSS 
/* object-fit qiymatlari */
.img-cover {
    width: 300px;
    height: 200px;
    object-fit: cover;      /* Konteynerni to'liq qoplaydi, kesilishi mumkin */
}

.img-contain {
    width: 300px;
    height: 200px;
    object-fit: contain;    /* Rasm to'liq ko'rinadi, bo'sh joy qolishi mumkin */
}

.img-fill {
    width: 300px;
    height: 200px;
    object-fit: fill;       /* Konteynerni to'ldiradi, nisbat buziladi (default) */
}

.img-scale-down {
    width: 300px;
    height: 200px;
    object-fit: scale-down; /* contain yoki none, qaysi kichikroq bo'lsa */
}

.img-none {
    width: 300px;
    height: 200px;
    object-fit: none;       /* Asl o'lcham, kesilishi mumkin */
}
QiymatTavsifQachon ishlatiladi
coverNisbat saqlanadi, konteyner to'liq qoplanadiProfil rasmi, galereya
containNisbat saqlanadi, rasm to'liq ko'rinadiLogotip, mahsulot rasmi
fillNisbat buziladi, konteyner to'liq to'ladiJuda kam (deyarli ishlatilmaydi)
object-fit + object-position: object-position xususiyati rasmning qaysi qismi ko'rinishini belgilaydi. Masalan: object-position: top left;

border-radius - Yumaloq rasmlar

border-radius xususiyati rasmga yumaloq burchaklar berish yoki butunlay doira shakliga keltirish uchun ishlatiladi.

CSS 
/* border-radius qiymatlari */
.rounded-sm {
    border-radius: 4px;        /* Kichik yumaloqlik */
}

.rounded-md {
    border-radius: 8px;        /* O'rta yumaloqlik */
}

.rounded-lg {
    border-radius: 16px;       /* Katta yumaloqlik */
}

.rounded-full {
    border-radius: 50%;        /* Doira shakli (kvadrat rasm uchun) */
}

/* Doira shaklidagi avatar */
.avatar {
    width: 100px;
    height: 100px;
    border-radius: 50%;
    object-fit: cover;         /* Rasm to'g'ri ko'rinishi uchun */
}

/* Alohida burchaklar */
.custom-radius {
    border-radius: 20px 5px 40px 10px;  /* yuqori-chap, yuqori-o'ng, pastki-o'ng, pastki-chap */
}

box-shadow va filter

box-shadow rasmga soya qo'shadi. filter esa rasmga turli xil vizual effektlar beradi.

CSS 
/* box-shadow */
.img-shadow {
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}

.img-shadow-lg {
    box-shadow: 0 10px 25px rgba(0, 0, 0, 0.2);
}

/* filter effektlari */
.img-grayscale {
    filter: grayscale(100%);    /* Oq-qora */
}

.img-blur {
    filter: blur(5px);          /* Xiralashgan */
}

.img-brightness {
    filter: brightness(1.2);    /* Yorqinroq (1 = normal) */
}

.img-contrast {
    filter: contrast(1.5);      /* Kontrastli */
}

.img-sepia {
    filter: sepia(100%);        /* Sepia (eski rasm) effekti */
}

/* Bir nechta filter birgalikda */
.img-combined {
    filter: grayscale(50%) blur(2px) brightness(1.1);
}

Hover effektlar

:hover psevdo-klassi va transition xususiyati orqali rasmlarga silliq animatsiyalar qo'shish.

CSS 
/* Asosiy hover effektlari */
.img-hover {
    transition: all 0.3s ease;
}

.img-hover:hover {
    transform: scale(1.05);         /* Kattalashish */
    box-shadow: 0 10px 25px rgba(0, 0, 0, 0.2);
}

/* Zoom effekti (overflow: hidden bilan) */
.img-zoom-container {
    overflow: hidden;
    border-radius: 8px;
}

.img-zoom {
    transition: transform 0.5s ease;
}

.img-zoom:hover {
    transform: scale(1.2);
}

/* Oq-qora dan rangliga */
.img-color-transition {
    filter: grayscale(100%);
    transition: filter 0.4s ease;
}

.img-color-transition:hover {
    filter: grayscale(0%);
}

/* Yorqinlik o'zgarishi */
.img-brightness-hover {
    transition: filter 0.3s ease;
}

.img-brightness-hover:hover {
    filter: brightness(1.2);
}

Professional galereya namunasi

CSS 
/* Zamonaviy galereya kartasi */
.gallery-card {
    position: relative;
    overflow: hidden;
    border-radius: 12px;
    cursor: pointer;
}

.gallery-card img {
    width: 100%;
    height: 300px;
    object-fit: cover;
    transition: transform 0.5s ease, filter 0.5s ease;
}

.gallery-card:hover img {
    transform: scale(1.1);
    filter: brightness(0.8);
}

/* Overlay effekti */
.gallery-overlay {
    position: absolute;
    top: 0;
    left: 0;

    height: 100%;
    background: rgba(0, 0, 0, 0.5);
    display: flex;
    align-items: center;
    justify-content: center;
    opacity: 0;
    transition: opacity 0.3s ease;
}

.gallery-card:hover .gallery-overlay {
    opacity: 1;
}

.gallery-overlay span {
    color: white;
    font-size: 1.5rem;
    font-weight: 600;
}

Amaliy mashq (Senior daraja)

Vazifa: Rasmlarni professional darajada stillab, zamonaviy galereya yarating.

  1. 3 ta rasm uchun karta yarating (display: grid yoki flex)
  2. Har bir rasmga object-fit: cover va belgilangan o'lcham bering
  3. border-radius bilan yumaloq burchaklar qo'shing
  4. Hover bo'lganda rasm kattalashsin (transform: scale())
  5. overflow: hidden bilan zoom effekti yarating
  6. Rasm ustiga shaffof overlay va matn qo'shing
  7. Barcha animatsiyalarga transition qo'shing

  • overflow: hidden - zoom effekti uchun muhim
