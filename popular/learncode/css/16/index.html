Text align, Vertical align va Float

23-yanvar, 2021 Display va Positioning Boshlang'ich daraja
Bu darsda nimalarni o'rganasiz?

CSS da matnlarni gorizontal tekislash (text-align), inline va inline-block elementlarni vertikal tekislash (vertical-align), va elementlarni suzdirib joylashtirish (float) xususiyatlarini o'rganasiz. Bu xususiyatlar sahifa layoutini yaratishda juda muhim.

Bu darsda nimalarni o'rganasiz?

  • text-align - matnni gorizontal tekislash
  • vertical-align - inline elementlarni vertikal tekislash
  • float - elementlarni suzdirib joylashtirish
  • clear - float ni tozalash
  • Float bilan layout yaratish usullari

Bu dars uchun maslahatlar

text-align faqat inline kontent uchun: Bu xususiyat block element ichidagi matn va inline elementlarni tekislaydi, block elementning o'zini emas. yodlash shart
vertical-align faqat inline/inline-block uchun: Bu xususiyat faqat inline yoki inline-block elementlarda ishlaydi. Block elementlarda ishlamaydi! muhim
Float o'rniga Flexbox: Zamonaviy layoutlar uchun float o'rniga Flexbox yoki Grid ishlatish tavsiya etiladi. amaliyot

Text align - matnni gorizontal tekislash

text-align xususiyati block element ichidagi matn va inline elementlarni gorizontal tekislash uchun ishlatiladi.

CSS 
/* text-align qiymatlari */
.text-left {
    text-align: left;      /* Chapga tekislash (default) */
}

.text-center {
    text-align: center;    /* Markazga tekislash */
}

.text-right {
    text-align: right;     /* O'ngga tekislash */
}

.text-justify {
    text-align: justify;   /* Ikki tomonga tekislash */
}

/* Block elementni markazga tekislash uchun */
.container {
    width: 80%;
    margin: 0 auto;        /* Block elementni markazga tekislash */
    text-align: center;    /* Ichidagi matnni markazga tekislash */
}
Eslatma: text-align block elementning o'zini emas, balki uning ichidagi kontentni tekislaydi. Block elementni markazga tekislash uchun margin: 0 auto; ishlatiladi.

Vertical align - vertikal tekislash

vertical-align xususiyati inline yoki inline-block elementlarni vertikal tekislash uchun ishlatiladi.

CSS 
/* vertical-align qiymatlari */
.inline-element {
    display: inline-block;
    vertical-align: baseline;    /* Asosiy chiziq bo'yicha (default) */
}

.top-aligned {
    vertical-align: top;         /* Tepaga tekislash */
}

.middle-aligned {
    vertical-align: middle;      /* O'rtaga tekislash */
}

.bottom-aligned {
    vertical-align: bottom;      /* Pastga tekislash */
}

/* Matn bilan rasmni tekislash */
img {
    vertical-align: middle;      /* Rasm matn bilan o'rtada turadi */
}
Diqqat: vertical-align faqat inline va inline-block elementlarda ishlaydi. Block elementlarda bu xususiyat ishlamaydi!
Inline-block elementlar orasidagi bo'shliq: Inline-block elementlar orasida kichik bo'shliq paydo bo'ladi. Uni yo'qotish uchun ota elementga font-size: 0 berib, elementlarga kerakli font-size ni qayta tiklang.

Float - elementlarni suzdirish

float xususiyati elementni chap yoki o'ng tomonga suzdirib, boshqa elementlarning uning atrofida joylashishini ta'minlaydi.

CSS 
/* float qiymatlari */
.float-left {
    float: left;        /* Chapga suzdirish */
    margin-right: 20px; /* O'ngdan bo'shliq */
}

.float-right {
    float: right;       /* O'ngga suzdirish */
    margin-left: 20px;  /* Chapdan bo'shliq */
}

.float-none {
    float: none;        /* Suzdirishni bekor qilish (default) */
}

/* Rasm va matn - klassik misol */
img {
    float: left;
    margin: 0 20px 20px 0;
    width: 300px;
}

Clear - float ni tozalash

clear xususiyati elementning qaysi tomondan float elementlar ta'siridan xalos bo'lishini belgilaydi.

CSS 
/* clear qiymatlari */
.clear-left {
    clear: left;        /* Chap float dan tozalash */
}

.clear-right {
    clear: right;       /* O'ng float dan tozalash */
}

.clear-both {
    clear: both;        /* Ikkala float dan tozalash */
}

/* Clearfix - float konteyner muammosini hal qilish */
.clearfix::after {
    content: "";
    display: table;
    clear: both;
}
Float konteyner muammosi: Agar ota element faqat float elementlardan iborat bo'lsa, uning balandligi 0 bo'lib qoladi. Buni clearfix yoki overflow: hidden bilan hal qilish mumkin.

Float bilan layout yaratish

Float yordamida ko'p ustunli layoutlar yaratish mumkin. Bu Flexbox va Grid dan oldin asosiy layout usuli edi.

CSS 
/* 2 ustunli layout */
.column {
    float: left;
    width: 50%;
    padding: 20px;
    box-sizing: border-box;
}

/* 3 ustunli layout */
.column-3 {
    float: left;
    width: 33.333%;
    padding: 15px;
    box-sizing: border-box;
}

/* Sidebar va asosiy kontent */
.sidebar {
    float: left;
    width: 30%;
}

.main-content {
    float: right;
    width: 70%;
}

/* Clearfix */
.row::after {
    content: "";
    display: table;
    clear: both;
}
Zamonaviy alternativ: Bugungi kunda float o'rniga display: flex yoki display: grid ishlatish tavsiya etiladi. Float asosan rasm atrofida matn joylashtirish uchun ishlatiladi.

16-dars testi

4 ta savol
1 text-align: center nimani markazga tekislaydi?
2 vertical-align qaysi turdagi elementlarda ishlaydi?
3 Elementni chap tomonga suzdirish uchun qaysi xususiyat ishlatiladi?
4 Float elementlardan keyin float ta'sirini tozalash uchun qaysi xususiyat ishlatiladi?

Amaliy mashq

Vazifa: Text align, vertical align va float xususiyatlaridan foydalanib sahifa yarating.

  1. Sarlavhani text-align: center bilan markazga tekislang
  2. Rasmni float: left qilib, matn uning atrofida joylashsin
  3. 3 ta inline-block element yarating va ularni vertical-align: middle bilan tekislang
  4. Float elementlardan keyin clear: both ishlating
  5. 2 ustunli layout yarating (sidebar va main content)

Namunaviy yechim:

<h1 style="text-align: center;">Sarlavha</h1>

<div class="clearfix">
    <img src="image.jpg" alt="Rasm" style="float: left; margin-right: 20px; width: 300px;">
    <p>Rasm atrofidagi matn...</p>
</div>

<div style="font-size: 0; margin: 30px 0;">
    <div style="display: inline-block; width: 100px; height: 100px; background: #3b82f6; vertical-align: middle; font-size: 16px;">Element 1</div>
    <div style="display: inline-block; width: 150px; height: 150px; background: #10b981; vertical-align: middle; font-size: 16px;">Element 2</div>
    <div style="display: inline-block; width: 80px; height: 80px; background: #f59e0b; vertical-align: middle; font-size: 16px;">Element 3</div>
</div>

<div class="row">
    <div class="sidebar">Sidebar</div>
    <div class="main-content">Asosiy kontent</div>
</div>

  • vertical-align - faqat inline va inline-block elementlarni vertikal tekislaydi
  • float - elementni chap yoki o'ngga suzdirib, boshqa elementlarni atrofida joylashtiradi
  • clear - float ta'sirini tozalaydi (left, right, both)
  • Clearfix - float konteyner muammosini hal qilish uchun
