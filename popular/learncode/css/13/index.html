Margin collapsing nima?

Margin collapsing - bu ikki elementning vertikal marginlari ustma-ust tushganda, ularning qo'shilmay, balki kattaroq margin qolib, kichigi "yopilib" qolish hodisasi.

Oddiy misol
Element 1: margin-bottom: 30px
Oraliq: 30px (50px emas!)
Element 2: margin-top: 20px

Qo'shni elementlarda margin collapsing

Ketma-ket kelgan ikki elementning vertikal marginlari orasida margin collapsing yuz beradi. Natijada ular orasidagi masofa marginlarning yig'indisi emas, balki kattaroq margin bo'ladi.

CSS
.box1 {
    margin-bottom: 30px;
}

.box2 {
    margin-top: 20px;
}

/* Oraliq 50px (30+20) emas, 30px bo'ladi! */
Diqqat: Agar ikkala margin teng bo'lsa, oraliq o'sha qiymatga teng bo'ladi. Agar biri manfiy bo'lsa, qiymatlar qo'shiladi (masalan, 30px + (-10px) = 20px).

Ota-bola margin collapsing

Agar ota elementning yuqori/past marginidan oldin hech qanday kontent (border, padding, matn) bo'lmasa, bola elementning margini ota elementdan "tashqariga chiqib" ketadi.

CSS
.parent {
    background: #f0f9ff;
    /* border yoki padding yo'q */
}

.child {
    margin-top: 40px;
}

/* .child ning margin-top .parent dan tashqariga chiqadi! */
Yechim: Ota elementga border, padding, overflow: hidden yoki display: flow-root berish orqali margin collapsing ni oldini olish mumkin.

Margin collapsing ni oldini olish

1
Border yoki padding qo'shish

Ota elementga border: 1px solid transparent yoki padding: 1px berish.

2
overflow: hidden

Ota elementga overflow: hidden berish BFC (Block Formatting Context) yaratadi.

3
display: flow-root

Eng zamonaviy va to'g'ri usul: display: flow-root.

4
Flexbox yoki Grid

display: flex yoki display: grid ichida margin collapsing yuz bermaydi.

Shorthand - bu bir nechta CSS xususiyatlarini bitta qatorda yozish imkonini beruvchi qisqa yozuv.

Margin/Padding qiymatlar tartibi
/* 1 ta qiymat - hamma tomon */
margin: 20px;

/* 2 ta qiymat - yuqori/past, o'ng/chap */
margin: 10px 20px;

/* 3 ta qiymat - yuqori, o'ng/chap, past */
margin: 10px 20px 15px;

/* 4 ta qiymat - yuqori, o'ng, past, chap (soat strelkasi) */
margin: 10px 20px 15px 25px;

Amaliy mashq

Vazifa: Margin collapsing ni kuzating va oldini oling.

  1. Ikkita div yarating, birinchisiga margin-bottom: 40px, ikkinchisiga margin-top: 30px bering. Oraliq qancha bo'ladi?
  2. Ota div va bola h1 yarating. h1 ga margin-top: 50px bering. Nima bo'ladi?
  3. Ota elementga border: 1px solid transparent qo'shib, collapsing ni oldini oling.
  4. Shorthand yordamida margin va padding larni qisqa yozuvda bering.

Namunaviy yechim:

<style>
.box1 { margin-bottom: 40px; background: #93c5fd; padding: 20px; }
.box2 { margin-top: 30px; background: #6ee7b7; padding: 20px; }
/* Oraliq 40px bo'ladi (30px collapsing bo'ladi) */

.parent { background: #fef3c7; /* border: 1px solid transparent; */ }
.child { margin-top: 50px; }
/* border qo'shilmasa, .child margin .parent dan tashqariga chiqadi */

/* Shorthand */
.card { margin: 20px 15px; padding: 25px 20px 15px; }
</style>

<div class="box1">Birinchi element (margin-bottom: 40px)</div>
<div class="box2">Ikkinchi element (margin-top: 30px)</div>

<div class="parent">
    <h1 class="child">Sarlavha (margin-top: 50px)</h1>
</div>
