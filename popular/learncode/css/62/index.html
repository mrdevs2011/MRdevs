Best practices

Bu darsda nimalarni o'rganasiz?

CSS yozishning professional standartlari va eng yaxshi amaliyotlari. Kodni toza, qayta ishlatiladigan, performansli va kengaytiriladigan qilish sirlari. Senior dasturchi bo'lish uchun bu bilimlar majburiy!

  • CSS metodologiyalari (BEM, OOCSS, SMACSS)
  • Kodni tashkil qilish va komponentlarga ajratish
  • CSS o'zgaruvchilari (Custom Properties)
  • Performans optimizatsiyasi
  • Accessibility (hamyonboplik)
  • CSS ni qayta ishlatish (reusability)

BEM (Block Element Modifier)

BEM - CSS klasslarini nomlashning eng mashhur metodologiyasi. Kodni tushunarli, qayta ishlatiladigan va konfliktlardan xoli qiladi.

BEM sintaksisi
/* BEM: .block__element--modifier */
.card { }                    /* Block */
.card__title { }             /* Element */
.card__title--large { }      /* Modifier */
.card--featured { }          /* Block modifier */

/* Amaliy misol */
.search-form { }
.search-form__input { }
.search-form__input--error { }
.search-form__button { }
.search-form__button--loading { }
Senior biladi: BEM specificity muammolarini butunlay yo'q qiladi. Barcha selektorlar bir xil specificity (10 ball) ga ega bo'ladi.

CSS Custom Properties

CSS o'zgaruvchilari ranglar, spacing, breakpointlar kabi qiymatlarni markazlashtirish imkonini beradi.

CSS
:root {
    /* Ranglar */
    --color-primary: #3b82f6;
    --color-primary-dark: #2563eb;
    --color-secondary: #10b981;
    
    /* Spacing */
    --spacing-xs: 0.4rem;
    --spacing-sm: 0.8rem;
    --spacing-md: 1.6rem;
    --spacing-lg: 2.4rem;
    --spacing-xl: 3.2rem;
    
    /* Breakpoints */
    --bp-mobile: 37.5em;
    --bp-tablet: 56.25em;
    --bp-desktop: 75em;
}

/* Ishlatish */
.btn { background: var(--color-primary); padding: var(--spacing-md) var(--spacing-lg); }
@media (min-width: var(--bp-tablet)) { }

Performans optimizatsiyasi

1
Animatsiya uchun transform va opacity

GPU da ishlaydi, silliq animatsiya.

2
will-change

Faqat animatsiya bo'ladigan elementlarga bering.

3
contain

contain: layout style paint; - brauzerga element mustaqil ekanligini bildiradi.

4
content-visibility

content-visibility: auto; - ekrandan tashqaridagi elementlarni render qilmaydi.

Accessibility (Hamyonboplik)

CSS
/* Focus ko'rinadigan qilish */
:focus-visible { outline: 3px solid #3b82f6; outline-offset: 2px; }

/* Animatsiyalarni kamaytirish */
@media (prefers-reduced-motion: reduce) {
    *, *::before, *::after { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; }
}

/* Kontrast ranglar - WCAG AA standarti */
.text-muted { color: #64748b; }  /* Kontrast nisbati 4.5:1 dan yuqori */

Amaliy mashq

Vazifa: O'z loyihangizga best practices larni qo'llang.

  1. BEM metodologiyasiga o'ting
  2. CSS o'zgaruvchilarini yarating (ranglar, spacing, breakpoints)
  3. prefers-reduced-motion media query qo'shing
  4. :focus-visible stillarini qo'shing

Namunaviy yechim:

:root { --color-primary: #3b82f6; --spacing-md: 1.6rem; }
.card { background: white; padding: var(--spacing-md); }
.card__title { font-size: 2rem; }
.card__title--large { font-size: 2.4rem; }
:focus-visible { outline: 3px solid var(--color-primary); outline-offset: 2px; }
@media (prefers-reduced-motion: reduce) { *, *::before, *::after { animation: none !important; transition: none !important; } }
