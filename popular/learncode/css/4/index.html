Selectors (Tanlab oluvchilar)

11-yanvar, 2021 CSS Asoslari Boshlang'ich daraja
Bu darsda nimalarni o'rganasiz?

CSS selektorlari - bu CSS ning eng muhim qismlaridan biri. Selektorlar yordamida siz HTML elementlarini tanlab, ularga stillar berasiz. Bu darsda asosiy selektor turlarini: universal, element, class va ID selektorlarini o'rganasiz.

Bu darsda nimalarni o'rganasiz?

  • Selektor nima va u qanday ishlaydi?
  • Universal selektor (*) - barcha elementlarni tanlash
  • Element selektor - teg nomi orqali tanlash
  • Class selektor (.) - klass nomi orqali tanlash
  • ID selektor (#) - ID orqali tanlash
  • Selektorlarni guruhlash - bir nechta selektorga bir xil stil berish

Bu dars uchun maslahatlar

Class selektorni ko'proq ishlating: Class selektorlar eng ko'p ishlatiladigan selektor turidir. ID selektorni CSS da kamroq ishlating. yodlash shart
Sintaksisni eslab qoling: Class uchun nuqta (.), ID uchun panjara (#), universal uchun yulduzcha (*). tushunish kerak
Selektorlarni guruhlang: Bir xil stillarni bir nechta selektorga berish uchun vergul (,) dan foydalaning. amaliyot

Selektor nima?

Selektor - bu CSS qoidasining qaysi HTML elementlariga qo'llanilishini belgilaydigan qism. Selektor orqali siz brauzerga "mana shu elementlarni ol va ularga quyidagi stillarni ber" deysiz.

CSS qoidasi tuzilishi
selektor
{
xususiyat: qiymat;
}

Selektor - "kimga", xususiyat: qiymat - "nima qilish kerak"

Universal selektor (*)

Universal selektor (*) - sahifadagi barcha elementlarni tanlaydi. Ko'pincha margin va padding ni nollash uchun ishlatiladi.

CSS 
/* Barcha elementlarning margin va paddingini nollash */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

/* Barcha elementlarga bir xil shrift berish */
* {
    font-family: 'Segoe UI', sans-serif;
}
Diqqat: Universal selektor juda kuchli. Uni ehtiyotkorlik bilan ishlating, chunki u sahifadagi barcha elementlarga ta'sir qiladi va sahifa yuklanishini sekinlashtirishi mumkin.

Element selektor (Teg selektori)

Element selektor - HTML tegining nomi orqali elementlarni tanlaydi. Masalan: h1, p, div, span.

CSS 
/* Barcha h1 sarlavhalarni tanlash */
h1 {
    color: #2563eb;
    text-align: center;
    font-size: 32px;
}

/* Barcha paragraflarni tanlash */
p {
    line-height: 1.6;
    margin-bottom: 16px;
    color: #334155;
}

/* Barcha div elementlarini tanlash */
div {
    padding: 20px;
    background-color: #f8fafc;
}
Qachon ishlatiladi? Element selektor sayt bo'ylab bir xil ko'rinishda bo'lishi kerak bo'lgan elementlar uchun ishlatiladi. Masalan, barcha paragraflar bir xil shrift va qator oralig'iga ega bo'lishi uchun.

Class selektor (.) - Eng ko'p ishlatiladigan

Class selektor - nuqta (.) bilan boshlanadi va klass nomi orqali elementlarni tanlaydi. Bir klassni ko'plab elementlarga berish mumkin.

HTML 
<h2 class="title">Sarlavha</h2>
<p class="title">Bu ham title klassiga ega</p>
<div class="box red">Bir nechta klass</div>
CSS 
/* title klassiga ega barcha elementlar */
.title {
    font-weight: bold;
    color: #2563eb;
}

/* box klassiga ega elementlar */
.box {
    padding: 20px;
    border-radius: 8px;
}

/* red klassiga ega elementlar */
.red {
    background-color: #fee2e2;
    color: #dc2626;
}
Muhim: Bitta elementga bir nechta klass berish mumkin. Klasslar orasiga bo'shliq qo'yiladi: class="box red large"

ID selektor (#)

ID selektor - panjara (#) bilan boshlanadi va ID nomi orqali elementni tanlaydi. ID sahifada faqat BIR marta ishlatilishi kerak.

HTML 
<div id="header">
    <h1>Sayt sarlavhasi</h1>
</div>

<div id="main-content">
    <p>Asosiy kontent</p>
</div>

<div id="footer">
    <p>Footer</p>
</div>
CSS 
#header {
    background-color: #1e293b;
    color: white;
    padding: 20px;
}

#main-content {
    max-width: 800px;
    margin: 0 auto;
}

#footer {
    text-align: center;
    padding: 10px;
    border-top: 1px solid #e2e8f0;
}
CSS da ID ishlatmang! ID selektor juda yuqori o'ziga xoslikka (specificity) ega. Bu keyinchalik stillarni qayta yozishda muammo tug'diradi. CSS da stillash uchun har doim class selektordan foydalaning. ID ni faqat JavaScript yoki sahifa ichidagi havolalar uchun ishlating.

Selektorlarni guruhlash

Bir xil stillarni bir nechta selektorga berish uchun ularni vergul (,) bilan ajratib yozish mumkin. Bu kodni qisqartiradi va toza saqlaydi.

CSS 
/* Guruhlanmagan - kod takrorlanadi */
h1 {
    font-family: 'Segoe UI', sans-serif;
    color: #1e293b;
}
h2 {
    font-family: 'Segoe UI', sans-serif;
    color: #1e293b;
}
h3 {
    font-family: 'Segoe UI', sans-serif;
    color: #1e293b;
}

/* Guruhlangan - toza va qisqa */
h1, h2, h3 {
    font-family: 'Segoe UI', sans-serif;
    color: #1e293b;
}

Selektorlar xulosasi

Selektor Sintaksis Misol Qachon ishlatiladi?
Universal * * { margin: 0; } Barcha elementlarga bir xil stil berish
Element teg p { color: blue; } Bir xil turdagi barcha elementlarga
Class .class .btn { padding: 10px; } Eng ko'p ishlatiladi, qayta ishlatish mumkin
ID #id #header { height: 80px; } Faqat bir marta, CSS da tavsiya etilmaydi
Guruhlash sel1, sel2 h1, h2, h3 { font: bold; } Bir xil stillarni bir nechta selektorga berish

Bilimingizni tekshiring

4-dars testi
1 Class selektor qaysi belgi bilan boshlanadi?
2 Qaysi selektor sahifadagi barcha elementlarni tanlaydi?
3 CSS da eng ko'p tavsiya etiladigan selektor turi qaysi?

Amaliy mashq

Vazifa: Har xil selektor turlaridan foydalanib, HTML elementlariga stillar bering.

  1. index.html va style.css fayllarini yarating
  2. Universal selektor yordamida barcha elementlarning margin va padding ini nollang
  3. Element selektor yordamida barcha h1 larni ko'k rangda, markazga tekislangan qiling
  4. Class selektor yordamida .card klassini yarating: oq fon, 20px padding, yumaloq burchaklar, soya
  5. ID selektor yordamida #footer elementini yarating: to'q fon, oq matn, markazga tekislangan

Namunaviy yechim:

index.html:

<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Selektorlar bilan tanishtiruv</h1>
    
    <div class="card">
        <h2>Karta sarlavhasi</h2>
        <p>Bu karta ichidagi matn.</p>
    </div>
    
    <div class="card">
        <h2>Ikkinchi karta</h2>
        <p>Yana bir karta.</p>
    </div>
    
    <div id="footer">
        <p>© 2026 Mening Saytim</p>
    </div>
</body>
</html>

style.css:

/* Universal selektor */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: 'Segoe UI', sans-serif;
    padding: 40px;
    background: #f0f9ff;
}

/* Element selektor */
h1 {
    color: #2563eb;
    text-align: center;
    margin-bottom: 30px;
}

/* Class selektor */
.card {
    background: white;
    padding: 20px;
    border-radius: 12px;
    box-shadow: 0 4px 6px rgba(0,0,0,0.1);
    margin-bottom: 20px;
}

.card h2 {
    color: #1e293b;
    margin-bottom: 10px;
}

/* ID selektor */
#footer {
    background: #1e293b;
    color: white;
    text-align: center;
    padding: 20px;
    margin-top: 40px;
    border-radius: 8px;
}

Dars xulosasi

  • Selektor - CSS qoidasining qaysi elementlarga qo'llanilishini belgilaydi
  • Universal (*) - barcha elementlarni tanlaydi
  • Element - teg nomi orqali tanlaydi (h1, p, div)
  • Class (.) - klass nomi orqali tanlaydi, eng ko'p ishlatiladi
  • ID (#) - ID orqali tanlaydi, CSS da tavsiya etilmaydi
  • Guruhlash (,) - bir nechta selektorga bir xil stil berish
