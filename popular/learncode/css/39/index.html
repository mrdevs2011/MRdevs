Bu dars uchun Senior maslahatlar

Mobile-first: Avval mobil versiyani yozib, keyin katta ekranlarga moslash. Bu eng zamonaviy va tavsiya etilgan yondashuv. Senior yodlaydi
Viewport meta: <meta name="viewport" content="width=device-width, initial-scale=1.0"> - har bir HTML faylida bo'lishi shart! Majburiy
Breakpointlar: Qurilmalar uchun emas, kontent uchun breakpoint tanlang. Dizayn buziladigan nuqtada media query qo'shing. Professional
Responsive images: srcset va <picture> orqali turli ekranlar uchun turli o'lchamdagi rasmlar yuklang. Optimizatsiya

Responsive Web Design nima?

Responsive Web Design (RWD) - bu veb-saytning turli ekran o'lchamlari va qurilmalarda (mobil, planshet, desktop) to'g'ri va chiroyli ko'rinishini ta'minlovchi dizayn yondashuvi.

RWD tushunchasi

Mobil
< 600px

Planshet
600px - 900px

Desktop
> 900px
Asosiy tamoyillar: Fluid gridlar (foizlar), Flexible images (max-width: 100%), Media queries (shartli stillar).

Mobile-first vs Desktop-first

XususiyatDesktop-firstMobile-first
YondashuvKatta ekrandan kichikkaKichik ekrandan kattaga
Media query@media (max-width: ...)@media (min-width: ...)
Default stillarDesktop uchunMobil uchun
Kod hajmi (mobil)Ko'proq (override)Kamroq
TavsiyaEski loyihalarZamonaviy loyihalar ⭐
Mobile-first misol
/* Mobile-first: default - mobil */
.container {
    width: 100%;
    padding: 1rem;
}

/* Planshet */
@media (min-width: 600px) {
    .container {
        width: 90%;
        padding: 2rem;
    }
}

/* Desktop */
@media (min-width: 900px) {
    .container {
        width: 80%;
        max-width: 1200px;
        padding: 3rem;
    }
}

Viewport Meta tegi

Mobil qurilmalarda to'g'ri ko'rinish uchun HTML faylning <head> qismiga viewport meta tegi qo'shilishi shart.

HTML
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- Kengaytirilgan versiya -->
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=5.0, user-scalable=yes">
Diqqat: Viewport meta tegisiz mobil qurilmalarda sayt desktop versiyasida ko'rinadi (kichiklashgan holda). Bu foydalanuvchi tajribasini yomonlashtiradi.

Responsive Images

Turli ekran o'lchamlari uchun turli o'lchamdagi rasmlar yuklash orqali sayt tezligini oshirish.

HTML
<!-- srcset - brauzer tanlaydi -->
<img 
    src="img/small.jpg" 
    srcset="img/small.jpg 480w, img/medium.jpg 800w, img/large.jpg 1200w"
    sizes="(max-width: 600px) 480px, (max-width: 900px) 800px, 1200px"
    alt="Responsive rasm"
>

<!-- Picture - aniq nazorat -->
<picture>
    <source media="(max-width: 600px)" srcset="img/mobile.jpg">
    <source media="(max-width: 900px)" srcset="img/tablet.jpg">
    <img src="img/desktop.jpg" alt="Responsive rasm">
</picture>

Amaliy mashq

Vazifa: Mobile-first yondashuvda oddiy responsive sahifa yarating.

  1. Viewport meta tegini qo'shing
  2. Default (mobil) stillarni yozing
  3. min-width media query orqali planshet va desktop stillarni qo'shing
  4. Rasmlar uchun srcset yoki picture ishlating

Namunaviy yechim:

/* Mobile-first - default mobil */
.container { width: 100%; padding: 1rem; }
h1 { font-size: 2.4rem; }
.grid { display: block; }
.card { margin-bottom: 2rem; }

/* Planshet */
@media (min-width: 600px) {
    .container { width: 90%; padding: 2rem; }
    h1 { font-size: 3.2rem; }
    .grid { display: grid; grid-template-columns: repeat(2, 1fr); gap: 2rem; }
}

/* Desktop */
@media (min-width: 900px) {
    .container { width: 80%; max-width: 1200px; padding: 3rem; }
    h1 { font-size: 4.8rem; }
    .grid { grid-template-columns: repeat(3, 1fr); }
}
