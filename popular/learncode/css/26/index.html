Background - Fon xususiyatlari

2-fevral, 2021 Background va Rasmlar O'rta daraja
Bu darsda nimalarni o'rganasiz?

CSS da background xususiyati va uning barcha qismlarini chuqur o'rganasiz. Bu bilimlar professional web dizayn yaratishda asosiy o'rin tutadi. Background orqali elementlarga rang, rasm, gradient berish va ularni boshqarishni o'rganasiz.

Bu darsda nimalarni o'rganasiz?

  • background-color - fon rangi berish
  • background-image - fon rasmi qo'shish
  • background-repeat - rasm takrorlanishini boshqarish
  • background-size - rasm o'lchamini belgilash
  • background-position - rasm joylashuvini belgilash
  • background-attachment - scroll effekti
  • background shorthand - barchasini bitta qatorda

Bu dars uchun Senior maslahatlar

Shorthand tartibi: background: color image repeat attachment position/size; - bu tartibni yodlang. Noto'g'ri tartibda yozilsa, brauzer tushunmasligi mumkin. Senior yodlaydi
Performance: Ko'p background-image ishlatish sayt tezligiga ta'sir qiladi. Iloji boricha SVG yoki CSS gradientlardan foydalaning. Optimizatsiya
cover vs contain: cover butun konteynerni qoplaydi (kesilishi mumkin), contain rasm to'liq ko'rinadi (bo'sh joy qolishi mumkin). Muhim farq
Mobile-first: Katta background rasmlar mobil qurilmalarda sekin yuklanadi. Media queries orqali kichikroq rasmlar yuklang. Senior yondashuv

background-color - Fon rangi

background-color xususiyati elementning fon rangini belgilaydi. Ranglar hex, rgb, rgba, hsl, hsla yoki nomi bilan berilishi mumkin.

CSS 
/* background-color qiymatlari */
.bg-hex {
    background-color: #3b82f6;     /* Hex format - eng ko'p ishlatiladi */
}

.bg-rgb {
    background-color: rgb(59, 130, 246);  /* RGB format */
}

.bg-rgba {
    background-color: rgba(59, 130, 246, 0.5);  /* Shaffof fon - 50% opacity */
}

.bg-hsl {
    background-color: hsl(217, 91%, 60%);  /* Hue, Saturation, Lightness */
}

.bg-named {
    background-color: blue;         /* Nom bilan (tavsiya etilmaydi) */
}

.bg-transparent {
    background-color: transparent;  /* Shaffof (default) */
}

/* Amaliy misol */
.hero {
    background-color: #1e293b;
    /* Qolgan stillar... */
}
Senior biladi: rgba() va hsla() dagi "a" alpha (shaffoflik) kanali. 0 - butunlay shaffof, 1 - butunlay shaffof emas. opacity dan farqi - faqat fon rangi shaffof bo'ladi, kontent emas!

background-image - Fon rasmi

background-image xususiyati elementga fon rasmi qo'shish uchun ishlatiladi. url() funksiyasi orqali rasm manzili ko'rsatiladi.

CSS 
/* Bitta fon rasmi */
.hero {
    background-image: url('../img/hero.jpg');
    background-size: cover;
    background-position: center;
}

/* Bir nechta fon rasmi (qatlamlar) */
.multiple-bg {
    background-image: 
        url('../img/overlay.png'),
        url('../img/hero.jpg');
    /* overlay.png yuqorida, hero.jpg pastda ko'rinadi */
}

/* Gradient fon rasmi o'rnida */
.gradient-bg {
    background-image: linear-gradient(to right, #3b82f6, #10b981);
}

/* Rasm va gradient birgalikda */
.hero-overlay {
    background-image: 
        linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)),
        url('../img/hero.jpg');
    background-size: cover;
    background-position: center;
}
Senior texnikasi - Overlay: Rasm ustiga shaffof qatlam qo'shish uchun linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)) ni rasm bilan birga ishlating. Bu matn o'qilishini yaxshilaydi.

background-repeat - Takrorlanish

background-repeat xususiyati fon rasmining takrorlanishini boshqaradi. Default holatda fon rasmi gorizontal va vertikal takrorlanadi.

CSS 
/* background-repeat qiymatlari */
.repeat-default {
    background-repeat: repeat;      /* Ikkala yo'nalishda (default) */
}

.repeat-x {
    background-repeat: repeat-x;    /* Faqat gorizontal */
}

.repeat-y {
    background-repeat: repeat-y;    /* Faqat vertikal */
}

.no-repeat {
    background-repeat: no-repeat;   /* Takrorlanmasin */
}

.space {
    background-repeat: space;       /* Teng oraliq bilan takrorlash */
}

.round {
    background-repeat: round;       /* Rasm o'lchamini moslab takrorlash */
}
Diqqat: background-size: cover yoki contain ishlatilganda background-repeat ko'pincha no-repeat bo'ladi.

background-size - O'lcham

background-size xususiyati fon rasmining o'lchamini belgilaydi. Eng ko'p ishlatiladigan qiymatlar cover va contain.

CSS 
/* background-size qiymatlari */
.bg-cover {
    background-size: cover;     /* Butun konteynerni qoplaydi (kesilishi mumkin) */
}

.bg-contain {
    background-size: contain;   /* Rasm to'liq ko'rinadi (bo'sh joy qolishi mumkin) */
}

.bg-auto {
    background-size: auto;      /* Asl o'lchami (default) */
}

.bg-custom {
    background-size: 300px 200px;  /* Kenglik 300px, Balandlik 200px */
}

.bg-percent {
    background-size: 100% 100%;    /* Konteyner o'lchamida cho'ziladi */
}

.bg-cover-specific {
    background-size: 100% auto;    /* Kenglik 100%, balandlik avtomatik */
}
QiymatTavsifQachon ishlatiladi
coverKonteynerni to'liq qoplaydi, nisbat saqlanadiHero section, banner
containRasm to'liq ko'rinadi, nisbat saqlanadiLogotip, ikonka
autoAsl o'lchamKichik pattern rasmlar

background-position - Joylashuv

background-position xususiyati fon rasmining joylashuvini belgilaydi. Kalit so'zlar, foiz yoki aniq o'lchamlar bilan berilishi mumkin.

CSS 
/* background-position qiymatlari */
.bg-center {
    background-position: center;        /* Markaz */
}

.bg-top {
    background-position: top;           /* Yuqori markaz */
}

.bg-top-right {
    background-position: top right;     /* Yuqori o'ng */
}

.bg-custom {
    background-position: 20% 80%;       /* Chapdan 20%, Yuqoridan 80% */
}

.bg-pixel {
    background-position: 50px 100px;    /* Chapdan 50px, Yuqoridan 100px */
}

.bg-bottom-left {
    background-position: bottom left;   /* Pastki chap */
}

/* Eng ko'p ishlatiladigan kombinatsiya */
.hero {
    background-image: url('../img/hero.jpg');
    background-size: cover;
    background-position: center;        /* Rasm markazda */
}

background-attachment - Scroll effekti

background-attachment xususiyati fon rasmining scroll bo'lganda qanday harakatlanishini belgilaydi.

CSS 
/* background-attachment qiymatlari */
.bg-scroll {
    background-attachment: scroll;  /* Sahifa bilan birga scroll (default) */
}

.bg-fixed {
    background-attachment: fixed;   /* Rasm joyida qoladi - parallax effekti */
}

.bg-local {
    background-attachment: local;   /* Element kontenti bilan scroll */
}

/* Parallax effekti uchun */
.parallax {
    background-image: url('../img/hero.jpg');
    background-size: cover;
    background-position: center;
    background-attachment: fixed;   /* Rasm joyida qoladi */
    height: 100vh;
}
Senior ogohlantirish: background-attachment: fixed mobil qurilmalarda muammo tug'dirishi mumkin (performance va render). Mobil uchun scroll ga o'zgartirishni unutmang.

Background Shorthand

background shorthand xususiyati barcha fon xususiyatlarini bitta qatorda yozish imkonini beradi.

CSS 
/* Background shorthand sintaksisi */
/*
    background: 
        [background-color] 
        [background-image] 
        [background-repeat] 
        [background-attachment] 
        [background-position] / [background-size];
*/

/* Oddiy misol */
.bg-shorthand {
    background: #3b82f6 url('../img/pattern.png') repeat-x fixed center / cover;
}

/* Hero section uchun professional shorthand */
.hero {
    background: 
        linear-gradient(rgba(0,0,0,0.4), rgba(0,0,0,0.4)),
        url('../img/hero.jpg') 
        no-repeat 
        center / cover;
    height: 100vh;
}

/* Alohida xususiyatlar bilan solishtirish */
.hero-long {
    background-image: url('../img/hero.jpg');
    background-repeat: no-repeat;
    background-position: center;
    background-size: cover;
    /* Shorthand bilan bir xil natija, lekin 4 qator kod */
}
Shorthand da / belgisi: background-size ni background-position bilan birga ishlatganda / belgisi ajratib turadi. Masalan: center / cover.

Amaliy mashq (Senior daraja)

Vazifa: Background xususiyatlaridan professional darajada foydalanib hero section yarating.

  1. Hero section uchun fon rasmi qo'shing (background-image)
  2. Rasm ustiga shaffof gradient overlay qo'shing
  3. background-size: cover va background-position: center bering
  4. Parallax effekti uchun background-attachment: fixed qo'shing
  5. Barchasini background shorthand da yozing
  6. Mobil qurilmalar uchun background-attachment: scroll ga o'zgartiring

Senior darajadagi yechim:

/* Hero section - Professional yondashuv */
.hero {
    /* Shorthand - barchasi bir qatorda */
    background: 
        linear-gradient(rgba(0, 0, 0, 0.5), rgba(0, 0, 0, 0.5)),
        url('../img/hero.jpg') 
        no-repeat 
        fixed 
        center / cover;
    
    height: 100vh;
    display: flex;
    align-items: center;
    justify-content: center;
    color: white;
    text-align: center;
}

/* Mobil qurilmalar uchun */
@media (max-width: 768px) {
    .hero {
        /* Mobil da fixed muammo tug'diradi */
        background-attachment: scroll;
        
        /* Yoki butunlay yangi shorthand */
        background: 
            linear-gradient(rgba(0, 0, 0, 0.6), rgba(0, 0, 0, 0.6)),
            url('../img/hero-mobile.jpg') 
            no-repeat 
            scroll 
            center / cover;
    }
}

/* Pattern background uchun */
.pattern-section {
    background: #f8fafc url('../img/pattern.png') repeat;
}

/* Ikki xil fon rasmi (qatlamlar) */
.double-bg {
    background: 
        url('../img/overlay.png') repeat-x top,
        linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}

Dars xulosasi (Senior bilishi kerak)

  • background-color - rgba() shaffof fon uchun, opacity dan afzal
  • background-image - url() va linear-gradient() birgalikda ishlatish
  • background-size - cover (hero), contain (logotip)
  • background-attachment: fixed - parallax effekti, lekin mobil da ehtiyot bo'ling
  • background shorthand - / bilan position/size ajratiladi
  • Performance - katta rasmlarni optimizatsiya qiling, SVG gradientlardan foydalaning
  • Keyingi darsda: Styling images
