Eslatma!

Bu dars oxirida 1-5 darslar bo'yicha 5 ta "test"dan iborat nazorat ishi mavjud. Test barcha o'tilgan mavzular bo'yicha bo'ladi.

Bu darsda nimalarni o'rganasiz?

CSS da izohlar (comments) qanday yoziladi, ular nima uchun kerak va professional dasturchilar izohlardan qanday foydalanadi. Shuningdek, 1-5 darslar bo'yicha nazorat ishi orqali bilimingizni mustahkamlaysiz.

  • CSS da izohlar (comments) nima?
  • Izohlar qanday yoziladi? (/* */ sintaksisi)
  • Izohlar nima uchun kerak?

  • 1-5 darslar bo'yicha nazorat ishi

Bu dars uchun maslahatlar

Izoh yozishni odat qiling: Professional dasturchilar har doim kodlariga izoh qo'shadilar. Bu sizga va jamoangizga kodni tushunishga yordam beradi. odat qiling

Nazorat ishiga tayyorlaning: 1-5 darslarni yaxshilab takrorlang. Nazorat ishi barcha mavzularni qamrab oladi. muhim

Izohlar nima?

Izohlar (comments) - bu CSS kodida yoziladigan, lekin brauzer tomonidan e'tiborga olinmaydigan matnlar. Ular faqat dasturchilar uchun mo'ljallangan.

CSS 
/* Bu izoh - brauzer buni o'qimaydi */
/* Izohlar bir nechta qatorga 
   yozilishi ham mumkin */

h1 {
    color: blue;  /* Matn rangini ko'k qilish */
    font-size: 32px;  /* Shrift o'lchami */
}

Izoh yozish sintaksisi

CSS da izohlar /* va */ belgilari orasiga yoziladi. Izohlar bir yoki bir nechta qator bo'lishi mumkin.

CSS 
/* ============================================
   HEADER STILLARI
   ============================================ */

.header {
    background-color: #1e293b;  /* To'q ko'k fon */
    padding: 20px 40px;         /* Ichki bo'shliq */
}

/* Navigatsiya menyusi */
.nav {
    display: flex;
    gap: 20px;
}

.nav a {
    color: white;
    text-decoration: none;
}

/* 
.nav a:hover {
    color: #3b82f6;  
}
*/
Diqqat: CSS da // bir qatorli izoh ishlamaydi! Bu JavaScript yoki PHP da ishlaydi, lekin CSS da ishlamaydi. CSS da faqat /* */ ishlatiladi.

Yaxshi izoh yozish qoidalari

1
Kerakli joyda izoh yozing

Har bir qatorga izoh yozish shart emas. Faqat murakkab yoki muhim joylarga izoh qo'shing.

2
Aniq va qisqa bo'ling

Izohlar tushunarli va qisqa bo'lishi kerak. "Bu yerda nima bo'lyapti?" savoliga javob berishi kerak.

3
Bo'lim sarlavhalari yozing

Katta CSS fayllarni mantiqiy bo'limlarga ajratib, har bir bo'limga sarlavha qo'ying.

4
Eski kodni o'chiring

Izohga olingan eski kodlarni uzoq vaqt saqlamang. Kerak bo'lsa, versiya nazorat tizimidan (Git) foydalaning.

Nazorat ishi

5 ta test

Bu nazorat ishi 1-darsdan 5-darsgacha o'tilgan barcha mavzularni qamrab oladi. Nazorat ishi 5 ta savoldan iborat bo'lib, CSS asoslari va selektorlar mavzularini o'z ichiga oladi.

1 CSS ning to'liq nomi nima? (1-dars)
2 CSS3 qachondan boshlab faol rivojlana boshlagan? (2-dars)
3 CSS ni HTML ga ulashning eng tavsiya etilgan usuli qaysi? (3-dars)
4 Class selektor qaysi belgi bilan boshlanadi? (4-dars)
5 CSS da izoh (comment) qanday yoziladi? (5-dars)

Amaliy mashq

Vazifa: CSS faylingizga izohlar qo'shing va 1-5 darslarda o'rganganlaringizni takrorlang.

  1. style.css fayl yarating
  2. Fayl boshiga loyiha haqida izoh yozing (sana, muallif, tavsif)
  3. CSS kodlarini mantiqiy bo'limlarga ajrating (Header, Main, Footer)
  4. Har bir bo'limga sarlavha izoh qo'shing
  5. Murakkab stillar yoniga tushuntirish izohi qo'shing
