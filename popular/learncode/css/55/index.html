Flexbox nima?

Flexbox (Flexible Box Layout) - bu elementlarni qator yoki ustun shaklida joylashtirish, ular orasidagi bo'shliqni avtomatik taqsimlash va ularni turli ekran o'lchamlariga moslash uchun yaratilgan CSS moduli. Float yoki positioning ga qaraganda ancha qulay va zamonaviy.

CSS
/* Flexbox konteyner */
.container {
    display: flex;  /* Bu konteyner flex konteynerga aylanadi */
}

/* Barcha bevosita bola elementlar avtomatik flex item bo'ladi */
.container > * {
    /* Flex itemlar */
}
Flexbox terminlari

Flex Container - display: flex berilgan ota element

Flex Items - flex container ichidagi bevosita bola elementlar

Main Axis - asosiy o'q (default: chapdan o'ngga)

Cross Axis - ko'ndalang o'q (default: yuqoridan pastga)

Amaliy mashq

Vazifa: Flexbox yordamida oddiy navigatsiya menyusi yarating.

  1. Konteynerga display: flex bering
  2. 5 ta link elementini joylashtiring
  3. Elementlar orasidagi bo'shliqni gap orqali bering

Namunaviy yechim:

.nav { display: flex; gap: 20px; list-style: none; padding: 0; }
.nav a { text-decoration: none; color: #333; padding: 10px 15px; }

Dars xulosasi

  • display: flex - elementni flex konteynerga aylantiradi
  • Flex Container - display: flex berilgan ota element
  • Flex Items - konteyner ichidagi bevosita bola elementlar
  • Main Axis - asosiy o'q (default: gorizontal)
  • Cross Axis - ko'ndalang o'q (default: vertikal)
