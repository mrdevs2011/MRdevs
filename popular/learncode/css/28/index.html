Gradient va Filterlar

4-fevral, 2021 Background va Rasmlar O'rta daraja
Bu darsda nimalarni o'rganasiz?

CSS da linear-gradient, radial-gradient, conic-gradient va filter xususiyatlarini chuqur o'rganasiz. Gradientlar orqali rangsiz o'tishlar (transition), filterlar orqali esa turli vizual effektlar yaratishni o'rganasiz. Bu bilimlar zamonaviy web dizaynning ajralmas qismidir.

Bu darsda nimalarni o'rganasiz?

  • linear-gradient - chiziqli gradient (yo'nalishli)
  • radial-gradient - radial gradient (markazdan tarqaluvchi)
  • conic-gradient - konus shaklidagi gradient
  • repeating-linear-gradient - takrorlanuvchi gradient
  • filter - blur, brightness, contrast, grayscale, sepia va boshqalar
  • backdrop-filter - element orqasidagi kontentga filter qo'llash

Bu dars uchun Senior maslahatlar

Gradient - bu rasm: CSS da gradientlar background-image sifatida ishlatiladi, background-color emas! Senior yodlaydi
Performance: Murakkab gradientlar va filterlar GPU yuklamasini oshiradi. Iloji boricha oddiy gradientlardan foydalaning. Optimizatsiya
backdrop-filter: Element orqasidagi kontentni xiralashtirish uchun. Glassmorphism dizayn uchun asosiy vosita. Zamonaviy dizayn
CSS Gradient Generator: Murakkab gradientlarni qo'lda yozish qiyin. Onlayn generatorlardan foydalaning. Vaqt tejash

linear-gradient - Chiziqli gradient

linear-gradient - bir yoki bir nechta ranglar orasidagi silliq o'tish. Eng ko'p ishlatiladigan gradient turi.

CSS 
/* linear-gradient sintaksisi */
/*
    linear-gradient([yo'nalish], rang1 [stop1], rang2 [stop2], ...)
    yo'nalish: to right, to bottom, 45deg, 0.25turn
*/

/* Asosiy gradientlar */
.gradient-1 {
    background-image: linear-gradient(to right, #3b82f6, #10b981);
    /* Chapdan o'ngga: ko'k → yashil */
}

.gradient-2 {
    background-image: linear-gradient(to bottom, #f59e0b, #ef4444);
    /* Yuqoridan pastga: sariq → qizil */
}

.gradient-3 {
    background-image: linear-gradient(45deg, #667eea, #764ba2);
    /* 45 gradus burchak ostida */
}

/* Bir nechta rang */
.gradient-4 {
    background-image: linear-gradient(to right, #ff0000, #ffff00, #00ff00, #00ffff, #0000ff);
    /* Ko'p rangli gradient */
}

/* Rang stoplari (to'xtash nuqtalari) */
.gradient-5 {
    background-image: linear-gradient(to right, #3b82f6 0%, #10b981 50%, #f59e0b 100%);
    /* 0% da ko'k, 50% da yashil, 100% da sariq */
}

/* Qattiq o'tish (stripes) */
.gradient-6 {
    background-image: linear-gradient(to right, #3b82f6 0%, #3b82f6 40%, #10b981 40%, #10b981 100%);
    /* Ko'k va yashil chiziqlar (stripes) */
}

radial-gradient - Radial gradient

radial-gradient - markazdan tashqariga tarqaluvchi gradient. Doira yoki ellips shaklida bo'lishi mumkin.

CSS 
/* radial-gradient sintaksisi */
/*
    radial-gradient([shakl] [o'lcham] at [joylashuv], rang1, rang2, ...)
*/

/* Asosiy radial gradient */
.radial-1 {
    background-image: radial-gradient(circle, #3b82f6, #1e3a5f);
    /* Doira shaklida, markazdan tashqariga */
}

.radial-2 {
    background-image: radial-gradient(ellipse, #f59e0b, #ef4444);
    /* Ellips shaklida (default) */
}

/* Joylashuv bilan */
.radial-3 {
    background-image: radial-gradient(circle at top left, #10b981, #064e3b);
    /* Yuqori chap burchakdan boshlanadi */
}

.radial-4 {
    background-image: radial-gradient(circle at 20% 80%, #667eea, #764ba2);
    /* Aniq koordinatadan (20% chapdan, 80% yuqoridan) */
}

/* O'lcham bilan */
.radial-5 {
    background-image: radial-gradient(circle closest-side, #3b82f6, #1e3a5f);
    /* closest-side, farthest-side, closest-corner, farthest-corner */
}

/* Bir nechta rang */
.radial-6 {
    background-image: radial-gradient(circle, #ff0000, #ffff00, #00ff00);
}

conic-gradient - Konus gradient

conic-gradient - markaz atrofida aylanuvchi gradient. Diagramma va loaderlar yaratish uchun juda foydali.

CSS 
/* conic-gradient sintaksisi */
/*
    conic-gradient([from burchak] [at joylashuv], rang1, rang2, ...)
*/

/* Asosiy konus gradient */
.conic-1 {
    background-image: conic-gradient(#3b82f6, #10b981, #f59e0b, #ef4444, #3b82f6);
    /* 360 gradus bo'ylab ranglar */
}

/* Boshlang'ich burchak bilan */
.conic-2 {
    background-image: conic-gradient(from 45deg, #3b82f6, #10b981, #f59e0b);
    /* 45 gradusdan boshlanadi */
}

/* Joylashuv bilan */
.conic-3 {
    background-image: conic-gradient(at 30% 70%, #3b82f6, #10b981, #f59e0b);
    /* Markaz 30% 70% nuqtada */
}

/* Pie chart (diagramma) */
.conic-pie {
    background-image: conic-gradient(
        #3b82f6 0deg 120deg,   /* 0-120 gradus: ko'k */
        #10b981 120deg 240deg, /* 120-240 gradus: yashil */
        #f59e0b 240deg 360deg  /* 240-360 gradus: sariq */
    );
    border-radius: 50%;
}

Takrorlanuvchi gradientlar

repeating-linear-gradient va repeating-radial-gradient - gradient patternlarini takrorlash uchun ishlatiladi.

CSS 
/* Takrorlanuvchi chiziqli gradient */
.repeating-linear {
    background-image: repeating-linear-gradient(
        45deg,
        #3b82f6 0px,
        #3b82f6 10px,
        #10b981 10px,
        #10b981 20px
    );
    /* 10px ko'k, 10px yashil chiziqlar takrorlanadi */
}

/* Takrorlanuvchi radial gradient */
.repeating-radial {
    background-image: repeating-radial-gradient(
        circle at center,
        #3b82f6 0px,
        #3b82f6 10px,
        #10b981 10px,
        #10b981 20px
    );
    /* Doira shaklida takrorlanuvchi halqalar */
}

/* Takrorlanuvchi konus gradient */
.repeating-conic {
    background-image: repeating-conic-gradient(
        from 0deg,
        #3b82f6 0deg 30deg,
        #10b981 30deg 60deg
    );
    border-radius: 50%;
}

filter - Vizual effektlar

filter xususiyati elementga (rasm, div, matn) turli xil vizual effektlar berish uchun ishlatiladi.

CSS 
/* filter funksiyalari */
.filter-blur {
    filter: blur(5px);           /* Xiralashtirish */
}

.filter-brightness {
    filter: brightness(1.5);     /* Yorqinlik (1 = normal, >1 yorqin, <1 qorong'i) */
}

.filter-contrast {
    filter: contrast(200%);      /* Kontrast (100% = normal) */
}

.filter-grayscale {
    filter: grayscale(100%);     /* Oq-qora (100% = to'liq oq-qora) */
}

.filter-sepia {
    filter: sepia(100%);         /* Sepia (eski rasm) effekti */
}

.filter-hue-rotate {
    filter: hue-rotate(90deg);   /* Rang o'zgartirish (0-360deg) */
}

.filter-invert {
    filter: invert(100%);        /* Ranglarni teskarilash */
}

.filter-saturate {
    filter: saturate(200%);      /* To'yinganlik (100% = normal) */
}

.filter-opacity {
    filter: opacity(50%);        /* Shaffoflik (100% = normal) */
}

.filter-drop-shadow {
    filter: drop-shadow(4px 4px 8px rgba(0,0,0,0.3)); /* Soya (box-shadow ga o'xshash) */
}

/* Bir nechta filter birgalikda */
.filter-combined {
    filter: grayscale(50%) blur(2px) brightness(1.1);
}

backdrop-filter - Glassmorphism

backdrop-filter xususiyati element orqasidagi kontentga filter qo'llash uchun ishlatiladi. Zamonaviy "glassmorphism" dizayn uchun asosiy vosita.

CSS 
/* backdrop-filter - Glassmorphism */
.glass-card {
    background: rgba(255, 255, 255, 0.2);
    backdrop-filter: blur(10px);
    -webkit-backdrop-filter: blur(10px); /* Safari uchun */
    border-radius: 16px;
    border: 1px solid rgba(255, 255, 255, 0.3);
    padding: 20px;
}

/* Bir nechta effekt */
.glass-combined {
    background: rgba(0, 0, 0, 0.3);
    backdrop-filter: blur(8px) brightness(1.2);
    border-radius: 12px;
    padding: 20px;
}

/* Navigatsiya menyusi uchun */
.glass-nav {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    background: rgba(255, 255, 255, 0.8);
    backdrop-filter: blur(8px);
    padding: 15px 0;
    z-index: 100;
}
backdrop-filter vs filter: filter elementning o'ziga ta'sir qiladi. backdrop-filter element ORQASIDAGI kontentga ta'sir qiladi. Shuning uchun glassmorphism effektlar uchun backdrop-filter ishlatiladi.

Amaliy mashq (Senior daraja)

Vazifa: Gradient va filterlardan foydalanib zamonaviy UI komponentlarini yarating.

  1. Hero section uchun linear-gradient overlay yarating
  2. Glassmorphism karta yarating (backdrop-filter: blur())
  3. Hover bo'lganda rasmga grayscale() va brightness() filter qo'llang
  4. conic-gradient yordamida progress loader yarating
  5. Takrorlanuvchi gradient bilan pattern background yarating

Senior darajadagi yechim:

/* 1. Hero section overlay */
.hero {
    background-image: 
        linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.3)),
        url('../img/hero.jpg');
    background-size: cover;
    background-position: center;
    height: 100vh;
}

/* 2. Glassmorphism karta */
.glass-card {
    background: rgba(255, 255, 255, 0.15);
    backdrop-filter: blur(12px);
    -webkit-backdrop-filter: blur(12px);
    border-radius: 20px;
    border: 1px solid rgba(255, 255, 255, 0.2);
    padding: 30px;
    box-shadow: 0 8px 32px rgba(0, 0, 0, 0.1);
}

/* 3. Hover filter effekti */
.hover-filter {
    transition: filter 0.4s ease;
}
.hover-filter:hover {
    filter: grayscale(100%) brightness(1.1);
}

/* 4. Conic gradient loader */
.loader {
    width: 80px;
    height: 80px;
    border-radius: 50%;
    background: conic-gradient(#3b82f6 0deg 120deg, #10b981 120deg 240deg, #f59e0b 240deg 360deg);
    animation: spin 2s linear infinite;
}
@keyframes spin {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
}

/* 5. Pattern background */
.pattern-bg {
    background-image: repeating-linear-gradient(
        45deg,
        rgba(59, 130, 246, 0.1) 0px,
        rgba(59, 130, 246, 0.1) 10px,
        transparent 10px,
        transparent 20px
    );
}
