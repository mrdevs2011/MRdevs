Breakpoints

17-fevral, 2021 RWD va Fontlar O'rta daraja
Bu darsda nimalarni o'rganasiz?

Responsive dizaynda breakpoints (uzilish nuqtalari) nima, ularni qanday tanlash kerak va qurilma o'lchamlariga emas, kontentga qarab breakpoint belgilash muhimligini o'rganasiz.

Bu dars uchun Senior maslahatlar

Kontent uchun breakpoint: Qurilma o'lchamiga emas, dizayningiz buziladigan nuqtaga qarab breakpoint qo'shing. Senior yodlaydi
Standart breakpointlar: 600px (planshet), 900px (desktop), 1200px (katta desktop) - boshlang'ich nuqta. Professional
em/rem breakpointlarda: Media query da em ishlating - foydalanuvchi shrift o'lchamini o'zgartirganda moslashadi. Best practice
CSS custom properties: Breakpointlarni o'zgaruvchilarda saqlang - kodni toza va oson boshqariladigan qiladi. Zamonaviy

Breakpoint nima?

Breakpoint - bu media query da ishlatiladigan, dizayn o'zgaradigan ekran o'lchami nuqtasi.

CSS
/* Breakpoint - ekran o'lchami 600px bo'lganda */
@media (min-width: 600px) {
    /* 600px va undan katta ekranlar uchun stillar */
    .container {
        width: 90%;
    }
}

/* Breakpoint - ekran o'lchami 900px bo'lganda */
@media (min-width: 900px) {
    /* 900px va undan katta ekranlar uchun stillar */
    .container {
        width: 80%;
        max-width: 1200px;
    }
}

Kontent uchun breakpoint

Qurilma o'lchamiga emas, kontentingiz buziladigan nuqtada breakpoint qo'shing.

CSS
/* Noto'g'ri - qurilma uchun */
@media (min-width: 768px) { }  /* iPad uchun */
@media (min-width: 1024px) { } /* iPad Pro uchun */

/* To'g'ri - kontent uchun */
@media (min-width: 540px) {
    /* Bu yerda 3 ta karta bir qatorga sig'may qolgan */
    .card-container {
        grid-template-columns: repeat(2, 1fr);
    }
}

@media (min-width: 800px) {
    /* Bu yerda 2 ta karta uchun joy ko'p bo'lib qolgan */
    .card-container {
        grid-template-columns: repeat(3, 1fr);
    }
}

Mobile-first breakpointlar

Mobile-first yondashuvda default stillar mobil uchun, breakpointlar esa min-width bilan.

CSS
/* Mobile-first - default mobil */
.container { width: 100%; padding: 1.5rem; }
h1 { font-size: 2.8rem; }
.grid { display: block; }

/* 600px - katta mobil / kichik planshet */
@media (min-width: 37.5em) {
    .container { width: 90%; padding: 2rem; }
    h1 { font-size: 3.6rem; }
    .grid { display: grid; grid-template-columns: repeat(2, 1fr); gap: 2rem; }
}

/* 900px - planshet / desktop */
@media (min-width: 56.25em) {
    .container { width: 85%; padding: 3rem; }
    h1 { font-size: 4.8rem; }
    .grid { grid-template-columns: repeat(3, 1fr); }
}

/* 1200px - katta desktop */
@media (min-width: 75em) {
    .container { max-width: 120rem; }
    .grid { grid-template-columns: repeat(4, 1fr); }
}

Breakpoint tanlash qoidalari

1
Kontentingizni kuzating

Ekran o'lchamini kichiklashtirib boring, dizayn buziladigan nuqtada breakpoint qo'shing.

2
Kamida 2-3 ta breakpoint

Mobil, planshet, desktop - asosiy uchlik.

3
em birligidan foydalaning

Media query da em ishlating, px emas.

4
CSS o'zgaruvchilarida saqlang

Breakpointlarni :root da saqlab, qayta ishlating.
