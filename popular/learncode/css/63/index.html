Bu darsda butun kurs davomida o'rgangan bilimlaringizni yakunlaymiz. Natour loyihasini to'liq ko'rib chiqamiz, kelajakda qaysi mavzularni takrorlash kerakligi haqida maslahatlar olasiz va keyingi o'rganish yo'lingizni belgilab olamiz.

Natour loyihasi - nimalarni o'rgandik?

Natour - bu kurs davomida yaratgan to'liq ishlaydigan, zamonaviy va responsive veb-sayt. Bu loyiha orqali quyidagi bilimlarni amaliyotda qo'lladingiz:

CSS Asoslari

Selektorlar, specificity, inheritance, combinators - barchasi loyihada ishlatilgan.

Box Model

Margin, padding, border, width, height - har bir elementda qo'llangan.

Positioning

Absolute, relative, fixed, sticky - elementlarni aniq joylashtirish.

Background va Rasmlar

Gradientlar, overlay, object-fit, filterlar - vizual effektlar.

Units va RWD

rem, em, vh, vw, media queries - barcha qurilmalarga moslashish.

Advanced CSS

Transforms, transitions, animations - interaktivlik va jonlilik.

Flexbox

Zamonaviy layout tizimi - elementlarni joylashtirish.

CSS Grid

Ikki o'lchovli layout - murakkab sahifa tuzilmalari.

Qaysi mavzularga e'tibor berish kerak?

CSS juda keng va ba'zi mavzular boshqalariga qaraganda murakkabroq. Quyida qaysi mavzularni ko'proq takrorlash kerakligi haqida maslahatlar:

Positioning (absolute, relative, fixed, sticky)

Bu eng ko'p xato qilinadigan mavzu. Ayniqsa absolute va relative farqini, qaysi element nimaga nisbatan joylashishini yaxshilab tushunib oling.

Yuqori ustuvorlik
z-index va stacking context

Elementlar ustma-ust tushganda qaysi biri oldinda bo'lishini boshqarish. Stacking context tushunchasini to'liq tushunish muhim.

Yuqori ustuvorlik
Flexbox va Grid birgalikda

Qachon Flexbox, qachon Grid ishlatish kerakligini bilish. Ko'pchilik bu ikkisini aralashtirib yuboradi.

Yuqori ustuvorlik
Animations va @keyframes

Murakkab animatsiyalar yozish. fill-mode, iteration-count, direction kabi xususiyatlarni to'liq o'zlashtirish.

O'rta ustuvorlik
CSS o'zgaruvchilari (Custom Properties)

Theming, dark mode, design tokens - zamonaviy loyihalarda juda muhim.

O'rta ustuvorlik
Pseudo-classes va pseudo-elements

::before, ::after, :nth-child, :not() - bularni yaxshi bilsangiz, kod ancha qisqaradi.

Past ustuvorlik
CSS best practices (BEM, ITCSS)

Katta loyihalarda CSS ni boshqarish metodologiyalari. Jamoada ishlaganda juda muhim.
