Height (balandlik), Width (kenglik). Max / Min - height / width

21-yanvar, 2021 Box Model Boshlang'ich daraja
Bu darsda nimalarni o'rganasiz?

CSS da elementlarning kengligi (width) va balandligini (height) qanday boshqarishni, shuningdek, ularning minimal va maksimal chegaralarini (min-width, max-width, min-height, max-height) qanday belgilashni o'rganasiz. Bu responsive dizayn uchun juda muhim.

Bu darsda nimalarni o'rganasiz?

  • width - element kengligini belgilash
  • height - element balandligini belgilash
  • min-width va max-width - kenglik chegaralari
  • min-height va max-height - balandlik chegaralari
  • auto qiymati va foizlar bilan ishlash

Bu dars uchun maslahatlar

max-width muhim: Responsive dizayn uchun max-width: 100% juda ko'p ishlatiladi. Bu element ota konteynerdan chiqib ketmasligini ta'minlaydi. yodlash shart
min-height foydali: min-height: 100vh yordamida element kamida ekran balandligicha bo'lishini ta'minlash mumkin. muhim
Foizlar ota elementga nisbatan: width: 50% ota element kengligining yarmini oladi. amaliyot

Width va Height

width va height xususiyatlari elementning kontent qismining kengligi va balandligini belgilaydi.

CSS 
.box {
    width: 300px;      /* Kenglik 300 piksel */
    height: 200px;     /* Balandlik 200 piksel */
    background: #3b82f6;
    color: white;
    padding: 20px;
}

/* Foizlarda */
.container {
    width: 80%;        /* Ota elementning 80% */
    margin: 0 auto;    /* Markazga tekislash */
}

/* Auto - avtomatik hisoblash */
.auto-box {
    width: auto;       /* Brauzer avtomatik hisoblaydi */
    height: auto;      /* Kontentga qarab */
}
Eslatma: width va height faqat kontent qismiga ta'sir qiladi. Padding, border va margin qo'shimcha joy egallaydi.

min-width va max-width

Element kengligining minimal va maksimal chegaralarini belgilash uchun ishlatiladi. Bu ayniqsa responsive dizaynda juda foydali.

CSS 
/* Minimal kenglik - bundan kichik bo'la olmaydi */
.card {
    min-width: 250px;
    width: 100%;
}

/* Maksimal kenglik - bundan katta bo'la olmaydi */
.container {
    max-width: 1200px;
    width: 90%;
    margin: 0 auto;
}

/* Responsive rasm */
img {
    max-width: 100%;   /* Ota elementdan katta bo'lmaydi */
    height: auto;      /* Nisbat saqlanadi */
}

/* Birgalikda ishlatish */
.responsive-box {
    min-width: 300px;
    max-width: 800px;
    width: 80%;
}
Eng ko'p ishlatiladigan pattern: max-width: 1200px; width: 90%; margin: 0 auto; - bu konteyner katta ekranlarda 1200px, kichik ekranlarda esa 90% kenglikda bo'ladi.

min-height va max-height

Element balandligining minimal va maksimal chegaralarini belgilash uchun ishlatiladi.

CSS 
/* To'liq ekran balandligi */
.hero {
    min-height: 100vh;  /* Kamida ekran balandligicha */
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}

/* Maksimal balandlik - scroll bilan */
.scrollable-content {
    max-height: 400px;
    overflow-y: auto;   /* Kontent ko'p bo'lsa scroll */
    padding: 20px;
}

/* Birgalikda */
.modal-content {
    min-height: 200px;
    max-height: 80vh;   /* Ekranning 80% dan oshmaydi */
    overflow-y: auto;
}
Diqqat: height va max-height birga ishlatilganda, max-height ustunlik qiladi. Agar height: 500px va max-height: 300px bo'lsa, element 300px bo'ladi.

O'lchov birliklari

Width va height uchun turli xil o'lchov birliklaridan foydalanish mumkin:

Birlik Tavsif Misol
px Piksel - qat'iy o'lcham width: 300px;
% Ota elementga nisbatan foiz width: 50%;
vw Viewport kengligining foizi width: 80vw;
vh Viewport balandligining foizi min-height: 100vh;
em Joriy shrift o'lchamiga nisbatan width: 20em;
rem Root shrift o'lchamiga nisbatan width: 40rem;
auto Avtomatik hisoblash width: auto;

Amaliy misol: Responsive karta

CSS 
/* Responsive karta komponenti */
.card {
    min-width: 280px;
    max-width: 400px;
    width: 100%;
    margin: 20px auto;
    padding: 25px;
    background: white;
    border-radius: 12px;
    box-shadow: 0 4px 6px rgba(0,0,0,0.1);
}

.card img {
    max-width: 100%;
    height: auto;
    border-radius: 8px;
}

.card-content {
    max-height: 200px;
    overflow-y: auto;
    padding-right: 10px;
}

/* Karta konteyneri */
.cards-container {
    max-width: 1200px;
    width: 90%;
    margin: 0 auto;
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
    gap: 30px;
}

Bilimingizni tekshiring

14-dars testi
1 Element ekran balandligicha bo'lishi uchun qaysi xususiyat ishlatiladi?
2 Rasm ota konteynerdan chiqib ketmasligi uchun qaysi xususiyat ishlatiladi?
3 width: 50%; nimaga nisbatan hisoblanadi?
4 Element balandligi kontentga qarab avtomatik o'zgarishi uchun qaysi qiymat ishlatiladi?

Amaliy mashq

Vazifa: Width va height xususiyatlaridan foydalanib responsive karta yarating.

  1. Karta uchun .card klassini yarating
  2. min-width: 250px va max-width: 350px bering
  3. width: 100% bering (ota elementga moslashsin)
  4. Karta ichidagi rasmga max-width: 100% va height: auto bering
  5. Karta kontentiga max-height: 150px va overflow-y: auto bering
  6. Kartalarni display: grid va grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)) bilan joylashtiring

        </div>
    </div>

</div>
