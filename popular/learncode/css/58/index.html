Flexbox: order, flex-grow va flex-shrink

6-mart, 2021 Flexbox O'rta daraja
Bu darsda nimalarni o'rganasiz?

Flexbox elementlarining o'ziga beriladigan xususiyatlarni o'rganamiz: order (tartibni o'zgartirish), flex-grow (bo'sh joyni egallash koeffitsienti) va flex-shrink (sig'ish koeffitsienti). Bu xususiyatlar orqali elementlarning o'lchamlarini va joylashuvini aniq boshqarish mumkin.

Bu darsda nimalarni o'rganasiz?

  • order - elementlarning tartib raqamini o'zgartirish
  • flex-grow - elementning bo'sh joyni egallash darajasi
  • flex-shrink - elementning siqilish darajasi
  • Bu xususiyatlarning responsive dizayndagi ahamiyati

order - Elementlar tartibini o'zgartirish

order xususiyati flex elementlarining ko'rinish tartibini HTML dagi tartibdan qat'i nazar o'zgartiradi. Default qiymat 0. Kichik raqam oldin, katta raqam keyin keladi.

CSS
/* order - tartibni o'zgartirish */
.item-1 { order: 3; }  /* Eng oxirga o'tadi */
.item-2 { order: 1; }  /* Eng birinchi bo'ladi */
.item-3 { order: 2; }  /* O'rtada */
.item-4 { order: 0; }  /* Default */

/* Manfiy qiymatlar ham mumkin */
.first { order: -1; }  /* Hamma narsadan oldin */
.last { order: 999; }  /* Hamma narsadan keyin */
Senior biladi: order faqat ko'rinish tartibini o'zgartiradi, HTML struktura yoki accessibility ga ta'sir qilmaydi. Ekran o'quvchilar HTML tartibida o'qiydi!

flex-grow - Bo'sh joyni egallash

flex-grow xususiyati elementning konteynerdagi bo'sh joyni qanchalik egallashini belgilaydi. Default qiymat 0 (o'smaydi). Qiymat qancha katta bo'lsa, element shuncha ko'p bo'sh joy egallaydi.

CSS
/* flex-grow - bo'sh joyni taqsimlash */
.no-grow { flex-grow: 0; }      /* Default - o'smaydi */
.grow-1 { flex-grow: 1; }       /* Bo'sh joydan 1 hissa oladi */
.grow-2 { flex-grow: 2; }       /* Bo'sh joydan 2 hissa oladi (grow-1 dan 2x ko'p) */

/* Amaliy misol - 3 ta element, o'rtadagisi bo'sh joyni egallasin */
.sidebar { flex-grow: 0; width: 250px; }
.main-content { flex-grow: 1; }  /* Qolgan barcha joyni egallaydi */

flex-shrink - Siqilish darajasi

flex-shrink xususiyati konteyner kichiklashganda elementning qanchalik siqilishini belgilaydi. Default qiymat 1 (siqiladi). 0 berilsa, element siqilmaydi.

CSS
/* flex-shrink - siqilish darajasi */
.shrink-normal { flex-shrink: 1; }  /* Default - siqiladi */
.no-shrink { flex-shrink: 0; }      /* Siqilmaydi - o'z kengligini saqlaydi */
.shrink-more { flex-shrink: 3; }    /* Boshqalardan 3x ko'proq siqiladi */

Amaliy mashq

Vazifa: flex-grow va order dan foydalanib layout yarating.

  1. 3 ustunli layout: sidebar (250px), main (flex-grow: 1), sidebar (250px)
  2. Mobil versiyada order orqali main kontentni birinchi o'ringa o'tkazing

Namunaviy yechim:

.container { display: flex; }
.sidebar { width: 250px; }
.main { flex-grow: 1; }
@media (max-width: 768px) {
    .container { flex-direction: column; }
    .main { order: -1; }
}

Dars xulosasi

  • order - tartibni o'zgartiradi (default: 0, kichik raqam oldin)
  • flex-grow - bo'sh joyni egallash (default: 0)
  • flex-shrink - siqilish darajasi (default: 1, 0 = siqilmaydi)
  • Keyingi darsda: flex-basis, flex va align-self
