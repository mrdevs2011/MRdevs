Fonts (Shriftlar)

20-fevral, 2021 Fontlar va Transforms Boshlang'ich daraja
Bu darsda nimalarni o'rganasiz?

CSS da shriftlar (fonts) bilan ishlashni o'rganasiz. font-family, font-size, font-weight, font-style, line-height, text-transform, letter-spacing va boshqa shrift xususiyatlarini batafsil o'rganamiz.

Bu darsda nimalarni o'rganasiz?

  • font-family - shrift turini belgilash
  • font-size - shrift o'lchami (px, rem, em)
  • font-weight - shrift qalinligi (100-900, bold, normal)
  • font-style - shrift uslubi (normal, italic, oblique)
  • line-height - qator balandligi
  • text-transform - matn transformatsiyasi
  • letter-spacing - harflar orasidagi masofa

Bu dars uchun Senior maslahatlar

Fallback fontlar: Har doim zaxira shriftlarni ko'rsating. Masalan: font-family: "Lato", "Arial", sans-serif; Senior yodlaydi
rem font-size: px o'rniga rem ishlating - foydalanuvchi shrift sozlamalariga moslashadi. Accessibility
Unitless line-height: line-height: 1.5; - foiz yoki px dan afzal, meros bo'lganda muammo tug'dirmaydi. Best practice
System fonts: font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif; - tez va ishonchli. Performance

font-family

font-family xususiyati shrift turini belgilaydi. Bir nechta shrift ko'rsatish mumkin - brauzer birinchisidan boshlab qidiradi.

CSS
/* Asosiy shrift oilalari */
body {
    /* Fallback: agar Lato bo'lmasa, Arial, u ham bo'lmasa sans-serif */
    font-family: "Lato", "Arial", sans-serif;
}

/* Serif shriftlar */
.serif-text {
    font-family: "Times New Roman", Times, serif;
}

/* Monospace shriftlar (kod uchun) */
code {
    font-family: "Fira Code", "Courier New", monospace;
}

/* System fonts - eng tez yuklanadi */
.system-font {
    font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, Oxygen, Ubuntu, sans-serif;
}
Generic oilalar: serif, sans-serif, monospace, cursive, fantasy. Har doim oxirgi variant sifatida generic oilani ko'rsating.

font-size va font-weight

CSS
/* font-size */
body { font-size: 1.6rem; }      /* 16px */
h1 { font-size: 4.8rem; }        /* 48px */
h2 { font-size: 3.6rem; }        /* 36px */
small { font-size: 1.2rem; }     /* 12px */

/* font-weight - shrift qalinligi */
.thin { font-weight: 100; }
.light { font-weight: 300; }
.normal { font-weight: 400; }    /* normal */
.medium { font-weight: 500; }
.bold { font-weight: 700; }      /* bold */
.black { font-weight: 900; }

/* Kalit so'zlar bilan */
.normal-weight { font-weight: normal; }    /* 400 */
.bold-weight { font-weight: bold; }        /* 700 */
.lighter-weight { font-weight: lighter; }  /* Ota elementdan yengilroq */
.bolder-weight { font-weight: bolder; }    /* Ota elementdan qalinroq */

font-style va text-transform

CSS
/* font-style */
.normal-style { font-style: normal; }
.italic-style { font-style: italic; }
.oblique-style { font-style: oblique; }  /* Sun'iy qiyalik */

/* text-transform */
.uppercase { text-transform: uppercase; }   /* BARCHA HARF KATTA */
.lowercase { text-transform: lowercase; }   /* barcha harf kichik */
.capitalize { text-transform: capitalize; } /* Har So'z Bosh Harf */
.none-transform { text-transform: none; }   /* O'zgarmaydi */

line-height va letter-spacing

CSS
/* line-height - unitless tavsiya etiladi */
body { line-height: 1.6; }        /* font-size * 1.6 */
h1 { line-height: 1.2; }
.small-line { line-height: 1.2; } /* Tighter */
.large-line { line-height: 1.8; } /* Looser */

/* letter-spacing - harflar orasidagi masofa */
.normal-spacing { letter-spacing: normal; }
.tight-spacing { letter-spacing: -0.5px; }
.wide-spacing { letter-spacing: 2px; }
.heading-spacing { letter-spacing: 0.5rem; }  /* Sarlavhalar uchun */

/* word-spacing - so'zlar orasidagi masofa */
.normal-word { word-spacing: normal; }
.wide-word { word-spacing: 5px; }
Senior biladi: line-height uchun unitless qiymat (masalan, 1.5) ishlating. Bu meros bo'lganda kutilmagan natijalarni oldini oladi.

Font shorthand

CSS
/* font shorthand sintaksisi */
/* font: font-style font-weight font-size/line-height font-family; */

/* Alohida xususiyatlar */
.text {
    font-style: italic;
    font-weight: 700;
    font-size: 1.6rem;
    line-height: 1.5;
    font-family: "Lato", sans-serif;
}

/* Shorthand - bitta qatorda */
.text-shorthand {
    font: italic 700 1.6rem/1.5 "Lato", sans-serif;
}

/* Minimal shorthand */
.minimal-font {
    font: 1.6rem "Lato", sans-serif;  /* font-size va font-family majburiy */
}
Shorthand tartibi: font-style, font-weight, font-size/line-height, font-family. font-size va font-family majburiy!

Amaliy mashq

Vazifa: Shrift xususiyatlaridan foydalanib typography tizimi yarating.

  1. font-family da system fonts yoki Google Fonts ishlating
  2. Sarlavhalar uchun font-weight, letter-spacing va text-transform bering
  3. Body uchun line-height: 1.6 va font-size: 1.6rem bering
  4. Font shorthand yordamida sarlavha stillarini yozing

Namunaviy yechim:

body { font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif; font-size: 1.6rem; line-height: 1.6; color: #333; }
h1 { font: 700 4.8rem/1.2 "Lato", sans-serif; letter-spacing: -0.5px; margin-bottom: 2rem; }
h2 { font: 600 3.6rem/1.3 "Lato", sans-serif; letter-spacing: -0.3px; margin-bottom: 1.5rem; }
h3 { font: 600 2.4rem/1.4 "Lato", sans-serif; margin-bottom: 1rem; }
.uppercase { text-transform: uppercase; letter-spacing: 1px; }

Dars xulosasi

  • font-family: fallback shriftlar ko'rsatish muhim
  • font-size: rem birligidan foydalaning
  • font-weight: 100-900 yoki normal/bold
  • line-height: unitless qiymat (1.5) eng yaxshi
  • text-transform: uppercase, lowercase, capitalize
  • letter-spacing: sarlavhalar uchun foydali
  • Keyingi darsda: Yangi font qo'shish - Nazorat ishi!
