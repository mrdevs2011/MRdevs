Viewport o'lchov birligi

13-fevral, 2021 Units va RWD O'rta daraja
Bu darsda nimalarni o'rganasiz?

CSS da viewport o'lchov birliklari - vh, vw, vmin, vmax haqida chuqur bilim olasiz. Bu birliklar brauzer oynasi (viewport) o'lchamlariga nisbatan hisoblanadi va to'liq ekranli hero section, responsive typography yaratishda juda qulay.

Bu darsda nimalarni o'rganasiz?

  • vh - viewport balandligi (1vh = viewport balandligining 1%)
  • vw - viewport kengligi (1vw = viewport kengligining 1%)
  • vmin - viewport kichik tomoniga nisbatan
  • vmax - viewport katta tomoniga nisbatan
  • Viewport birliklari bilan responsive dizayn

Bu dars uchun Senior maslahatlar

vh/vw nima? 100vh = to'liq ekran balandligi. Hero section, full-screen overlay uchun ideal. Senior yodlaydi
vmin/vmax: vmin kvadrat elementlar, vmax landshaft rejim uchun. Professional
Mobile viewport: Mobil brauzerlarda manzil satri tufayli 100vh muammo tug'dirishi mumkin. min-height: 100vh ishlating. Muhim
Fluid typography: font-size: calc(1.6rem + 1vw); - ekran o'lchamiga qarab shrift o'zgaradi. Zamonaviy

vh - Viewport Height

vh (viewport height) - viewport balandligiga nisbatan. 1vh = viewport balandligining 1%.

CSS 
/* To'liq ekran hero section */
.hero {
    min-height: 100vh;      /* Kamida ekran balandligicha */
    display: flex;
    align-items: center;
    justify-content: center;
    background: linear-gradient(135deg, #667eea, #764ba2);
    color: white;
}

/* Yarim ekran balandligi */
.half-screen {
    height: 50vh;           /* Ekran balandligining yarmi */
}

/* Mobil muammo uchun yechim */
.full-screen-safe {
    min-height: 100vh;      /* height emas, min-height! */
}

/* Header balandligi */
.header {
    height: 8vh;            /* Ekran balandligining 8% */
}
Mobil muammo: Mobil brauzerlarda manzil satri tufayli height: 100vh kutilgandek ishlamasligi mumkin. Yechim: min-height: 100vh ishlating!

vw - Viewport Width

vw (viewport width) - viewport kengligiga nisbatan. 1vw = viewport kengligining 1%.

CSS 
/* To'liq kenglik */
.full-width {
    width: 100vw;
}

/* Ekran kengligining yarmi */
.half-width {
    width: 50vw;
}

/* Fluid typography - ekran kengligiga qarab o'zgaruvchi shrift */
.fluid-text {
    font-size: calc(1.6rem + 1vw);
    /* 1.6rem = 16px + ekran kengligining 1% */
}

/* Katta sarlavha */
.hero-title {
    font-size: 8vw;         /* Ekran kengligining 8% */
}

/* Container kengligi */
.container {
    width: 90vw;
    max-width: 1200px;
    margin: 0 auto;
}

vmin va vmax

vmin - viewport kichik tomoniga nisbatan. vmax - viewport katta tomoniga nisbatan.

CSS 
/* vmin - har doim kvadrat bo'ladi (kichik tomonga nisbatan) */
.square {
    width: 50vmin;
    height: 50vmin;         /* Har doim kvadrat */
    background: #3b82f6;
}

/* vmin - landshaft va portrait da bir xil ko'rinish */
.responsive-padding {
    padding: 5vmin;         /* Kichik ekranda kichik, katta ekranda katta */
}

/* vmax - katta tomonga nisbatan */
.landscape-element {
    width: 80vmax;          /* Landshaft rejimda kengroq bo'ladi */
    max-width: 100%;
}

/* Modal oyna */
.modal {
    width: 80vmin;          /* Ekranga moslashuvchi kenglik */
    max-width: 600px;
    padding: 4vmin;
}

Fluid Typography (Oquvchan shrift)

calc() va viewport birliklari yordamida ekran o'lchamiga qarab avtomatik o'zgaruvchi shrift yaratish.

CSS 
/* Asosiy formula: calc(min-size + viewport-unit) */
html {
    font-size: 62.5%;       /* 1rem = 10px */
}

/* Fluid shrift - ekranga qarab o'zgaradi */
body {
    font-size: calc(1.6rem + 0.5vw);
    /* 16px + ekran kengligining 0.5% */
}

h1 {
    font-size: calc(3.2rem + 2vw);
    /* 32px + ekran kengligining 2% */
}

/* clamp() - zamonaviy yondashuv */
h1 {
    font-size: clamp(3.2rem, 5vw, 6.4rem);
    /* min: 32px, ideal: 5vw, max: 64px */
}

p {
    font-size: clamp(1.4rem, 2vw, 1.8rem);
    /* min: 14px, ideal: 2vw, max: 18px */
}
Senior biladi: clamp() funksiyasi fluid typography uchun eng zamonaviy yechim. Media queries yozmasdan responsive shrift yaratish imkonini beradi.

Amaliy mashq (Senior daraja)

Vazifa: Viewport birliklaridan foydalanib responsive sahifa yarating.

  1. Hero section uchun min-height: 100vh bering
  2. Sarlavha uchun font-size: clamp(3.2rem, 5vw, 6.4rem) ishlating
  3. Konteyner uchun width: 90vw; max-width: 1200px; bering
  4. Kvadrat element yaratish uchun width: 30vmin; height: 30vmin; ishlating
  5. Fluid typography uchun calc() va clamp() dan foydalaning

Senior darajadagi yechim:

.hero { min-height: 100vh; display: flex; align-items: center; justify-content: center; background: linear-gradient(135deg, #667eea, #764ba2); color: white; text-align: center; padding: 4vmin; }

.hero-title { font-size: clamp(3.2rem, 8vw, 8rem); margin-bottom: 2rem; }
.hero-subtitle { font-size: clamp(1.6rem, 3vw, 2.4rem); opacity: 0.9; }

.container { width: 90vw; max-width: 1200px; margin: 0 auto; padding: 4vmin; }

.square-grid { display: flex; gap: 4vmin; justify-content: center; }
.square { width: 20vmin; height: 20vmin; background: #3b82f6; border-radius: 2vmin; }

.fluid-text { font-size: calc(1.6rem + 0.5vw); line-height: 1.6; }
.clamp-text { font-size: clamp(1.4rem, 2vw, 1.8rem); }

Dars xulosasi (Senior bilishi kerak)

  • vh/vw: viewport balandligi/kengligiga nisbatan (1% = 1vh/1vw)
  • vmin/vmax: viewport kichik/katta tomoniga nisbatan
  • Hero section: min-height: 100vh (height emas!)
  • Fluid typography: calc() va clamp()
  • clamp(): clamp(min, ideal, max) - eng zamonaviy
  • Keyingi darsda: Units yordamida loyihani yaxshilab olish
