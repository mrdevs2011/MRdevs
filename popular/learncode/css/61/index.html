CSS Grid

9-mart, 2021 CSS Grid Yuqori daraja
Bu darsda nimalarni o'rganasiz?

CSS Grid - bu ikki o'lchovli layout tizimi bo'lib, bir vaqtning o'zida qator va ustunlar bilan ishlash imkonini beradi. Flexbox bir o'lchovli (qator yoki ustun), Grid esa ikki o'lchovli (qator va ustun) layoutlar uchun mo'ljallangan. Bu darsda Grid asoslarini o'rganamiz.

Bu darsda nimalarni o'rganasiz?

  • display: grid - Grid konteyner yaratish
  • grid-template-columns - ustunlar soni va kengligi
  • grid-template-rows - qatorlar soni va balandligi
  • gap - qator va ustunlar orasidagi masofa
  • fr birligi - fraction (bo'sh joyni nisbatda taqsimlash)
  • repeat() funksiyasi

Bu dars uchun Senior maslahatlar

Grid vs Flexbox: Grid - ikki o'lchovli (qator va ustun), Flexbox - bir o'lchovli. Sahifa layouti uchun Grid, komponentlar uchun Flexbox. Senior yodlaydi
fr birligi: 1fr - qolgan bo'sh joyni nisbatda taqsimlaydi. Flexbox dagi flex-grow ga o'xshaydi. Professional
repeat() funksiyasi: repeat(3, 1fr) - 3 ta teng ustun. Kodni qisqartiradi. Amaliyot
Responsive Grid: repeat(auto-fit, minmax(250px, 1fr)) - ekran o'lchamiga qarab ustunlar soni avtomatik o'zgaradi. Zamonaviy

Grid Konteyner va Asosiy Xususiyatlar

display: grid xususiyati elementni Grid konteynerga aylantiradi. Barcha bevosita bola elementlar Grid item bo'ladi.

CSS
/* Grid konteyner */
.container {
    display: grid;
    grid-template-columns: 200px 1fr 200px;  /* 3 ta ustun */
    grid-template-rows: 100px 1fr 100px;    /* 3 ta qator */
    gap: 20px;                              /* Qator va ustunlar orasidagi masofa */
    height: 100vh;
}

fr birligi va repeat() funksiyasi

fr (fraction) - bo'sh joyni nisbatda taqsimlaydi. repeat() - takrorlanuvchi ustun/qatorlar uchun.

CSS
/* fr birligi - bo'sh joyni nisbatda taqsimlash */
.grid-fr {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;  /* 1:2:1 nisbatda */
}

/* repeat() funksiyasi */
.grid-repeat {
    display: grid;
    grid-template-columns: repeat(4, 1fr);  /* 4 ta teng ustun */
    grid-template-rows: repeat(3, 150px);  /* 3 ta qator, har biri 150px */
}

/* Responsive grid */
.grid-responsive {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    /* Ekranga qarab ustunlar soni avtomatik o'zgaradi */
}

Amaliy mashq

Vazifa: CSS Grid yordamida sahifa layouti yarating.

  1. Header, sidebar, main, footer dan iborat layout
  2. grid-template-columns: 250px 1fr;
  3. grid-template-rows: auto 1fr auto;
  4. gap: 20px; va min-height: 100vh;

Namunaviy yechim:

.layout {
    display: grid;
    grid-template-columns: 250px 1fr;
    grid-template-rows: auto 1fr auto;
    gap: 20px;
    min-height: 100vh;
}
.header { grid-column: 1 / -1; }
.footer { grid-column: 1 / -1; }
