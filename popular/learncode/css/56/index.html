Flexbox: flex-direction, flex-wrap, flex-flow va justify-content

4-mart, 2021 Flexbox O'rta daraja
Bu darsda nimalarni o'rganasiz?

Flexbox ning eng muhim xususiyatlarini o'rganamiz: flex-direction (asosiy o'q yo'nalishi), flex-wrap (elementlarni qatorga sig'masa pastga tushirish), flex-flow (shorthand) va justify-content (asosiy o'q bo'ylab elementlarni tekislash). Bu xususiyatlar zamonaviy layoutlar yaratishning asosidir.

Bu darsda nimalarni o'rganasiz?

  • flex-direction - asosiy o'q yo'nalishini belgilash
  • flex-wrap - elementlarni ko'p qatorli qilish
  • flex-flow - direction va wrap shorthand
  • justify-content - asosiy o'q bo'ylab tekislash

flex-direction - Asosiy o'q yo'nalishi

flex-direction xususiyati flex konteyner ichidagi elementlarning asosiy o'q bo'ylab qanday yo'nalishda joylashishini belgilaydi. Bu xususiyat asosiy o'q (main axis) yo'nalishini o'zgartiradi!

CSS
/* flex-direction qiymatlari */
.row { flex-direction: row; }              /* Default - chapdan o'ngga */
.row-reverse { flex-direction: row-reverse; } /* O'ngdan chapga */
.column { flex-direction: column; }        /* Yuqoridan pastga */
.column-reverse { flex-direction: column-reverse; } /* Pastdan yuqoriga */
Asosiy o'q tushunchasi

Muhim: flex-direction faqat elementlarning joylashish yo'nalishini emas, balki asosiy o'q (main axis) ning o'zini ham o'zgartiradi! flex-direction: column bo'lganda, asosiy o'q vertikal bo'ladi. Bu justify-content va align-items xususiyatlarining ishlashiga ta'sir qiladi.

flex-wrap - Ko'p qatorli flex

flex-wrap xususiyati flex elementlari konteynerga sig'masa, ularni keyingi qatorga o'tkazish yoki o'tkazmaslikni belgilaydi. Default holatda barcha elementlar bir qatorga siqiladi (hatto kenglikdan oshib ketsa ham!).

CSS
/* flex-wrap qiymatlari */
.nowrap { flex-wrap: nowrap; }        /* Default - bir qatorga siqiladi */
.wrap { flex-wrap: wrap; }            /* Sig'masa pastga tushadi */
.wrap-reverse { flex-wrap: wrap-reverse; } /* Sig'masa yuqoriga chiqadi */

flex-flow - Shorthand

flex-flow - bu flex-direction va flex-wrap xususiyatlarining shorthand (qisqa yozuv) versiyasi.

CSS
/* flex-flow: flex-direction flex-wrap; */
.flex-flow-1 { flex-flow: row wrap; }         /* Chapdan o'ngga, sig'masa pastga */
.flex-flow-2 { flex-flow: column nowrap; }    /* Yuqoridan pastga, bir ustun */
.flex-flow-3 { flex-flow: row-reverse wrap-reverse; } /* O'ngdan chapga, yuqoriga */

justify-content - Asosiy o'q bo'ylab tekislash

justify-content xususiyati flex elementlarini asosiy o'q (main axis) bo'ylab qanday tekislashni belgilaydi. Bu xususiyat bo'sh joyni elementlar orasida yoki atrofida taqsimlaydi.

CSS
/* justify-content qiymatlari */
.flex-start { justify-content: flex-start; }   /* Boshiga tekislash (default) */
.flex-end { justify-content: flex-end; }       /* Oxiriga tekislash */
.center { justify-content: center; }           /* Markazga tekislash */
.space-between { justify-content: space-between; } /* Bir tekis taqsimlash, chetlarga yopishadi */
.space-around { justify-content: space-around; }   /* Har bir element atrofiga teng bo'shliq */
.space-evenly { justify-content: space-evenly; }   /* Elementlar orasidagi barcha bo'shliqlar teng */
Senior biladi: space-between - navigatsiya menyulari uchun ideal (birinchi element chapda, oxirgi o'ngda). space-evenly - barcha bo'shliqlar teng, galereyalar uchun yaxshi.

justify-content va flex-direction bog'liqligi

Muhim: justify-content har doim asosiy o'q (main axis) bo'ylab ishlaydi. Asosiy o'q esa flex-direction ga bog'liq!

flex-directionAsosiy o'qjustify-content qanday ishlaydi
rowGorizontal (→)Gorizontal tekislaydi
columnVertikal (↓)Vertikal tekislaydi

Amaliy mashq

Vazifa: Flexbox xususiyatlaridan foydalanib layout yarating.

  1. Navigatsiya menyusi uchun justify-content: space-between
  2. Kartalar uchun flex-wrap: wrap va justify-content: center
  3. flex-direction: column yordamida vertikal layout

Namunaviy yechim:

.nav { display: flex; justify-content: space-between; align-items: center; }
.card-container { display: flex; flex-wrap: wrap; justify-content: center; gap: 20px; }
.card { flex: 0 1 300px; }

Dars xulosasi

  • flex-direction - asosiy o'q yo'nalishi (row, column, *-reverse)
  • flex-wrap - ko'p qatorli (wrap, nowrap, wrap-reverse)
  • flex-flow - direction va wrap shorthand
  • justify-content - asosiy o'q bo'ylab tekislash
  • Keyingi darsda: align-items va align-content
