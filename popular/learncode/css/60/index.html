Bu darsda nimalarni o'rganasiz?

Float Grid dan Flexbox ga o'tish

Loyihadagi eski float grid tizimini zamonaviy flexbox bilan almashtiramiz.

ESKI - Float Grid
.row { max-width: 114rem; margin: 0 auto; }
.row::after { content: ""; display: table; clear: both; }
[class^="col-"] { float: left; }
[class^="col-"]:not(:last-child) { margin-right: 6rem; }
.col-1-of-2 { width: calc((100% - 6rem) / 2); }
YANGI - Flexbox Grid
.row { display: flex; max-width: 114rem; margin: 0 auto; flex-wrap: wrap; }
.col { flex: 1; padding: 0 1.5rem; }
.col-1-of-2 { flex: 0 0 50%; }
.col-1-of-3 { flex: 0 0 33.333%; }
.col-1-of-4 { flex: 0 0 25%; }

Amaliy mashq

Vazifa: Loyihangizdagi float grid ni flexbox ga o'tkazing.

  1. Eski float grid ni o'chiring
  2. display: flex va flex-wrap: wrap qo'shing
  3. Ustunlar uchun flex: 0 0 50% (2 ustun) yoki flex: 1 ishlating
  4. gap xususiyatidan foydalaning

Namunaviy yechim:

.row { display: flex; flex-wrap: wrap; gap: 6rem; }
.col-1-of-2 { flex: 0 0 calc(50% - 3rem); }
.col-1-of-3 { flex: 0 0 calc(33.333% - 4rem); }

Dars xulosasi

  • Float o'rniga Flexbox - kod toza, responsive, ishonchli
  • flex: 1 - qolgan barcha bo'sh joyni egallash
  • gap - elementlar orasidagi masofa (margin o'rniga)
  • order - mobil versiyada tartibni o'zgartirish
