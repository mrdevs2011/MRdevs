2D transforms

25-fevral, 2021 Transforms va Animations O'rta daraja
Bu darsda nimalarni o'rganasiz?

CSS da transform xususiyati yordamida elementlarni 2D fazoda harakatlantirish, aylantirish, kattalashtirish va qiyalashtirishni o'rganasiz. Bu zamonaviy web dizayn va animatsiyalar uchun asosiy vosita hisoblanadi.

Bu darsda nimalarni o'rganasiz?

  • translate() - elementni siljitish (X, Y)
  • scale() - elementni kattalashtirish/kichiklashtirish
  • rotate() - elementni aylantirish
  • skew() - elementni qiyalashtirish
  • transform-origin - transform markazini belgilash
  • Bir nechta transformni birgalikda ishlatish

Bu dars uchun Senior maslahatlar

Transform tartibi muhim: translate() rotate() va rotate() translate() - natija har xil! Senior yodlaydi
Performance: transform va opacity GPU da ishlaydi, top/left ga qaraganda ancha tez. Optimizatsiya
transform-origin: Aylantirish markazini o'zgartirish. Default: center (50% 50%). Professional
Hover effektlar: transform + transition - silliq animatsiyalar uchun eng yaxshi kombinatsiya. UI/UX

translate() - Siljitish

translate() funksiyasi elementni X va Y o'qlari bo'ylab siljitadi.

CSS
/* translate() qiymatlari */
.translate-xy {
    transform: translate(50px, 30px);    /* X: 50px o'ngga, Y: 30px pastga */
}

.translate-x {
    transform: translateX(100px);        /* Faqat X o'qi bo'ylab */
}

.translate-y {
    transform: translateY(-50px);        /* Faqat Y o'qi bo'ylab (yuqoriga) */
}

/* Foizda - elementning o'z o'lchamiga nisbatan */
.translate-percent {
    transform: translate(-50%, -50%);    /* Markazga tekislash uchun */
}

/* Markazga tekislash klassik texnikasi */
.centered {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}

scale() - Kattalashtirish

scale() funksiyasi element o'lchamini o'zgartiradi. 1 - normal, >1 - kattalashtirish, <1 - kichiklashtirish.

CSS
/* scale() qiymatlari */
.scale-both {
    transform: scale(1.5);          /* 150% kattalashtirish */
}

.scale-small {
    transform: scale(0.8);          /* 80% kichiklashtirish */
}

.scale-xy {
    transform: scale(1.2, 0.8);     /* X: 120%, Y: 80% */
}

.scale-x {
    transform: scaleX(1.5);         /* Faqat X o'qi bo'ylab */
}

.scale-y {
    transform: scaleY(0.5);         /* Faqat Y o'qi bo'ylab */
}

/* Hover effekt */
.btn:hover {
    transform: scale(1.05);
}

rotate() - Aylantirish

rotate() funksiyasi elementni soat strelkasi bo'yicha aylantiradi. Birlik: deg, rad, turn.

CSS
/* rotate() qiymatlari */
.rotate-45 {
    transform: rotate(45deg);       /* 45 gradus soat strelkasi bo'yicha */
}

.rotate-negative {
    transform: rotate(-30deg);      /* Soat strelkasiga teskari */
}

.rotate-turn {
    transform: rotate(0.5turn);     /* Yarim aylanish (180deg) */
}

.rotate-rad {
    transform: rotate(3.14rad);     /* Radianlarda (~180deg) */
}

/* Hover da aylantirish */
.icon:hover {
    transform: rotate(360deg);
    transition: transform 0.5s;
}

skew() - Qiyalashtirish

skew() funksiyasi elementni X va Y o'qlari bo'ylab qiyalashtiradi.

CSS
/* skew() qiymatlari */
.skew-both {
    transform: skew(10deg, 5deg);   /* X: 10deg, Y: 5deg */
}

.skew-x {
    transform: skewX(15deg);        /* Faqat X o'qi bo'ylab */
}

.skew-y {
    transform: skewY(-10deg);       /* Faqat Y o'qi bo'ylab */
}

/* Creative effektlar */
.skew-box {
    transform: skewY(-5deg);
    background: #3b82f6;
    padding: 20px;
}

transform-origin

transform-origin xususiyati transform markazini belgilaydi. Default: center (50% 50%).

CSS
/* transform-origin qiymatlari */
.origin-center {
    transform-origin: center;       /* Default */
    transform: rotate(45deg);
}

.origin-top-left {
    transform-origin: top left;
    transform: rotate(45deg);       /* Yuqori chap burchakdan aylanadi */
}

.origin-bottom {
    transform-origin: bottom;
    transform: scale(1.5);          /* Pastki qismidan kattalashadi */
}

.origin-custom {
    transform-origin: 20% 80%;      /* Aniq koordinatalar */
    transform: rotate(30deg);
}

/* Piksellarda */
.origin-px {
    transform-origin: 50px 100px;
}

Bir nechta transform

Bir nechta transform funksiyasini bo'shliq bilan ajratib yozish mumkin. Tartib muhim!

CSS
/* Bir nechta transform - tartib muhim! */
.multiple-1 {
    transform: translateX(100px) rotate(45deg);
    /* Avval siljitiladi, keyin aylantiriladi */
}

.multiple-2 {
    transform: rotate(45deg) translateX(100px);
    /* Avval aylantiriladi, keyin aylangan o'q bo'ylab siljitiladi */
    /* NATIJA HAR XIL! */
}

/* Amaliy kombinatsiyalar */
.card:hover {
    transform: translateY(-5px) scale(1.02);
    box-shadow: 0 10px 20px rgba(0,0,0,0.1);
    transition: all 0.3s;
}

.btn:active {
    transform: translateY(2px) scale(0.98);
}

Amaliy mashq (Senior daraja)

Vazifa: 2D transformlardan foydalanib interaktiv elementlar yarating.

  1. Tugma hover bo'lganda translateY(-3px) va scale(1.05)
  2. Karta hover bo'lganda scale(1.02) va soya qo'shing
  3. Ikonka hover bo'lganda rotate(360deg) aylantiring
  4. transform-origin ni o'zgartirib ko'ring

Senior darajadagi yechim:

.btn { transition: all 0.3s; }
.btn:hover { transform: translateY(-3px) scale(1.05); box-shadow: 0 10px 20px rgba(0,0,0,0.15); }
.btn:active { transform: translateY(2px) scale(0.98); box-shadow: 0 5px 10px rgba(0,0,0,0.1); }

.card { transition: all 0.3s; }
.card:hover { transform: scale(1.02); box-shadow: 0 15px 30px rgba(0,0,0,0.1); }

.icon { transition: transform 0.5s; }
.icon:hover { transform: rotate(360deg); }

Dars xulosasi (Senior bilishi kerak)

  • skew(): qiyalashtirish
  • transform-origin: transform markazi
  • Bir nechta transform: tartib muhim! o'ngdan chapga emas, chapdan o'ngga
