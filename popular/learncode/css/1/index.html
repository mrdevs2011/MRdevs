Bu dars kimlar uchun?

Agar siz HTML ni o'rgangan bo'lsangiz va endi web sahifalaringizni chiroyli qilishni istasangiz - bu dars aynan siz uchun. HTML bilan tanish bo'lgan junior dasturchilar CSS ga o'tishda hech qanday qiyinchilikka duch kelmaydilar. Bu darsda hech qanday kod yozilmaydi, faqat CSS haqida tushuncha beriladi.

Bu dars uchun maslahatlar

Tushunib oling: CSS ning asosiy vazifasi - HTML elementlariga ko'rinish berish. Bu darsda kod yozish shart emas, faqat tushuncha hosil qiling! tushunish kerak
Eslab qoling: CSS = Cascading Style Sheets. "Cascading" so'zi juda muhim, keyingi darslarda nima uchun cascading deb atalishini tushunasiz. yodlash shart
HTML bilimingiz: CSS HTML bilan chambarchas bog'liq. Agar HTML ni yaxshi bilsangiz, CSS juda oson tuyuladi. HTML muhim

CSS nima?

CSS (Cascading Style Sheets) - bu HTML elementlariga ko'rinish (styling) berish uchun ishlatiladigan maxsus til. CSS yordamida siz web sahifangizning ranglarini, shriftlarini, elementlarning joylashuvini va hatto animatsiyalarini boshqarishingiz mumkin.

Oddiy tushuntirish: HTML va CSS
HTML
uyning skeleti va g'ishtlari
CSS
uyning bo'yog'i, mebellari va bezaklari
Natija
chiroyli va yashashga tayyor uy

HTML sizga sahifaning tuzilishini beradi: sarlavhalar, paragraflar, ro'yxatlar, rasmlar. CSS esa ularning qanday ko'rinishini belgilaydi: ranglar, o'lchamlar, joylashuv, animatsiyalar. CSS bo'lmasa, barcha web sahifalar faqat qora matn va oq fondan iborat bo'lardi.

CSS qanday ishlaydi?

Brauzer HTML hujjatni yuklaganda, CSS fayllarini ham yuklaydi va CSS qoidalarini HTML elementlariga qo'llaydi. Bu jarayon quyidagi bosqichlardan iborat:

1
Brauzer HTML ni yuklaydi

DOM (Document Object Model) deb ataladigan daraxtsimon tuzilmani yaratadi

2
CSS ni yuklaydi va tahlil qiladi

Barcha CSS fayllarini o'qiydi va qoidalarni ajratib oladi

3
CSS qoidalarini qo'llaydi

Har bir HTML elementiga mos keladigan CSS xususiyatlarini biriktiradi

4
Render (chizish)

Yakuniy natijani hisoblab, ekranda ko'rsatadi

Bilib qo'ying: Brauzer CSS ni yuqoridan pastga qarab o'qiydi. Bir elementga bir nechta CSS qoidasi berilgan bo'lsa, qaysi biri kuchliroq ekanligini "Specificity" (o'ziga xoslik) belgilaydi. Bu mavzuni 6-darsda batafsil o'rganamiz.

CSS yordamida nimalar qilish mumkin?

CSS juda kuchli vosita. Quyida CSS yordamida amalga oshirish mumkin bo'lgan asosiy imkoniyatlar keltirilgan:

Ranglar va fonlar

Matn ranglari, fon ranglari, gradientlar, fon rasmlari

Tipografiya

Shriftlar, matn o'lchamlari, qalinlik, qator oralig'i

Chegaralar va soyalar

Border, border-radius, box-shadow, text-shadow

Joylashuv

Margin, padding, position, display, flexbox, grid

Animatsiyalar

Transition, transform, keyframe animatsiyalar

Responsive dizayn

Media queries, turli ekran o'lchamlariga moslashish

Bu imkoniyatlarning barchasini kurs davomida batafsil o'rganamiz. Har bir darsda yangi CSS xususiyatlarini o'zlashtirib borasiz.

Amaliy mashq

Vazifa: CSS haqida olgan bilimlaringizni mustahkamlash uchun quyidagi savollarga javob bering:

  1. CSS nima uchun kerak? O'z so'zlaringiz bilan tushuntirib bering.
  2. CSS yordamida qanday o'zgarishlar qilish mumkin? Kamida 5 ta imkoniyatni sanab bering.
  3. Nima uchun CSS "Cascading" deb nomlangan? (Video diqqat bilan tinglagan bo'lsangiz, javobini bilasiz)

Namunaviy javoblar:

1. CSS nima uchun kerak?
CSS web sahifalarga dizayn va ko'rinish berish uchun kerak. HTML sahifaning tuzilishini bersa, CSS uni chiroyli, tartibli va foydalanuvchiga qulay qiladi. CSS bo'lmasa, barcha saytlar faqat qora matn va oq fondan iborat bo'lib qolardi.

2. CSS imkoniyatlari (5 ta):
• Ranglarni o'zgartirish (matn rangi, fon rangi)
• Shriftlarni tanlash va o'lchamini belgilash
• Elementlarni sahifada joylashtirish (layout)
• Animatsiyalar va hover effektlar yaratish
• Turli ekran o'lchamlariga moslash (responsive dizayn)

3. "Cascading" so'zining ma'nosi:
"Cascading" inglizcha "sharshara" degan ma'noni anglatadi. CSS da bu - qoidalarning ustunlik tartibini bildiradi. Agar bir elementga bir nechta CSS qoidasi berilgan bo'lsa, brauzer ularni "sharshara" kabi yuqoridan pastga qarab o'qiydi va eng kuchli (specific) qoidani qo'llaydi.

Dars xulosasi

  • CSS - Cascading Style Sheets, HTML elementlariga ko'rinish berish tili
  • HTML = tuzilish (skelet), CSS = ko'rinish (dizayn) - ular bir-birini to'ldiradi
  • Brauzer HTML va CSS ni yuklab, qoidalarni qo'llaydi va natijani ekranda ko'rsatadi
  • CSS yordamida ranglar, shriftlar, joylashuv, animatsiyalar va responsive dizayn yaratish mumkin
  • Keyingi darsda: CSS versiyalari va kurs loyihasi haqida tanishamiz
