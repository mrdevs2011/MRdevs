Flexbox: flex-basis, flex va align-self

7-mart, 2021 Flexbox O'rta daraja
Bu darsda nimalarni o'rganasiz?

Flexbox elementlariga beriladigan oxirgi xususiyatlarni o'rganamiz: flex-basis (elementning boshlang'ich o'lchami), flex (grow, shrink, basis shorthand) va align-self (bitta element uchun align-items ni bekor qilish).

Bu darsda nimalarni o'rganasiz?

  • flex-basis - elementning boshlang'ich o'lchami
  • flex - grow, shrink, basis shorthand
  • align-self - bitta element uchun align-items ni bekor qilish

flex-basis - Boshlang'ich o'lcham

flex-basis xususiyati flex elementining asosiy o'q bo'ylab boshlang'ich o'lchamini belgilaydi. width yoki height ga o'xshaydi, lekin flex konteksida ishlaydi. Default qiymat auto.

CSS
/* flex-basis - boshlang'ich o'lcham */
.basis-auto { flex-basis: auto; }     /* Kontentga qarab (default) */
.basis-200 { flex-basis: 200px; }     /* 200px dan boshlanadi */
.basis-50 { flex-basis: 50%; }        /* Konteynerning 50% */
.basis-0 { flex-basis: 0; }           /* 0 dan boshlanadi (flex-grow bilan ideal) */
Senior biladi: flex-basis: 0 + flex-grow: 1 - barcha elementlar teng kenglikda bo'lishini ta'minlaydi.

flex - Shorthand

flex - bu flex-grow, flex-shrink va flex-basis xususiyatlarining shorthand (qisqa yozuv) versiyasi. Har doim flex ishlating!

CSS
/* flex: flex-grow flex-shrink flex-basis; */
.flex-1 { flex: 1; }           /* flex: 1 1 0% - teng o'sadi, teng siqiladi */
.flex-auto { flex: auto; }     /* flex: 1 1 auto - kontentga qarab */
.flex-initial { flex: initial; } /* flex: 0 1 auto - o'smaydi, siqiladi */
.flex-none { flex: none; }     /* flex: 0 0 auto - o'smaydi, siqilmaydi */

/* Keng tarqalgan qiymatlar */
flex: 1;        /* Barcha bo'sh joyni teng egallaydi */
flex: 2;        /* flex: 2 1 0% - 2 hissa bo'sh joy */
flex: 0 0 200px; /* O'lchami qat'iy 200px, o'smaydi, siqilmaydi */

align-self - Bitta elementni alohida tekislash

align-self xususiyati bitta flex elementi uchun align-items qiymatini bekor qilish imkonini beradi.

CSS
/* align-self - align-items ni bekor qilish */
.container { align-items: center; }  /* Hamma element markazda */
.special-item { align-self: flex-start; }  /* Faqat bu element tepada */

Amaliy mashq

Vazifa: flex shorthand dan foydalanib layout yarating.

  1. 3 ta teng kenglikdagi ustun: flex: 1
  2. Sidebar qat'iy 250px: flex: 0 0 250px
  3. Main content qolgan joyni egallasin: flex: 1
  4. Bitta elementni align-self: flex-end bilan pastga tekislang

Namunaviy yechim:

.container { display: flex; align-items: center; }
.sidebar { flex: 0 0 250px; }
.main { flex: 1; }
.special { align-self: flex-end; }

Dars xulosasi

  • flex-basis - boshlang'ich o'lcham (auto, 0, px, %)
  • flex - grow shrink basis shorthand (doim ishlating!)
  • flex: 1 - barcha elementlar teng kenglikda
  • flex: 0 0 200px - qat'iy o'lcham
  • align-self - bitta element uchun align-items ni bekor qilish
  • Keyingi darsda: Flexbox yordamida loyihani yaxshilash - Nazorat ishi!
