Flexbox: align-items va align-content

5-mart, 2021 Flexbox O'rta daraja
Bu darsda nimalarni o'rganasiz?

Flexbox da elementlarni ko'ndalang o'q (cross axis) bo'ylab tekislashni o'rganamiz. align-items - bir qatorli flex konteynerlar uchun, align-content - ko'p qatorli flex konteynerlar uchun. Bu xususiyatlar vertikal tekislashni nihoyatda osonlashtiradi!

Bu darsda nimalarni o'rganasiz?

  • align-items - bir qatorli flex da ko'ndalang o'q bo'ylab tekislash
  • align-content - ko'p qatorli flex da qatorlarni tekislash
  • align-items vs align-content farqi
  • Vertikal markazga tekislash siri

align-items - Ko'ndalang o'q bo'ylab tekislash

align-items xususiyati flex elementlarini ko'ndalang o'q (cross axis) bo'ylab qanday tekislashni belgilaydi. Bu xususiyat bir qatorli flex konteynerlar uchun mo'ljallangan.

CSS
/* align-items qiymatlari */
.stretch { align-items: stretch; }      /* Default - cho'ziladi */
.flex-start { align-items: flex-start; } /* Boshiga tekislash */
.flex-end { align-items: flex-end; }     /* Oxiriga tekislash */
.center { align-items: center; }         /* Markazga tekislash - ENG KO'P ISHLATILADI! */
.baseline { align-items: baseline; }     /* Matn asosiy chizig'i bo'yicha */
Senior biladi: align-items: center - vertikal markazga tekislash uchun eng oson va to'g'ri yo'l. margin: auto yoki position hiylalaridan ko'ra ancha ishonchli.

align-content - Ko'p qatorli flex da qatorlarni tekislash

align-content xususiyati faqat ko'p qatorli flex konteynerlarda (flex-wrap: wrap yoki wrap-reverse) ishlaydi. Qatorlar orasidagi bo'sh joyni qanday taqsimlashni belgilaydi.

CSS
/* align-content qiymatlari (faqat wrap bo'lganda ishlaydi) */
.stretch { align-content: stretch; }        /* Default - qatorlar cho'ziladi */
.flex-start { align-content: flex-start; }  /* Qatorlar boshiga yig'iladi */
.flex-end { align-content: flex-end; }      /* Qatorlar oxiriga yig'iladi */
.center { align-content: center; }          /* Qatorlar markazga yig'iladi */
.space-between { align-content: space-between; } /* Qatorlar orasida teng bo'shliq */
.space-around { align-content: space-around; }   /* Har bir qator atrofida teng bo'shliq */
Muhim: align-content faqat flex-wrap: wrap yoki wrap-reverse bo'lganda ishlaydi! Bir qatorli flex da hech qanday ta'siri yo'q.

Amaliy mashq

Vazifa: align-items va align-content dan foydalanib layout yarating.

  1. Hero section uchun align-items: center + justify-content: center
  2. Kartalar galereyasi uchun flex-wrap: wrap + align-content: center

Namunaviy yechim:

.hero { display: flex; align-items: center; justify-content: center; min-height: 100vh; }
.gallery { display: flex; flex-wrap: wrap; align-content: center; gap: 20px; }

Dars xulosasi

  • align-items - bir qatorli flex da ko'ndalang o'q bo'ylab tekislash
  • align-items: center - vertikal markazga tekislash (eng oson yo'l!)
  • align-content - faqat wrap bo'lganda, qatorlarni tekislash
  • Keyingi darsda: order, flex-grow va flex-shrink
