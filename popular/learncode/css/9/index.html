"Class"lar va qo'shma (combined) selektorlar

16-yanvar, 2021 CSS Asoslari O'rta daraja
Bu darsda nimalarni o'rganasiz?

Class selektorlarini chuqurroq o'rganamiz: bir elementga bir nechta klass berish, klasslarni biriktirib yozish (combined selectors) va klasslar bilan ishlashning eng yaxshi amaliyotlari.

Bu darsda nimalarni o'rganasiz?

  • Bir elementga bir nechta klass berish
  • Qo'shma (combined) selektorlar - klasslarni biriktirish
  • Element va klassni birgalikda ishlatish
  • Class nomlash qoidalari va BEM metodologiyasi
  • Klasslar bilan ishlashda eng yaxshi amaliyotlar

Bu dars uchun maslahatlar

Bir nechta klass - kuchli vosita: Bitta elementga bir nechta klass berish kodni qayta ishlatish imkonini beradi. yodlash shart
Qo'shma selektor: .btn.primary - ikkala klassga ham ega elementni tanlaydi (bo'shliqsiz!). muhim
BEM metodologiyasi: Block__Element--Modifier - katta loyihalarda klass nomlash uchun standart. amaliyot

Bir elementga bir nechta klass berish

HTML da bitta elementga bir nechta klass berish mumkin. Klasslar bo'shliq bilan ajratiladi. Bu kodni qayta ishlatish va modulli yozish imkonini beradi.

HTML 
<!-- Bir nechta klassga ega element -->
<button class="btn btn-primary btn-large">Yuborish</button>

<div class="card shadow rounded">
    <p>Karta kontenti</p>
</div>
CSS 
/* Asosiy tugma stillari */
.btn {
    padding: 10px 20px;
    border: none;
    border-radius: 6px;
    cursor: pointer;
}

/* Asosiy tugma rangi */
.btn-primary {
    background: #3b82f6;
    color: white;
}

/* Katta tugma */
.btn-large {
    padding: 14px 28px;
    font-size: 18px;
}
Afzalligi: Kod takrorlanmaydi. .btn asosiy stillarni, .btn-primary rangni, .btn-large o'lchamni boshqaradi.

Qo'shma (combined) selektorlar

Qo'shma selektor - bir nechta klassni bo'shliqsiz yonma-yon yozish. Bu faqat barcha ko'rsatilgan klasslarga ega elementlarni tanlaydi.

CSS 
/* Ikkala klassga ham ega element */
.btn.primary {
    background: #3b82f6;
    color: white;
}

/* Uchta klassga ega element */
.card.shadow.rounded {
    box-shadow: 0 4px 6px rgba(0,0,0,0.1);
    border-radius: 12px;
}
Diqqat! Bo'shliq bor yoki yo'q?
  • .btn .primary (bo'shliq bor) - .btn ichidagi .primary elementini tanlaydi
  • .btn.primary (bo'shliq yo'q) - ikkala klassga ega elementni tanlaydi

Element va klassni birgalikda ishlatish

Element selektor va klassni birgalikda ishlatish orqali aniqroq tanlash mumkin.

CSS 
/* Faqat button elementidagi .primary klassini tanlaydi */
button.primary {
    background: #3b82f6;
}

/* Faqat p elementidagi .highlight klassini tanlaydi */
p.highlight {
    background: #fef3c7;
    padding: 10px;
}
Qachon ishlatiladi? Bir xil klass nomi turli elementlarda turlicha ko'rinishda bo'lishi kerak bo'lganda.

Class nomlash qoidalari

Katta loyihalarda klass nomlarini to'g'ri tanlash juda muhim. Eng mashhur metodologiyalardan biri - BEM (Block Element Modifier).

BEM namunasi 
<!-- Block -->
<div class="card">
    <!-- Element -->
    <h2 class="card__title">Sarlavha</h2>
    <p class="card__text">Matn</p>
    <!-- Modifier -->
    <button class="card__button card__button--primary">Bosish</button>
</div>

Block

Mustaqil komponent (.card, .btn)

Element

Blokning qismi (.card__title)

Modifier

O'zgartirilgan holat (.btn--large)

Klasslar bilan eng yaxshi amaliyotlar

1
Mazmunli nomlar bering

.red-text emas, .error-message deb nomlang. Agar keyin rang o'zgarsa, nom mos kelmay qoladi.

2
Modulli yondashing

Har bir klass faqat bitta vazifani bajarsin. .btn asosiy, .btn-primary rang uchun.

3
ID o'rniga klass ishlating

CSS da stillash uchun ID ishlatmang. Klasslar qayta ishlatilishi mumkin va specificity muammolarini keltirib chiqarmaydi.

4
Kichik harflar va defis

Klass nomlarida kichik harflar va so'zlar orasida defis ishlating: .main-header, .btn-primary.

Amaliy mashq

Vazifa: Klasslar va qo'shma selektorlardan foydalanib tugmalar tizimini yarating.

  1. .btn klassi - asosiy tugma stillari (padding, border-radius, cursor)
  2. .btn-primary - ko'k tugma
  3. .btn-secondary - kulrang tugma
  4. .btn-large - katta tugma
  5. button.primary - faqat button elementidagi primary klassiga qo'shimcha stil
  6. BEM uslubida .card blokini yarating: .card__title, .card__text, .card__button

Namunaviy yechim:

<!-- HTML -->
<button class="btn btn-primary">Yuborish</button>
<button class="btn btn-secondary btn-large">Bekor qilish</button>
<a href="#" class="btn btn-primary">Havola tugma</a>

<div class="card">
    <h3 class="card__title">Karta sarlavhasi</h3>
    <p class="card__text">Karta ichidagi matn.</p>
    <button class="card__button card__button--primary">Batafsil</button>
</div>

<!-- CSS -->
.btn {
    display: inline-block;
    padding: 10px 20px;
    border: none;
    border-radius: 6px;
    cursor: pointer;
    font-size: 16px;
    text-decoration: none;
}

.btn-primary {
    background: #3b82f6;
    color: white;
}

.btn-secondary {
    background: #64748b;
    color: white;
}

.btn-large {
    padding: 14px 28px;
    font-size: 18px;
}

button.primary {
    font-weight: bold;
}

/* BEM */
.card {
    border: 1px solid #e2e8f0;
    border-radius: 12px;
    padding: 20px;
}
.card__title { color: #1e293b; margin-bottom: 10px; }
.card__text { color: #64748b; margin-bottom: 15px; }
.card__button--primary { background: #3b82f6; color: white; }

Dars xulosasi

  • Bir elementga bir nechta klass berish: class="btn btn-primary btn-large"
  • Qo'shma selektor: .btn.primary (bo'shliqsiz) - ikkala klassga ega element
  • Element + klass: button.primary - aniq element uchun
  • BEM: Block__Element--Modifier - klass nomlash standarti
  • Keyingi darsda: Class yoki ID. "!important" haqida
