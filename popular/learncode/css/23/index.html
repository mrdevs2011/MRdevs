Absolute va Sticky

30-yanvar, 2021 Positioning O'rta daraja
Bu darsda nimalarni o'rganasiz?

CSS da position: absolute va position: sticky qiymatlarini batafsil o'rganasiz. Absolute elementni aniq nuqtaga joylashtirish, Sticky esa elementni scroll bo'lganda yopishqoq qilish uchun ishlatiladi.

Bu darsda nimalarni o'rganasiz?

  • position: absolute - absolyut joylashuv
  • Absolute element qayerga nisbatan joylashadi?
  • position: sticky - yopishqoq joylashuv
  • Sticky element qanday ishlaydi?
  • Absolute va Sticky ning farqlari va qo'llanilishi

Bu dars uchun maslahatlar

Absolute ota elementga bog'liq: position: absolute elementi position: relative bo'lgan eng yaqin ota elementga nisbatan joylashadi. yodlash shart
Sticky - gibrid: position: sticky relative va fixed ning aralashmasi. Scroll bo'lmaganda relative, scroll bo'lganda fixed kabi ishlaydi. muhim
Sticky uchun top kerak: position: sticky ishlashi uchun top, right, bottom yoki left qiymati berilishi shart. amaliyot

Position: Absolute

position: absolute - elementni normal oqimdan chiqarib, position: relative, absolute, fixed yoki sticky bo'lgan eng yaqin ota elementga nisbatan joylashtiradi. Agar bunday ota element bo'lmasa, <html> ga nisbatan joylashadi.

CSS 
/* Ota element - relative */
.parent {
    position: relative;
    width: 400px;
    height: 300px;
    background: #f3f4f6;
}

/* Bola element - absolute */
.child {
    position: absolute;
    top: 20px;
    right: 30px;
    background: #3b82f6;
    color: white;
    padding: 15px;
}

/* Markazga joylashtirish */
.centered {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}

/* Badge - kartada */
.card {
    position: relative;
}

.badge {
    position: absolute;
    top: -10px;
    right: -10px;
    background: #ef4444;
    color: white;
    padding: 5px 10px;
    border-radius: 20px;
}
Absolute tushunchasi
Ota element
(position: relative)
Bola element
(position: absolute)
top: 20px; right: 30px;
Diqqat: position: absolute element normal oqimdan butunlay chiqariladi. Uning asl joyi band bo'lmaydi va boshqa elementlar bu bo'shliqqa joylashadi.
Markazga tekislash siri: top: 50%; left: 50%; transform: translate(-50%, -50%); - bu kombinatsiya elementni ota konteynerning to'liq markaziga joylashtiradi.

Position: Sticky

position: sticky - bu relative va fixed ning gibridi. Element scroll bo'lmaganda relative kabi, belgilangan chegaraga yetganda esa fixed kabi yopishib qoladi.

CSS 
/* Sticky header */
.sticky-header {
    position: sticky;
    top: 0;                /* Yuqoriga yopishadi */
    background: #1e293b;
    color: white;
    padding: 20px;
    z-index: 100;
}

/* Sticky sidebar */
.sticky-sidebar {
    position: sticky;
    top: 20px;             /* Yuqoridan 20px masofada yopishadi */
    align-self: flex-start;
}

/* Sticky kategoriya sarlavhasi */
.sticky-category {
    position: sticky;
    top: 0;
    background: #f3f4f6;
    padding: 15px;
    font-weight: bold;
}
Xususiyat position: sticky position: fixed
Ota elementga bog'liq Ha, ota element ichida qoladi Yo'q, viewport ga nisbatan
Normal oqim Qoladi (yopishmaguncha) Chiqariladi
Offset talab qiladi Ha (top, right, bottom, left) Shart emas
Asosiy ishlatilishi Header, kategoriya sarlavhalari Navigatsiya, popup
Sticky ishlashi uchun shartlar:
  • top, right, bottom yoki left qiymati berilishi shart
  • Ota elementda overflow: hidden bo'lmasligi kerak
  • Ota element balandligi sticky elementdan katta bo'lishi kerak

Barcha Position qiymatlari taqqoslash

Qiymat Normal oqim Nisbatan Offset ishlaydi
static Ha - Yo'q
relative Ha (joyi band) O'z joyiga Ha
absolute Yo'q Position berilgan ota elementga Ha
fixed Yo'q Viewport ga Ha
sticky Ha (yopishmaguncha) Ota elementga / Viewport ga Ha

Amaliy mashq

Vazifa: Absolute va Sticky position lardan foydalanib sahifa elementlarini yarating.

  1. Karta va badge: Ota elementga position: relative, badge ga position: absolute bering
  2. Badge ni kartaning yuqori o'ng burchagiga joylashtiring (top: -10px; right: -10px)
  3. Sticky header: Yuqoriga yopishib qoladigan header yarating (position: sticky; top: 0)
  4. Sticky kategoriya: Kategoriya sarlavhalarini sticky qiling
  5. Modal oyna: position: fixed overlay va position: absolute yopish tugmasi yarating
