Display xossasi

22-yanvar, 2021 Box Model Boshlang'ich daraja
Eslatma!

Bu dars oxirida 1-15 darslar bo'yicha 15 ta "test"dan iborat nazorat ishi mavjud. Test barcha o'tilgan mavzular bo'yicha bo'ladi.

Bu darsda nimalarni o'rganasiz?

  • display xossasi nima va nima uchun kerak?
  • block - blok elementlar
  • inline - satr ichidagi elementlar
  • inline-block - gibrid elementlar
  • none - elementni yashirish

Bu dars uchun maslahatlar

Block vs Inline: Block elementlar to'liq kenglikni egallaydi va yangi qatordan boshlanadi. Inline elementlar esa faqat kontenticha joy egallaydi. yodlash shart
inline-block: Inline kabi bir qatorda turadi, lekin block kabi width, height, margin, padding larni to'liq qabul qiladi. muhim
display: none: Elementni butunlay yashiradi va joyni bo'shatadi. visibility: hidden dan farqi - joy saqlanib qoladi. amaliyot

Display xossasi nima?

display xossasi elementning sahifada qanday ko'rinishini va boshqa elementlar bilan qanday munosabatda bo'lishini belgilaydi. Bu CSS dagi eng muhim xususiyatlardan biri hisoblanadi.

Block elementlar

Block elementlar quyidagi xususiyatlarga ega:

  • Har doim yangi qatordan boshlanadi
  • Mavjud bo'lgan to'liq kenglikni egallaydi (default: width: 100%)
  • Width, height, margin, padding larni to'liq qabul qiladi
  • Block elementlar ichida boshqa block yoki inline elementlar bo'lishi mumkin
CSS 
/* Default block elementlar */
div, p, h1-h6, ul, ol, li, header, footer, section, article {
    display: block;
}

/* Block elementga misol */
.block-example {
    display: block;
    width: 300px;
    height: 100px;
    margin: 20px;
    padding: 15px;
    background: #3b82f6;
    color: white;
}
Block elementlarga misollar: <div>, <p>, <h1>-<h6>, <ul>, <ol>, <li>, <header>, <footer>, <section>, <article>

Inline elementlar

Inline elementlar quyidagi xususiyatlarga ega:

  • Yangi qatordan boshlanmaydi, bir qatorda ketma-ket joylashadi
  • Faqat kontenti egallagan joyni oladi
  • Width va height ishlamaydi!
  • Margin va padding faqat gorizontal (chap/o'ng) ishlaydi, vertikal (yuqori/past) ishlamaydi
CSS 
/* Default inline elementlar */
span, a, strong, em, i, b, u, small {
    display: inline;
}

/* Inline elementga misol */
.inline-example {
    display: inline;
    /* width: 200px; ISHLAMAYDI! */
    /* height: 100px; ISHLAMAYDI! */
    margin: 0 10px;      /* Faqat gorizontal ishlaydi */
    padding: 5px 15px;   /* Gorizontal ishlaydi, vertikal ko'rinadi lekin joy egallamaydi */
    background: #10b981;
    color: white;
}
Diqqat: Inline elementlarga width va height berib bo'lmaydi! Vertikal margin va padding ko'rinadi lekin atrofdagi elementlarga ta'sir qilmaydi (joy egallamaydi).
Inline elementlarga misollar: <span>, <a>, <strong>, <em>, <i>, <b>, <u>, <small>, <img> (maxsus holat)

Inline-Block - gibrid element

display: inline-block - bu block va inline elementlarning eng yaxshi xususiyatlarini birlashtiradi:

Inline-Block afzalliklari

  • Inline kabi - bir qatorda turadi
  • Block kabi - width, height, margin, padding larni to'liq qabul qiladi
  • Vertikal margin va padding to'liq ishlaydi
  • Navigatsiya menyulari, kartalar uchun ideal

Kamchiliklari

  • Elementlar orasida kichik bo'shliq qoladi (HTML dagi probel tufayli)
  • Bu bo'shliqni yo'qotish uchun ota elementga font-size: 0 berish kerak
CSS 
/* Inline-block - navigatsiya uchun ideal */
.nav {
    font-size: 0; /* Bo'shliqni yo'qotish uchun */
}

.nav-item {
    display: inline-block;
    width: 120px;
    height: 50px;
    margin: 10px;
    padding: 15px 20px;
    background: #3b82f6;
    color: white;
    font-size: 16px; /* Qayta tiklash */
    text-align: center;
    border-radius: 8px;
}

/* Kartalar uchun inline-block */
.card {
    display: inline-block;
    width: 280px;
    margin: 15px;
    padding: 20px;
    background: white;
    border-radius: 12px;
    box-shadow: 0 4px 6px rgba(0,0,0,0.1);
    vertical-align: top; /* Kartalarni tepadan tekislash */
}
Bo'shliq muammosini hal qilish: Inline-block elementlar orasidagi bo'shliq HTML dagi probel yoki yangi qator tufayli paydo bo'ladi. Yechim: ota elementga font-size: 0 berib, bola elementlarga kerakli font-size ni qayta tiklash.

Display: none

display: none - elementni sahifadan butunlay yashiradi. Element nafaqat ko'rinmaydi, balki sahifada joy ham egallamaydi.

Xususiyat display: none visibility: hidden opacity: 0
Ko'rinishi Yo'q Yo'q Yo'q
Joy egallashi Yo'q Ha Ha
Event lar ishlashi Yo'q Yo'q Ha
Animatsiya qilish Qiyin Oson Oson
CSS 
/* Elementni yashirish */
.hidden {
    display: none;
}

/* Responsive dizaynda ishlatish */
.mobile-only {
    display: none;
}

@media (max-width: 768px) {
    .desktop-only {
        display: none;
    }
    
    .mobile-only {
        display: block;
    }
}

Amaliy mashq

Vazifa: Display xususiyatlaridan foydalanib navigatsiya menyusi va kartalar yarating.

  1. Navigatsiya menyusi: 4 ta linkdan iborat gorizontal menyu yarating
  2. Linklarni display: inline-block qiling
  3. Har bir linkka width: 120px, height: 50px, margin: 10px bering
  4. Fon rangi, matn rangi va border-radius qo'shing
  5. Kartalar: 3 ta karta yarating va display: inline-block qiling
  6. Kartalar orasidagi bo'shliqni yo'qotish uchun ota elementga font-size: 0 bering
  7. Kartalarga kerakli font-size ni qayta tiklang

Dars xulosasi

  • display: block - yangi qator, to'liq kenglik, width/height ishlaydi
  • display: inline - bir qatorda, width/height ISHLAMAYDI
  • display: inline-block - bir qatorda, lekin width/height ISHLAYDI
  • display: none - elementni yashiradi, joy egallamaydi
  • Inline-block elementlar orasidagi bo'shliqni font-size: 0 bilan yo'qotish mumkin
  • Keyingi darsda: Text align, Vertical align va Float
