Formalarga style berish (1-qism)

6-fevral, 2021 Formalar va Units O'rta daraja
Eslatma!

Bu dars oxirida 1-30 darslar bo'yicha 30 ta "test"dan iborat nazorat ishi mavjud. Test barcha o'tilgan mavzular bo'yicha bo'ladi.

Bu darsda nimalarni o'rganasiz?

CSS da formalar (input, textarea, select, button) bilan ishlashni o'rganamiz. Formalarga style berish, ularning holatlarini (:focus, :hover, :valid, :invalid) boshqarish va zamonaviy dizayn yaratishni o'rganasiz. Shuningdek, 1-30 darslar bo'yicha nazorat ishi orqali bilimingizni mustahkamlaysiz.

Bu darsda nimalarni o'rganasiz?

  • <input> elementiga style berish
  • Input turlari: text, email, password, number, date
  • :focus, :hover, :valid, :invalid psevdo-klasslari
  • <textarea> va <select> stillari
  • Placeholder matniga style berish
  • 1-30 darslar bo'yicha 30 talik nazorat ishi

Bu dars uchun Senior maslahatlar

Input reset: Brauzer default stillarini olib tashlash uchun appearance: none va outline: none ishlating. Senior yodlaydi
:focus holati muhim: Foydalanuvchi qayerda ekanligini bilishi uchun input fokus bo'lganda aniq ko'rinadigan style bering. Accessibility
Placeholder stili: ::placeholder psevdo-elementi orqali placeholder matniga style berish mumkin. Professional
Nazorat ishiga tayyorlaning: Barcha darslarni yaxshilab takrorlang. Nazorat ishi barcha mavzularni qamrab oladi. muhim

Input elementlari

Formalardagi asosiy element bu <input>. Uning turli xil turlari mavjud: text, email, password, number, date va boshqalar.

HTML 
<form class="form">
    <div class="form-group">
        <input type="text" class="form-input" placeholder="To'liq ismingiz" required>
        <label class="form-label">To'liq ismingiz</label>
    </div>
    
    <div class="form-group">
        <input type="email" class="form-input" placeholder="Email manzilingiz" required>
        <label class="form-label">Email manzilingiz</label>
    </div>
    
    <div class="form-group">
        <input type="password" class="form-input" placeholder="Parol" required minlength="8">
        <label class="form-label">Parol</label>
    </div>
    
    <button type="submit" class="btn-submit">Yuborish</button>
</form>
CSS 
/* Form stillari */
.form-group {
    position: relative;
    margin-bottom: 2rem;
}

.form-input {
    width: 100%;
    padding: 1.5rem 2rem;
    font-size: 1.6rem;
    font-family: inherit;
    color: #333;
    background-color: rgba(255, 255, 255, 0.9);
    border: none;
    border-bottom: 3px solid transparent;
    border-radius: 4px;
    transition: all 0.3s;
}

.form-input:focus {
    outline: none;
    border-bottom: 3px solid #55c57a;
    box-shadow: 0 1rem 2rem rgba(0, 0, 0, 0.1);
}

.form-input:focus:invalid {
    border-bottom: 3px solid #ff7730;
}

.form-input::-webkit-input-placeholder {
    color: #999;
}

Amaliy mashq

Vazifa: Zamonaviy forma yarating.

  1. Ism, email, parol va xabar uchun inputlar yarating
  2. Barcha inputlarga bir xil style bering (padding, border, border-radius)
  3. :focus holati uchun alohida style bering
  4. Placeholder matniga style bering
  5. :invalid holati uchun qizil border qo'shing
  6. Yuborish tugmasiga hover effekt qo'shing

Namunaviy yechim:

.form-group { margin-bottom: 20px; }
.form-input {
    width: 100%; padding: 12px 16px; font-size: 16px;
    border: 2px solid #e2e8f0; border-radius: 8px;
    transition: all 0.3s;
}
.form-input:focus {
    outline: none; border-color: #3b82f6;
    box-shadow: 0 0 0 3px rgba(59,130,246,0.1);
}
.form-input:invalid:not(:placeholder-shown) { border-color: #ef4444; }
.form-input::placeholder { color: #94a3b8; font-size: 14px; }
.btn-submit {
    background: #3b82f6; color: white; padding: 12px 24px;
    border: none; border-radius: 8px; cursor: pointer;
    transition: all 0.3s;
}
.btn-submit:hover { background: #2563eb; transform: translateY(-2px); }

Dars xulosasi

  • ::placeholder - placeholder matniga style berish
  • :valid / :invalid - to'g'ri/noto'g'ri to'ldirilgan inputlar
  • appearance: none - brauzer default stillarini olib tashlash
