Local font qo'shish

22-fevral, 2021 Fontlar va Transforms O'rta daraja
Bu darsda nimalarni o'rganasiz?

CSS da @font-face qoidasi yordamida lokal (o'z serveringizdagi) shriftlarni qanday qo'shishni o'rganasiz. Bu Google Fonts ga bog'liq bo'lmaslik va maxsus shriftlarni ishlatish imkonini beradi.

Bu darsda nimalarni o'rganasiz?

  • @font-face qoidasi sintaksisi
  • Shrift formatlari (woff, woff2, ttf, eot, svg)
  • font-family nomini belgilash
  • font-weight va font-style ko'rsatish
  • font-display xususiyati
  • Bir nechta shrift qalinliklarini qo'shish

Bu dars uchun Senior maslahatlar

WOFF2 formati: Eng yaxshi siqish va barcha zamonaviy brauzerlar qo'llaydi. WOFF2 > WOFF > TTF. Senior yodlaydi
font-display: swap; Shrift yuklanmaguncha fallback shrift ko'rinadi. FOIT (Flash of Invisible Text) muammosini hal qiladi. Professional
local() funksiyasi: src: local('Shrift Nomi'), url('shrift.woff2'); - avval foydalanuvchi kompyuteridan qidiradi. Optimizatsiya
Papka tuzilmasi: Shriftlarni fonts/ papkasida saqlang va to'g'ri yo'l ko'rsating. Organizatsiya

@font-face sintaksisi

@font-face qoidasi orqali o'z shriftlaringizni CSS da e'lon qilishingiz mumkin.

CSS
/* @font-face asosiy sintaksisi */
@font-face {
    font-family: 'MyCustomFont';           /* Shrift nomi (o'zingiz nomlaysiz) */
    src: url('../fonts/mycustomfont.woff2') format('woff2'),
         url('../fonts/mycustomfont.woff') format('woff'),
         url('../fonts/mycustomfont.ttf') format('truetype');
    font-weight: normal;                   /* 400 */
    font-style: normal;
    font-display: swap;                    /* FOIT muammosini hal qiladi */
}

/* Endi bu shriftni ishlatish mumkin */
body {
    font-family: 'MyCustomFont', sans-serif;
}
Formatlar tartibi: Eng yaxshi formatni birinchi ko'rsating. Zamonaviy brauzerlar uchun woff2, eskilari uchun woff yoki ttf.

Senior biladi: 2025-yilda faqat woff2 va woff formatlarini qo'llash yetarli. TTF/EOT/SVG kerak emas.

Bir nechta shrift qalinligini qo'shish

Har bir shrift qalinligi uchun alohida @font-face qoidasi yozish kerak.

CSS
/* Regular (400) */
@font-face {
    font-family: 'Lato';
    src: url('../fonts/Lato-Regular.woff2') format('woff2');
    font-weight: 400;
    font-style: normal;
    font-display: swap;
}

/* Bold (700) */
@font-face {
    font-family: 'Lato';
    src: url('../fonts/Lato-Bold.woff2') format('woff2');
    font-weight: 700;
    font-style: normal;
    font-display: swap;
}

/* Light (300) */
@font-face {
    font-family: 'Lato';
    src: url('../fonts/Lato-Light.woff2') format('woff2');
    font-weight: 300;
    font-style: normal;
    font-display: swap;
}

/* Italic */
@font-face {
    font-family: 'Lato';
    src: url('../fonts/Lato-Italic.woff2') format('woff2');
    font-weight: 400;
    font-style: italic;
    font-display: swap;
}

/* Endi bir xil font-family bilan ishlatish mumkin */
.text { font-family: 'Lato', sans-serif; }
.light { font-weight: 300; }  /* Lato-Light.woff2 yuklanadi */
.normal { font-weight: 400; } /* Lato-Regular.woff2 yuklanadi */
.bold { font-weight: 700; }   /* Lato-Bold.woff2 yuklanadi */
.italic { font-style: italic; } /* Lato-Italic.woff2 yuklanadi */

local() funksiyasi

local() funksiyasi avval foydalanuvchi kompyuterida shrift bor-yo'qligini tekshiradi. Bu keraksiz yuklashni oldini oladi.

CSS
@font-face {
    font-family: 'Roboto';
    /* Avval kompyuterdan qidirish, topilmasa yuklash */
    src: local('Roboto'), local('Roboto-Regular'),
         url('../fonts/Roboto-Regular.woff2') format('woff2');
    font-weight: 400;
    font-style: normal;
    font-display: swap;
}

/* Bir nechta nom variantlari */
@font-face {
    font-family: 'Helvetica Neue';
    src: local('Helvetica Neue'), local('HelveticaNeue'),
         url('../fonts/HelveticaNeue.woff2') format('woff2');
    font-weight: 400;
    font-style: normal;
}

font-display xususiyati

font-display shrift yuklanayotgan vaqtda nima ko'rinishini belgilaydi.

CSS
/* font-display qiymatlari */
@font-face {
    font-family: 'MyFont';
    src: url('myfont.woff2') format('woff2');
    font-display: swap;  /* Eng tavsiya etilgan */
}

/*
font-display qiymatlari:
- auto: Brauzer qaror qiladi (default)
- block: Qisqa vaqt yashirin, keyin fallback (FOIT)
- swap: Darhol fallback ko'rinadi, yuklanganda almashtiriladi (FOUT) ⭐
- fallback: Qisqa yashirin, yuklanmasa fallback
- optional: Yuklanmasa fallback, keyin ham almashtirilmaydi
*/
FOUT vs FOIT: swap - FOUT (matn darhol ko'rinadi), block - FOIT (matn yashirin). Foydalanuvchi tajribasi uchun swap yaxshiroq.

Amaliy mashq (Senior daraja)

Vazifa: @font-face yordamida lokal shriftlarni qo'shing.

  1. fonts/ papkasiga shrift fayllarini joylang
  2. 3 xil qalinlik uchun @font-face qoidalarini yozing (300, 400, 700)
  3. WOFF2 va WOFF formatlarini ko'rsating
  4. font-display: swap qo'shing
  5. local() funksiyasidan foydalaning
  6. Yaratilgan shriftni sahifada qo'llang
