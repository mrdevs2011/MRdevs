Foiz (%) o'lchov birligi

10-fevral, 2021 Units va RWD Boshlang'ich daraja
Bu darsda nimalarni o'rganasiz?

CSS da eng ko'p ishlatiladigan nisbiy birlik - % (foiz) haqida chuqur bilim olasiz. Foiz birligi nimaga nisbatan hisoblanishi, turli xususiyatlarda qanday ishlashi va responsive dizayn yaratishdagi ahamiyatini o'rganasiz.

Bu darsda nimalarni o'rganasiz?

  • Foiz birligi nimaga nisbatan hisoblanadi?
  • width va height da foiz
  • margin va padding da foiz
  • position (top, right, bottom, left) da foiz
  • transform: translate() da foiz
  • font-size va line-height da foiz

Bu dars uchun Senior maslahatlar

Foiz nimaga nisbatan? Har bir xususiyat uchun turlicha! width ota element kengligiga, height ota element balandligiga nisbatan. Senior yodlaydi
margin/padding foiz: Gorizontal VA vertikal margin/padding foizlari ota element kengligiga nisbatan hisoblanadi! Ajablanarli fakt
translate(-50%, -50%): Markazga tekislashda % elementning o'z o'lchamiga nisbatan hisoblanadi. Professional texnika
Responsive layout: width: 90%; max-width: 1200px; kombinatsiyasi eng yaxshi amaliyot. Mobile-first

Width va Height da foiz

width va height xususiyatlarida foiz ota element o'lchamlariga nisbatan hisoblanadi.

CSS 
.parent {
    width: 800px;
    height: 500px;
    background: #f3f4f6;
}

/* width: ota element kengligiga nisbatan */
.child-width {
    width: 50%;             /* 800px * 0.5 = 400px */
}

/* height: ota element balandligiga nisbatan */
.child-height {
    height: 50%;            /* 500px * 0.5 = 250px */
}

/* Agar ota element balandligi belgilanmagan bo'lsa */
.parent-auto {
    width: 800px;
    /* height belgilanmagan - avtomatik */
}
.child-height-fail {
    height: 50%;            /* ISHLAMAYDI! Ota element balandligi aniq emas */
}

/* Responsive konteyner */
.container {
    width: 90%;             /* Ota element kengligining 90% */
    max-width: 1200px;      /* 1200px dan katta bo'lmaydi */
    margin: 0 auto;
}
Diqqat: height: 100% ishlashi uchun ota elementning balandligi aniq belgilangan bo'lishi kerak (px, vh, yoki boshqa foiz). Aks holda ishlamaydi!

Margin va Padding da foiz

Muhim: margin va padding da foiz (gorizontal ham, vertikal ham) har doim ota element kengligiga nisbatan hisoblanadi!

CSS 
.parent {
    width: 800px;           /* Kenglik 800px */
    height: 500px;          /* Balandlik 500px */
}

.child {
    /* Barcha margin va padding foizlari ota element KENGLIGIGA nisbatan! */
    margin-top: 10%;        /* 800px * 0.1 = 80px */
    margin-bottom: 10%;     /* 800px * 0.1 = 80px */
    margin-left: 5%;        /* 800px * 0.05 = 40px */
    margin-right: 5%;       /* 800px * 0.05 = 40px */
    
    padding-top: 5%;        /* 800px * 0.05 = 40px */
    padding-bottom: 5%;     /* 800px * 0.05 = 40px */
}
Senior biladi: Bu xususiyat tufayli kvadrat elementlar yaratish oson: width: 50%; padding-bottom: 50%; - element balandligi kengligiga teng bo'ladi!

Position (top, right, bottom, left) da foiz

position: absolute yoki fixed elementlarida top, right, bottom, left foizlari ota element (yoki viewport) o'lchamlariga nisbatan.

CSS 
.parent {
    position: relative;
    width: 800px;
    height: 500px;
}

.child {
    position: absolute;
    top: 20%;               /* 500px * 0.2 = 100px (ota balandligiga) */
    left: 10%;              /* 800px * 0.1 = 80px (ota kengligiga) */
    right: 10%;             /* 800px * 0.1 = 80px */
    bottom: 20%;            /* 500px * 0.2 = 100px */
}

/* Markazga tekislash - klassik texnika */
.centered {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    /* translate da % elementning o'z o'lchamiga nisbatan! */
}

Transform translate() da foiz

transform: translate() da foiz elementning o'z o'lchamiga nisbatan hisoblanadi. Bu markazga tekislash uchun juda qulay!

CSS 
.box {
    width: 200px;
    height: 150px;
}

/* Elementning o'z o'lchamiga nisbatan */
.box:hover {
    transform: translateX(50%);     /* 200px * 0.5 = 100px o'ngga */
    transform: translateY(-100%);   /* 150px * 1 = 150px yuqoriga */
}

/* Markazga aniq tekislash */
.modal {
    position: fixed;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    /* Element o'zining yarim kengligi va balandligiga siljiydi */
}

font-size va line-height da foiz

font-size da foiz ota elementning font-size iga nisbatan. line-height da foiz joriy elementning font-size iga nisbatan.

CSS 
.parent {
    font-size: 20px;
}

.child {
    font-size: 150%;        /* 20px * 1.5 = 30px (ota font-size iga) */
}

/* line-height: joriy font-size ga nisbatan */
.text {
    font-size: 16px;
    line-height: 150%;      /* 16px * 1.5 = 24px */
}

/* Unitless line-height (tavsiya etiladi) */
.text-best {
    font-size: 16px;
    line-height: 1.5;       /* Foiz emas, son - meros bo'lganda muammo tug'dirmaydi */
}
Senior biladi: line-height uchun foiz emas, unitless qiymat ishlating (masalan, 1.5). Bu meros bo'lganda kutilmagan natijalarni oldini oladi.

Amaliy mashq (Senior daraja)

Vazifa: Foiz birligidan foydalanib responsive layout yarating.

  1. Konteyner uchun width: 90%; max-width: 1200px; margin: 0 auto;
  2. Grid yaratish uchun ustunlarga width: 33.333% bering
  3. Elementni markazga tekislash uchun position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%);
  4. padding va margin uchun foiz ishlatib ko'ring
  5. Kvadrat element yarating: width: 30%; padding-bottom: 30%;

Senior darajadagi yechim:

.container { width: 90%; max-width: 1200px; margin: 0 auto; }

.row { display: flex; flex-wrap: wrap; }
.col { width: 33.333%; padding: 2%; }

.square { width: 30%; padding-bottom: 30%; background: #3b82f6; position: relative; }

.square-content { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); color: white; }

.hero { position: relative; height: 400px; background: #f3f4f6; }
.centered-box { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); width: 80%; padding: 5%; background: white; }

Dars xulosasi (Senior bilishi kerak)

  • width/height %: ota element o'lchamlariga nisbatan
  • margin/padding %: har doim ota element kengligiga nisbatan
  • position %: ota element (yoki viewport) o'lchamlariga nisbatan
  • transform translate %: elementning o'z o'lchamiga nisbatan
  • font-size %: ota element font-size iga nisbatan
  • line-height: unitless qiymat (1.5) foizdan afzal
  • Keyingi darsda: "em" o'lchov birligi - Nazorat ishi!
