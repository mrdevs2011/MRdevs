Bu darsda 49-53 darslarda o'rgangan 2D/3D transforms, transitions va animations mavzularini amaliyotda qo'llab, Natour loyihasiga professional darajadagi animatsiyalar va interaktiv elementlar qo'shamiz. Bu bilimlar zamonaviy web dizaynning ajralmas qismidir.

Bu darsda nimalarni qilamiz?

  • Kartalarga 3D flip effektlari berish
  • Navigatsiya menyusiga transition qo'shish

  • transform va transition ni to'g'ri kombinatsiyalash
  • will-change orqali performans optimizatsiyasi

Tugmalarga animatsiyalar qo'shish

CTA (Call-to-Action) tugmalariga transition va transform yordamida professional hover effektlar qo'shamiz. Bu foydalanuvchi tajribasini sezilarli darajada yaxshilaydi.

CSS
/* Asosiy tugma */
.btn:link, .btn:visited {
    display: inline-block;
    text-transform: uppercase;
    text-decoration: none;
    padding: 1.5rem 4rem;
    border-radius: 10rem;
    transition: all 0.2s ease-out;  /* Barcha xususiyatlar silliq o'zgaradi */
    position: relative;
    font-size: 1.6rem;
    will-change: transform, box-shadow;  /* Performans optimizatsiyasi */
}

.btn:hover {
    transform: translateY(-3px);  /* Yuqoriga ko'tarilish */
    box-shadow: 0 1rem 2rem rgba(0, 0, 0, 0.2);  /* Soya qo'shiladi */
}

.btn:active {
    transform: translateY(-1px);  /* Bosilganda biroz pasayish */
    box-shadow: 0 0.5rem 1rem rgba(0, 0, 0, 0.2);
}

/* Oq tugma */
.btn--white {
    background-color: #fff;
    color: #777;
}

.btn--white::after {
    background-color: #fff;
}

/* Yashil tugma */
.btn--green {
    background-color: #55c57a;
    color: #fff;
}

/* Animatsiyali tugma - entrance effect */
.btn--animated {
    animation: moveInBottom 0.5s ease-out 0.75s;
    animation-fill-mode: backwards;  /* Animatsiya boshlanishidan oldin birinchi kadr stillari */
}

@keyframes moveInBottom {
    0% {
        opacity: 0;
        transform: translateY(3rem);
    }
    100% {
        opacity: 1;
        transform: translateY(0);
    }
}
Senior biladi: will-change xususiyati brauzerga qaysi xususiyatlar o'zgarishini oldindan aytib, animatsiyani silliqroq qiladi. Faqat haqiqatan ham animatsiya bo'ladigan elementlarga qo'llang.

Kartalarga 3D Flip effekti

Tour kartalariga 3D aylanish (flip) effekti qo'shamiz. Bu orqali kartaning old tomonida rasm va sarlavha, orqa tomonida narx va "Book now" tugmasi ko'rinadi.

CSS
/* Karta konteyneri - perspective beriladi */
.card {
    perspective: 150rem;  /* 3D chuqurlik effekti - qancha kichik bo'lsa, shuncha kuchli */
    -moz-perspective: 150rem;
    position: relative;
    height: 52rem;
}

/* Kartaning ikkala tomoni */
.card__side {
    height: 52rem;
    transition: all 0.8s ease;
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    backface-visibility: hidden;  /* Orqa tomon ko'rinmasligi uchun */
    border-radius: 3px;
    overflow: hidden;
    box-shadow: 0 1.5rem 4rem rgba(0, 0, 0, 0.15);
    will-change: transform;  /* Performans uchun */
}

/* Old tomon */
.card__side--front {
    background-color: #fff;
}

/* Orqa tomon - oldindan 180 gradusga aylantirilgan */
.card__side--back {
    transform: rotateY(180deg);
}

/* Hover bo'lganda */
.card:hover .card__side--front {
    transform: rotateY(-180deg);  /* Old tomon -180 gradusga aylanadi */
}

.card:hover .card__side--back {
    transform: rotateY(0);  /* Orqa tomon normal holatga keladi */
}

/* Orqa tomon ranglari */
.card__side--back-1 {
    background-image: linear-gradient(to right bottom, #ffb900, #ff7730);
}
.card__side--back-2 {
    background-image: linear-gradient(to right bottom, #7ed56f, #28b485);
}
.card__side--back-3 {
    background-image: linear-gradient(to right bottom, #2998ff, #5643fa);
}
Muhim: backface-visibility: hidden ni unutmang! Busiz karta aylanganda orqa tomon teskari ko'rinib qoladi.

Header elementlariga entrance animatsiyalar

Sahifa yuklanganda sarlavha va tugmalar chiroyli animatsiya bilan paydo bo'ladi.

CSS
/* Sarlavha asosiy qismi - chapdan kirib keladi */
@keyframes moveInLeft {
    0% {
        opacity: 0;
        transform: translateX(-10rem);
    }
    80% {
        transform: translateX(1rem);  /* Kichik qaytish effekti */
    }
    100% {
        opacity: 1;
        transform: translateX(0);
    }
}

/* Sarlavha ikkinchi qismi - o'ngdan kirib keladi */
@keyframes moveInRight {
    0% {
        opacity: 0;
        transform: translateX(10rem);
    }
    80% {
        transform: translateX(-1rem);
    }
    100% {
        opacity: 1;
        transform: translateX(0);
    }
}

.heading-primary--main {
    animation: moveInLeft 1s ease-out;
    animation-fill-mode: backwards;  /* Animatsiya boshlanishidan oldin birinchi kadr stillari */
}

.heading-primary--sub {
    animation: moveInRight 1s ease-out;
    animation-fill-mode: backwards;
}

Navigatsiya hover effektlari

Navigatsiya linklariga hover bo'lganda chiroyli underline animatsiyasi qo'shamiz.

CSS
.navigation__link:link, .navigation__link:visited {
    display: inline-block;
    font-size: 3rem;
    font-weight: 300;
    padding: 1rem 2rem;
    color: #fff;
    text-decoration: none;
    text-transform: uppercase;
    background-image: linear-gradient(120deg, transparent 0%, transparent 50%, #fff 50%);
    background-size: 220%;
    transition: all 0.4s;
}
.navigation__link:hover, .navigation__link:active {
    background-position: 100%;
    color: #55c57a;
    transform: translateX(1rem);
}

Amaliy mashq (Senior daraja)

Vazifa: O'z loyihangizga professional animatsiyalar qo'shing.

  1. Tugmalarga hover bo'lganda translateY(-3px) va box-shadow qo'shing
  2. Kartalarga 3D flip effekti yarating
  3. Sarlavhalarga entrance animatsiyalar qo'shing
  4. Barcha animatsiyalarga will-change qo'shing

Senior darajadagi yechim:

.btn { transition: all 0.2s; will-change: transform, box-shadow; }
.btn:hover { transform: translateY(-3px); box-shadow: 0 10px 20px rgba(0,0,0,0.15); }

.card { perspective: 150rem; }
.card__side { transition: all 0.8s ease; backface-visibility: hidden; will-change: transform; }
.card__side--back { transform: rotateY(180deg); }
.card:hover .card__side--front { transform: rotateY(-180deg); }
.card:hover .card__side--back { transform: rotateY(0); }

@keyframes fadeIn { from { opacity: 0; transform: translateY(20px); } to { opacity: 1; transform: translateY(0); } }
.hero-title { animation: fadeIn 0.8s ease-out forwards; }

Dars xulosasi (Senior bilishi kerak)

  • perspective - 3D effektlar uchun ota elementga beriladi
  • backface-visibility: hidden - 3D flip da orqa tomonni yashirish
  • animation-fill-mode: backwards - sahifa yuklanganda elementlar sakrab qolmasligi uchun
  • will-change - performans optimizatsiyasi (faqat kerakli elementlarga!)
  • transition + transform - hover effektlar uchun eng yaxshi kombinatsiya
  • Keyingi darsda: Flexbox: asoslari va terminlari - Nazorat ishi!
