Global reset va rem o'rnatish

Loyihada 62.5% trick orqali 1rem = 10px qilib olamiz va barcha px larni rem ga o'tkazamiz.

CSS 
/* Global reset */
*,
*::before,
*::after {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

/* 62.5% trick - 1rem = 10px */
html {
    font-size: 62.5%;
}

/* Body - asosiy shrift */
body {
    font-family: "Lato", sans-serif;
    font-weight: 400;
    line-height: 1.7;
    color: #777;
    padding: 3rem;
    font-size: 1.6rem;  /* 16px */
}

/* px dan rem ga o'tkazish formulasi: px / 10 = rem */
.heading-primary {
    margin-bottom: 6rem;  /* 60px / 10 = 6rem */
}
px → rem formula: Dizayndagi px qiymatni 10 ga bo'ling. Masalan: 24px = 2.4rem, 8px = 0.8rem, 2px = 0.2rem.

Responsive Typography

Media queries orqali turli ekran o'lchamlari uchun shrift o'lchamlarini moslashtiramiz.

CSS 
/* Desktop-first yondashuv */
html {
    font-size: 62.5%;       /* 1rem = 10px (default) */
}

/* Planshet - 900px dan kichik */
@media (max-width: 900px) {
    html {
        font-size: 56.25%;  /* 1rem = 9px */
    }
}

/* Mobil - 600px dan kichik */
@media (max-width: 600px) {
    html {
        font-size: 50%;     /* 1rem = 8px */
    }
}

/* Barcha rem o'lchamlari avtomatik o'zgaradi! */
.heading-primary--main {
    font-size: 6rem;        /* Desktop: 60px, Planshet: 54px, Mobil: 48px */
}

Komponent spacing (em vs rem)

Komponent ichidagi spacing uchun em, global spacing uchun rem ishlatamiz.

CSS 
/* Global spacing - rem */
.section-book {
    padding: 15rem 0;
}

.row:not(:last-child) {
    margin-bottom: 8rem;
}

/* Komponent spacing - em */
.btn {
    font-size: 1.6rem;
    padding: 1.5em 4em;     /* font-size ga proporsional */
    border-radius: 10rem;
}

.btn--small {
    font-size: 1.4rem;      /* padding avtomatik 1.5em * 1.4 = 2.1rem bo'ladi */
}

.btn--large {
    font-size: 2rem;        /* padding avtomatik 1.5em * 2 = 3rem bo'ladi */
}

.feature-box {
    font-size: 1.5rem;
    padding: 2.5em;         /* 1.5rem * 2.5 = 3.75rem */
}

Viewport birliklari

To'liq ekranli sectionlar uchun vh birligidan foydalanamiz.

CSS 
/* Header - to'liq ekran */
.header {
    height: 95vh;
    position: relative;
}

/* Agar section to'liq ekran bo'lishi kerak bo'lsa */
.section-full {
    min-height: 100vh;
    display: flex;
    align-items: center;
}

/* Modal overlay - to'liq ekran */
.modal-overlay {
    position: fixed;
    top: 0;
    left: 0;
    width: 100vw;
    height: 100vh;
    background: rgba(0, 0, 0, 0.5);
}

Responsive rasmlar

Barcha rasmlar uchun max-width: 100% va height: auto berish.

CSS 
/* Barcha rasmlar responsive */
img {
    max-width: 100%;
    height: auto;
    display: block;
}

/* Composition rasmlari */
.composition__photo {
    width: 55%;
    box-shadow: 0 1.5rem 4rem rgba(0, 0, 0, 0.4);
    position: absolute;
    z-index: 10;
    transition: all 0.2s;
    outline-offset: 2rem;
}

/* Mobil uchun composition */
@media (max-width: 600px) {
    .composition__photo {
        width: 70%;
    }
}
