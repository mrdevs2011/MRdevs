RWD yordamida loyihani yaxshilash (2-qism)

19-fevral, 2021 Amaliy loyiha O'rta daraja
Bu darsda nimalarni o'rganasiz?

Bu darsda 42-darsda boshlagan ishimizni davom ettiramiz. Natour loyihasining qolgan qismlarini (cards, booking form, footer) mobil va planshet qurilmalar uchun to'liq responsive qilamiz.

Bu darsda nimalarni qilamiz?

  • Cards section ni responsive qilish
  • Booking formasini responsive qilish
  • Footer va navigatsiyani responsive qilish
  • Mobil menyu (hamburger) yaratish
  • Loyihani barcha qurilmalarda test qilish

Bu dars uchun Senior maslahatlar

Mobil menyu: Checkbox hack yoki JavaScript orqali hamburger menyu yaratish. CSS da :checked psevdo-klassidan foydalanish. Senior yodlaydi
Forma responsive: Mobil qurilmalarda inputlar to'liq kenglikda, label font-size kichikroq bo'lishi kerak. Professional
Direction-aware: Ba'zi hover effektlar faqat desktop uchun. Mobil qurilmalarda keraksiz. Best practice
Testing: Har bir o'zgarishdan keyin barcha breakpointlarda test qiling. Muhim

Cards section responsive

CSS
/* Cards section */
.section-tours { padding: 25rem 0 15rem 0; margin-top: -10rem; }

.card { perspective: 150rem; -moz-perspective: 150rem; position: relative; height: 52rem; }
.card__side { height: 52rem; transition: all 0.8s ease; position: absolute; top: 0; left: 0; width: 100%; backface-visibility: hidden; border-radius: 3px; overflow: hidden; box-shadow: 0 1.5rem 4rem rgba(0,0,0,0.15); }
.card__side--front { background-color: white; }
.card__side--back { transform: rotateY(180deg); }
.card:hover .card__side--front { transform: rotateY(-180deg); }
.card:hover .card__side--back { transform: rotateY(0); }

/* Mobil - hover effektlarni o'chirish */
@media (max-width: 37.5em), only screen and (hover: none) {
    .card { height: auto; border-radius: 3px; background-color: white; box-shadow: 0 1.5rem 4rem rgba(0,0,0,0.15); }
    .card__side { height: auto; position: relative; box-shadow: none; }
    .card__side--back { transform: rotateY(0); clip-path: polygon(0 15%, 100% 0, 100% 100%, 0 100%); }
    .card:hover .card__side--front { transform: rotateY(0); }
    .card__details { padding: 1rem 2rem; }
}

Mobil menyu (Hamburger)

CSS
/* Navigatsiya */
.navigation__checkbox { display: none; }
.navigation__button { background-color: white; height: 7rem; width: 7rem; position: fixed; top: 6rem; right: 6rem; border-radius: 50%; z-index: 2000; box-shadow: 0 1rem 3rem rgba(0,0,0,0.1); text-align: center; cursor: pointer; }

/* Mobil */
@media (max-width: 37.5em) {
    .navigation__button { top: 3rem; right: 3rem; height: 5rem; width: 5rem; }
}

.navigation__background { height: 6rem; width: 6rem; border-radius: 50%; position: fixed; top: 6.5rem; right: 6.5rem; background-image: radial-gradient(#7ed56f, #28b485); z-index: 1000; transition: transform 0.8s cubic-bezier(0.86, 0, 0.07, 1); }

@media (max-width: 37.5em) {
    .navigation__background { top: 3.5rem; right: 3.5rem; height: 4rem; width: 4rem; }
}

.navigation__nav { height: 100vh; position: fixed; top: 0; right: 0; z-index: 1500; opacity: 0; width: 0; transition: all 0.8s cubic-bezier(0.68, -0.55, 0.265, 1.55); }
.navigation__checkbox:checked ~ .navigation__background { transform: scale(80); }
.navigation__checkbox:checked ~ .navigation__nav { opacity: 1; width: 100%; }

    .card:hover .card__side--front { transform: rotateY(0); }
