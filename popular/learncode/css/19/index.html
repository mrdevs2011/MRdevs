Asosiy xossalar yordamida loyihani yaxshilash (2-qism)

26-yanvar, 2021 Amaliy loyiha O'rta daraja
Bu darsda nimalarni o'rganasiz?

Bu darsda 18-darsda boshlagan Natour loyihasini davom ettiramiz. Asosiy kontent qismini (About section) yaratamiz, sarlavhalarga stillar beramiz, "composition" rasmlar blokini yaratamiz va hover effektlar qo'shamiz.

Bu darsda nimalarni qilamiz?

  • About section uchun HTML tuzilmasini yaratish
  • Ikkilamchi sarlavhalar (heading-secondary) uchun stillar
  • Composition - bir-biriga nisbatan joylashgan rasmlar bloki
  • Rasmlarga hover effektlar va scale transform qo'shish
  • Utility klasslar yaratish (u-center-text, u-margin-bottom)

Bu dars uchun maslahatlar

Utility klasslar: Tez-tez ishlatiladigan stillarni alohida klasslar ko'rinishida yaratish kodni qayta ishlatish imkonini beradi. muhim
BEM metodologiyasi: Loyihada BEM (Block Element Modifier) nomlash metodologiyasidan foydalanamiz. Masalan: .composition__photo. yodlash shart
outline va outline-offset: Rasmlarga chiroyli chegara effekti berish uchun outline xususiyatidan foydalanamiz. amaliyot

Grid layout (Float yordamida)

Float yordamida 2 ustunli grid tizimini yaratamiz:

CSS 
/* ==================== GRID ==================== */
.row {
    max-width: 114rem;
    margin: 0 auto;
}

.row:not(:last-child) {
    margin-bottom: 8rem;
}

.row::after {
    content: "";
    display: table;
    clear: both;
}

[class^="col-"] {
    float: left;
}

[class^="col-"]:not(:last-child) {
    margin-right: 6rem;
}

.col-1-of-2 {
    width: calc((100% - 6rem) / 2);
}
[class^="col-"] selektori: Bu "col-" bilan boshlanadigan barcha klasslarni tanlaydi. Bu orqali barcha ustunlarga bir xil asosiy stillarni berish mumkin.

Sarlavha stillari

Ikkilamchi va uchlamchi sarlavhalar uchun stillar:

CSS 
/* ==================== SARLAVHALAR ==================== */
.heading-secondary {
    font-size: 3.5rem;
    text-transform: uppercase;
    font-weight: 700;
    display: inline-block;
    background-image: linear-gradient(to right, #7ed56f, #28b485);
    -webkit-background-clip: text;
    background-clip: text;
    color: transparent;
    letter-spacing: 0.2rem;
    transition: all 0.2s;
}

.heading-secondary:hover {
    transform: skewY(2deg) skewX(15deg) scale(1.1);
    text-shadow: 0.5rem 1rem 2rem rgba(0, 0, 0, 0.2);
}

.heading-tertiary {
    font-size: 1.6rem;
    font-weight: 700;
    text-transform: uppercase;
}

.paragraph {
    font-size: 1.6rem;
}

.paragraph:not(:last-child) {
    margin-bottom: 3rem;
}
background-clip: text: Bu xususiyat fon rasmi yoki gradientni faqat matn ichida ko'rsatish imkonini beradi. Matn rangi transparent bo'lishi kerak.

Composition - rasmlar kompozitsiyasi

Bir-biriga nisbatan joylashgan, hover bo'lganda kattalashadigan rasmlar bloki:

CSS 
/* ==================== COMPOSITION ==================== */
.composition {
    position: relative;
}

.composition__photo {
    width: 55%;
    box-shadow: 0 1.5rem 4rem rgba(0, 0, 0, 0.4);
    border-radius: 2px;
    position: absolute;
    z-index: 10;
    transition: all 0.2s;
    outline-offset: 2rem;
}

.composition__photo--p1 {
    left: 0;
    top: -2rem;
}

.composition__photo--p2 {
    right: 0;
    top: 2rem;
}

.composition__photo--p3 {
    left: 20%;
    top: 10rem;
}

.composition__photo:hover {
    outline: 1.5rem solid #55c57a;
    transform: scale(1.05) translateY(-0.5rem);
    box-shadow: 0 2.5rem 4rem rgba(0, 0, 0, 0.5);
    z-index: 20;
}

.composition:hover .composition__photo:not(:hover) {
    transform: scale(0.95);
}
Diqqat: outline border dan farqli o'laroq, elementning o'lchamiga ta'sir qilmaydi. outline-offset outline va element orasidagi masofani belgilaydi.

Utility klasslar

Tez-tez ishlatiladigan stillarni alohida utility klasslar ko'rinishida yaratamiz:

CSS 
/* ==================== UTILITY KLASSLAR ==================== */
.u-center-text {
    text-align: center !important;
}

.u-margin-bottom-small {
    margin-bottom: 1.5rem !important;
}

.u-margin-bottom-medium {
    margin-bottom: 4rem !important;
}

.u-margin-bottom-big {
    margin-bottom: 8rem !important;
}

.u-margin-top-big {
    margin-top: 8rem !important;
}

.u-margin-top-huge {
    margin-top: 10rem !important;
}
Utility klasslarda !important: Utility klasslar har doim ishlashi kerak bo'lgani uchun !important ishlatiladi. Bu !important ni to'g'ri ishlatish holatlaridan biridir.

btn-text stillari

CSS 
/* ==================== BTN-TEXT ==================== */
.btn-text:link,

    color: #55c57a;
    display: inline-block;
    text-decoration: none;
    border-bottom: 1px solid #55c57a;

.btn-text:hover {
    background-color: #55c57a;
    color: #fff;
    box-shadow: 0 1rem 2rem rgba(0, 0, 0, 0.15);
    transform: translateY(-2px);
}

.btn-text:active {
    box-shadow: 0 0.5rem 1rem rgba(0, 0, 0, 0.15);
    transform: translateY(0);
}

Amaliy mashq

Vazifa: Darsda yaratilgan About section ni o'zingiz takrorlab ko'ring.

  1. Grid tizimini float yordamida yarating (2 ustunli)
  2. Sarlavha uchun gradient matn yarating (background-clip: text)
  3. 3 ta rasmni bir-biriga nisbatan joylashtirib, hover effekti qo'shing
  4. Rasmlarga outline va outline-offset qo'shing
  5. Utility klasslar yarating (u-center-text, u-margin-bottom-big)
