Transitions

28-fevral, 2021 Transitions va Animations O'rta daraja
Bu darsda nimalarni o'rganasiz?

CSS da transition xususiyati yordamida element holatlari o'rtasida silliq o'tish effektlarini yaratishni o'rganasiz. Bu zamonaviy web dizayn va interaktiv elementlar uchun asosiy vosita hisoblanadi.

Bu darsda nimalarni o'rganasiz?

  • transition-property - qaysi xususiyat o'zgarishi
  • transition-duration - qancha vaqt davom etishi
  • transition-timing-function - tezlik egri chizig'i
  • transition-delay - kechikish vaqti
  • transition shorthand - barchasini bir qatorda

Bu dars uchun Senior maslahatlar

all dan qoching: transition: all performans muammolariga sabab bo'ladi. Faqat kerakli xususiyatlarni ko'rsating. Senior yodlaydi
ease-out foydalanuvchi uchun: ease-out - tez boshlanib sekin tugaydi, eng tabiiy ko'rinadi. UI/UX
GPU tezlashtirish: transform va opacity GPU da ishlaydi, silliq animatsiyalar uchun. Performance
prefers-reduced-motion: Foydalanuvchi animatsiyalarni kamaytirishni so'ragan bo'lsa, transitionlarni o'chiring. Accessibility

Transition nima?

transition xususiyati elementning bir holatdan ikkinchi holatga silliq o'tishini ta'minlaydi. Odatda :hover, :focus, :active kabi psevdo-klasslar bilan birga ishlatiladi.

CSS
/* Transitionsiz - keskin o'zgarish */
.btn {
    background: #3b82f6;
    /* hover bo'lganda birdaniga o'zgaradi */
}
.btn:hover {
    background: #2563eb;
    transform: translateY(-3px);
}

/* Transition bilan - silliq o'tish */
.btn {
    background: #3b82f6;
    transition: all 0.3s ease;  /* Barcha xususiyatlar 0.3s davomida silliq o'zgaradi */
}
.btn:hover {
    background: #2563eb;
    transform: translateY(-3px);
}

transition-property

transition-property qaysi CSS xususiyatlari transition bilan o'zgarishini belgilaydi.

CSS
/* Bitta xususiyat */
.btn {
    transition-property: background;
    transition-duration: 0.3s;
}

/* Bir nechta xususiyat */
.card {
    transition-property: transform, box-shadow, background;
    transition-duration: 0.3s;
}

/* Barcha xususiyatlar (tavsiya etilmaydi) */
.all-transition {
    transition-property: all;  /* Performans muammolari */
}

transition-duration

transition-duration transition qancha vaqt davom etishini belgilaydi. s (sekund) yoki ms (millisekund) da beriladi.

CSS
/* Sekundlarda */
.fast { transition-duration: 0.15s; }    /* 150ms - tez */
.normal { transition-duration: 0.3s; }   /* 300ms - o'rtacha */
.slow { transition-duration: 0.5s; }     /* 500ms - sekin */

/* Millisekundlarda */
.ms-example { transition-duration: 200ms; }

/* Bir nechta xususiyat uchun turli vaqtlar */
.multiple {
    transition-property: background, transform;
    transition-duration: 0.3s, 0.5s;  /* background 0.3s, transform 0.5s */
}

transition-timing-function

transition-timing-function transition tezligining vaqt bo'yicha qanday o'zgarishini belgilaydi.

CSS
/* Asosiy timing funksiyalari */
.ease { transition-timing-function: ease; }           /* default - sekin-tez-sekin */
.linear { transition-timing-function: linear; }       /* bir xil tezlik */
.ease-in { transition-timing-function: ease-in; }     /* sekin boshlanib tezlashadi */
.ease-out { transition-timing-function: ease-out; }   /* tez boshlanib sekinlashadi ⭐ */
.ease-in-out { transition-timing-function: ease-in-out; } /* sekin-tez-sekin */

/* Custom cubic-bezier */
.custom { transition-timing-function: cubic-bezier(0.68, -0.55, 0.27, 1.55); }
Senior biladi: ease-out eng tabiiy ko'rinadi va UI elementlar uchun eng yaxshi tanlovdir.

transition-delay

transition-delay transition boshlanishidan oldin qancha kutish kerakligini belgilaydi.

CSS
/* Kechikish */
.delayed { transition-delay: 0.2s; }    /* 0.2s kutib, keyin boshlanadi */

/* Staggered effekt */
.menu-item { transition: all 0.3s; }
.menu-item:nth-child(1) { transition-delay: 0s; }
.menu-item:nth-child(2) { transition-delay: 0.1s; }
.menu-item:nth-child(3) { transition-delay: 0.2s; }

Transition shorthand

transition shorthand barcha xususiyatlarni bir qatorda yozish imkonini beradi.

CSS
/* Shorthand sintaksisi */
/* transition: property duration timing-function delay; */

.btn {
    transition: all 0.3s ease;
}

.btn-2 {
    transition: background 0.2s ease-out, transform 0.3s cubic-bezier(0.68, -0.55, 0.27, 1.55);
}

.card {
    transition: transform 0.3s, box-shadow 0.3s, background 0.2s;
}

Amaliy mashq

Vazifa: Transition yordamida interaktiv tugmalar yarating.

  1. Tugmaga hover bo'lganda background va transform o'zgarsin
  2. transition-duration: 0.3s va ease-out ishlating
  3. Karta uchun transform va box-shadow transition qo'shing

Namunaviy yechim:

.btn { background: #3b82f6; padding: 12px 24px; border: none; border-radius: 8px; color: white; cursor: pointer; transition: all 0.3s ease-out; }
.btn:hover { background: #2563eb; transform: translateY(-3px); box-shadow: 0 10px 20px rgba(0,0,0,0.1); }

.card { background: white; padding: 20px; border-radius: 12px; box-shadow: 0 4px 6px rgba(0,0,0,0.05); transition: transform 0.3s, box-shadow 0.3s; }
.card:hover { transform: scale(1.02); box-shadow: 0 15px 30px rgba(0,0,0,0.1); }

Dars xulosasi

  • transition-duration: qancha vaqt (s yoki ms)
  • transition-timing-function: ease, ease-out, linear
  • transition-delay: kechikish
  • Shorthand: transition: property duration timing delay;
  • Keyingi darsda: Animations (1-qism)
