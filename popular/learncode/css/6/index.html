Specificity (O'ziga xoslik)

13-yanvar, 2021 CSS Asoslari O'rta daraja
Bu darsda nimalarni o'rganasiz?

CSS da bir elementga bir nechta qoida berilganda, qaysi biri ustunlik qilishini belgilaydigan "Specificity" (o'ziga xoslik) tushunchasini o'rganasiz. Bu CSS ning eng muhim va eng ko'p chalkashtiradigan mavzularidan biri.

Bu darsda nimalarni o'rganasiz?

  • Specificity nima va u qanday ishlaydi?
  • Specificity ball tizimi (0,0,0,0)
  • ID vs Class vs Element - qaysi biri kuchliroq?
  • !important nima va qachon ishlatiladi?
  • Specificity ni hisoblash qoidalari

Bu dars uchun maslahatlar

Ball tizimini eslab qoling: ID = 100 ball, Class = 10 ball, Element = 1 ball. Ballar yig'indisi yuqori bo'lgan qoida g'alaba qiladi. yodlash shart

Class selektor ishlating: Specificity muammolarini oldini olish uchun CSS da ID o'rniga class selektordan foydalaning. amaliyot

Specificity nima?

Specificity (o'ziga xoslik) - bu brauzerning qaysi CSS qoidasi elementga qo'llanilishini aniqlash usuli. Agar bir elementga bir nechta qoida berilgan bo'lsa, eng yuqori specificity ga ega bo'lgan qoida g'alaba qiladi.

Oddiy tushuntirish

Tasavvur qiling, siz sinfdagi o'quvchilarga murojaat qilyapsiz:

"Hamma o'quvchilar"

Element selektor (1 ball)

"O'g'il bolalar"

Class selektor (10 ball)

"Aziz Karimov"

ID selektor (100 ball)

Qanchalik aniq (specific) bo'lsa, shunchalik kuchli!

Specificity ball tizimi

Specificity 4 xonali raqam sifatida hisoblanadi: (0,0,0,0)

Selektor turi Ball Misol
Inline style 1,0,0,0 (1000 ball) <p style="color: red;">
ID selektor 0,1,0,0 (100 ball) #header
Class, Pseudo-class, Attribut 0,0,1,0 (10 ball) .button, :hover, [type="text"]
Element, Pseudo-element 0,0,0,1 (1 ball) p, div, ::before
Universal selektor (*) 0,0,0,0 (0 ball) *

Specificity misoli

Quyidagi misolda bir xil elementga uch xil qoida berilgan. Qaysi biri g'alaba qiladi?

CSS 
/* 1-qoida: Element selektor (1 ball) */
p {
    color: green;
}

/* 2-qoida: Class selektor (10 ball) */
.text {
    color: blue;
}

/* 3-qoida: ID selektor (100 ball) - G'OLIB! */
#special {
    color: red;
}
HTML 
<p class="text" id="special">Bu matn qizil rangda bo'ladi</p>
Natija: Matn qizil rangda bo'ladi, chunki ID selektor (100 ball) eng yuqori specificity ga ega.

Murakkab specificity hisoblash

Bir nechta selektor birgalikda ishlatilganda, ballar qo'shiladi:

CSS 
/* div p - 2 ta element = 2 ball */
div p {
    color: orange;
}

/* .content p - 1 class + 1 element = 11 ball */
.content p {
    color: purple;
}

/* #main .text - 1 ID + 1 class = 110 ball - G'OLIB! */
#main .text {
    color: navy;
}
Selektor ID Class Element Jami ball
div p 0 0 2 0,0,0,2
.content p 0 1 1 0,0,1,1
#main .text 1 1 0 0,1,1,0 ⭐

!important

!important - bu maxsus kalit so'z bo'lib, u barcha specificity qoidalarini bekor qiladi va o'z qoidasini eng kuchli qiladi.

CSS 
/* Oddiy qoida */
p {
    color: blue;
}

/* !important bilan - bu g'alaba qiladi! */
p {
    color: red !important;
}
!important dan qoching! !important specificity zanjirini buzadi va kodni boshqarishni qiyinlashtiradi. Faqat quyidagi hollarda ishlating:
  • Uchinchi tomon kutubxonalarini bekor qilishda
  • Foydalanuvchi stillari (user styles) ni bekor qilishda
  • Vaqtinchalik test uchun

Bilimingizni tekshiring

Amaliy mashq

Vazifa: Specificity ni tushunganingizni tekshirish uchun quyidagi misollarni hisoblang.

  1. #header .nav a - necha ball?
  2. div.container > p.text - necha ball?
  3. * - necha ball?
  4. style="color: red;" - necha ball?
  5. Quyidagi kodda <p class="intro" id="first"> elementi qanday rangda bo'ladi? 
    p { color: green; }
.intro { color: blue; }
#first { color: red; }

Javoblar:

  1. #header .nav a = ID (100) + Class (10) + Element (1) = 111 ball
  2. div.container > p.text = 2 ta Element (div, p) + 2 ta Class (container, text) = 22 ball
  3. * = Universal selektor = 0 ball
  4. style="color: red;" = Inline style = 1000 ball
  5. Matn qizil rangda bo'ladi, chunki ID selektor (100 ball) eng yuqori specificity ga ega.

Dars xulosasi

  • Specificity - CSS qoidalarining ustunlik tartibini belgilaydi
  • Ball tizimi: Inline (1000) > ID (100) > Class (10) > Element (1) > Universal (0)
  • Ballar qo'shiladi: #main .text = 110 ball
  • !important barcha qoidalarni bekor qiladi, lekin uni kam ishlating
  • Keyingi darsda: Inheritance (Meros olish)
