Joylashish (Position) va "static" qiymati

Bu darsda nimalarni o'rganasiz?

CSS da position xossasi va uning asosiy qiymatlari haqida tushuncha olasiz. static qiymati - bu barcha elementlarning default holati bo'lib, elementlar HTML dagi normal oqim (normal flow) bo'yicha joylashadi.

Position xossasi nima?

position xossasi elementning sahifada qanday joylashishini belgilaydi. Bu CSS dagi eng muhim xususiyatlardan biri bo'lib, murakkab layoutlar yaratish imkonini beradi.

CSS 
/* Position qiymatlari */
.element {
    position: static;      /* Default - normal oqim */
    position: relative;    /* Nisbiy - o'z joyiga nisbatan */
    position: absolute;    /* Absolyut - ota elementga nisbatan */
    position: fixed;       /* Qat'iy - viewport ga nisbatan */
    position: sticky;      /* Yopishqoq - scroll bo'lganda */
}

Normal Flow (Normal oqim)

Normal flow - bu elementlarning hech qanday position yoki float berilmagandagi tabiiy joylashuv tartibi.

Block elementlar

Vertikal (yuqoridan pastga) joylashadi. Har biri yangi qatordan boshlanadi va to'liq kenglikni egallaydi.

Inline elementlar

Gorizontal (chapdan o'ngga) joylashadi. Bir qatorda ketma-ket turadi.

Margin collapsing

Vertikal marginlar ustma-ust tushganda kattasi qoladi (13-darsda o'tilgan).

Z-index ishlamaydi

Normal flow da elementlar HTML dagi tartib bo'yicha joylashadi.

CSS 
/* Normal flow dagi elementlar */
div {
    /* Hech qanday position berilmagan - static */
    margin: 10px 0;  /* Vertikal joylashuv */
}

span {
    /* Inline element - gorizontal joylashuv */
    margin: 0 10px;
}

Position: static

position: static - bu barcha elementlarning default qiymati. Element normal oqimda qoladi va top, right, bottom, left xususiyatlari hech qanday ta'sir qilmaydi.

CSS 
/* static - default holat */
.static-box {
    position: static;  /* Bu yozilmasa ham bo'ladi */
    top: 100px;        /* ISHLAMAYDI! */
    left: 50px;        /* ISHLAMAYDI! */
    background: #3b82f6;
    padding: 20px;
}

/* Normal oqimda qoladi */
.normal-flow {
    /* position berilmagan = static */
    margin: 20px;
    padding: 15px;
}
Diqqat: position: static bo'lganda top, right, bottom, left va z-index xususiyatlari ISHLAMAYDI!
Qachon static ishlatiladi? Aslida static ni alohida yozish shart emas, chunki bu default holat. Lekin ba'zida media queries da boshqa position qiymatini bekor qilish uchun ishlatiladi.

Offset xususiyatlari

top, right, bottom, left xususiyatlari elementning joylashuvini aniq belgilash uchun ishlatiladi. Lekin ular faqat position static bo'lmaganda ishlaydi!

CSS 
/* Offset xususiyatlari */
.positioned {
    position: relative;  /* yoki absolute, fixed, sticky */
    top: 20px;           /* Yuqoridan 20px pastga */
    right: 30px;         /* O'ngdan 30px chapga */
    bottom: 40px;        /* Pastdan 40px yuqoriga */
    left: 50px;          /* Chapdan 50px o'ngga */
}

/* Barcha offsetlar birgalikda */
.offset-example {
    position: absolute;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    /* Element ota konteynerni to'liq qoplaydi */
}
Musbat va manfiy qiymatlar: Musbat qiymatlar elementni ichkariga, manfiy qiymatlar tashqariga siljitadi. Masalan, top: -10px elementni yuqoriga 10px siljitadi.

Vazifa: Position xususiyatlarini tushunganingizni tekshiring.

  1. 3 ta div elementi yarating va ularga fon rangi bering
  2. Ularning default holatda (static) qanday joylashishini kuzating
  3. Har bir div ga margin va padding berib ko'ring
  4. Elementlarga top va left xususiyatlarini berib ko'ring - ishlamasligini kuzating
  5. Xulosa qiling: static holatda nimalar ishlaydi, nimalar ishlamaydi?

Xulosa: position: static holatda:

  • ✅ margin, padding, width, height - ISHLAYDI
  • ✅ color, background, border - ISHLAYDI
  • ❌ top, right, bottom, left - ISHLAMAYDI
  • ❌ z-index - ISHLAMAYDI
