HTML hujjat va CSS. Loyiha strukturasi

10-yanvar, 2021 CSS Asoslari Boshlang'ich daraja
Bu darsda nimalarni o'rganasiz?

CSS ni HTML hujjatiga ulashning 3 xil usulini (inline, internal, external), har bir usulning afzallik va kamchiliklarini, va professional loyihalar uchun to'g'ri papka tuzilmasini o'rganasiz. Bu bilimlar har bir web dasturchi uchun fundamental ahamiyatga ega.

Bu darsda nimalarni o'rganasiz?

  • CSS ni HTML ga ulashning 3 ta usuli
  • Inline CSS - satr ichidagi stillar
  • Internal CSS - ichki stillar
  • External CSS - tashqi stillar (eng tavsiya etilgan)
  • Professional loyiha papka tuzilmasi

Bu dars uchun maslahatlar

External CSS dan foydalaning: Har doim CSS ni alohida faylda saqlang. Bu kodni qayta ishlatish, brauzer kesh xotirasi va jamoaviy ishlash uchun eng yaxshi usul. yodlash shart
Inline CSS dan qoching: Inline CSS faqat juda kam holatlarda, masalan JavaScript orqali dinamik stillashda ishlatiladi. Doimiy stillar uchun ishlatmang. tushunish kerak
Papka tuzilmasi muhim: CSS fayllarni css/ papkasida, rasmlarni img/ papkasida saqlang. Bu loyihangizni tartibli saqlaydi. amaliyot

1-usul: Inline CSS (Satr ichidagi)

Inline CSS - bu CSS kodlarini to'g'ridan-to'g'ri HTML elementining style atributi ichida yozish usuli. Bu usul faqat bitta elementga ta'sir qiladi.

HTML 
<p style="color: red; font-size: 20px;">
    Bu qizil va katta matn
</p>

<div style="background-color: #f0f0f0; padding: 15px; border-radius: 8px;">
    Bu och kulrang fonli va yumaloq burchakli blok
</div>

Afzalliklari

  • Tez va oson test qilish uchun qulay
  • JavaScript orqali dinamik o'zgartirish uchun
  • Alohida fayl kerak emas

Kamchiliklari

  • Kod takrorlanadi (har bir elementga alohida yozish kerak)
  • HTML va CSS aralashib ketadi, kod o'qilishi qiyinlashadi
  • Katta loyihalarda boshqarish juda qiyin
  • Specificity muammolarini keltirib chiqaradi
Muhim: Inline CSS ni faqat test uchun yoki JavaScript orqali dinamik stillash kerak bo'lganda ishlating. Doimiy stillar uchun bu usuldan foydalanmang!

2-usul: Internal CSS (Ichki)

Internal CSS - CSS kodlarini HTML hujjatining <head> qismida <style> tegi ichida yozish usuli. Bu usul faqat bitta sahifaga ta'sir qiladi.

HTML 
<!DOCTYPE html>
<html lang="uz">
<head>
    <meta charset="UTF-8">
    <title>Sahifa nomi</title>
    <style>
        h1 {
            color: navy;
            text-align: center;
            font-family: 'Segoe UI', sans-serif;
        }
        
        .highlight {
            background-color: #fef3c7;
            padding: 10px;
            border-left: 4px solid #f59e0b;
        }
        
        #main-content {
            max-width: 800px;
            margin: 0 auto;
        }
    </style>
</head>
<body>
    <h1>Salom Dunyo!</h1>
    <p class="highlight">Bu sariq fonli matn</p>
</body>
</html>

Afzalliklari

  • Bitta sahifa uchun barcha stillar bir joyda
  • Inline CSS ga qaraganda ancha toza va tartibli
  • Alohida fayl yaratish shart emas

Kamchiliklari

  • Bir nechta sahifalarda kod takrorlanadi
  • HTML fayl hajmi kattalashadi
  • Brauzer kesh xotirasida saqlanmaydi (har safar yuklanadi)
  • Katta loyihalar uchun noqulay

3-usul: External CSS (Tashqi) ⭐ Tavsiya etiladi

External CSS - CSS kodlarini alohida .css faylda saqlash va <link> tegi orqali HTML ga ulash usuli. Bu eng professional va keng tarqalgan usul.

HTML (index.html) 
<!DOCTYPE html>
<html lang="uz">
<head>
    <meta charset="UTF-8">
    <title>Sahifa nomi</title>
    <!-- CSS faylni ulash -->
    <link rel="stylesheet" href="css/style.css">
</head>
<body>
    <h1>Salom Dunyo!</h1>
    <p class="text">Bu paragraf</p>
</body>
</html>
CSS (css/style.css) 
/* style.css */
h1 {
    color: navy;
    text-align: center;
    font-family: 'Segoe UI', sans-serif;
}

.text {
    font-size: 16px;
    line-height: 1.6;
    color: #333;
}

Afzalliklari

  • HTML va CSS ajratilgan - kod toza va o'qishga qulay
  • Bitta CSS fayl ko'plab sahifalarda ishlatiladi - kod takrorlanmaydi
  • Brauzer kesh xotirasida saqlanadi - sahifa tez yuklanadi
  • Katta loyihalar uchun ideal
  • Jamoa bilan ishlash oson

Kamchiliklari

  • Alohida fayl yaratish kerak
  • CSS fayl yuklanmaguncha sahifa ko'rinishi buzilishi mumkin (FOUC)
Professional maslahat: Har doim External CSS dan foydalaning. Bu nafaqat kodni toza saqlaydi, balki brauzer kesh xotirasi tufayli saytingiz tezroq yuklanadi.

Professional loyiha strukturasi

Katta loyihalarda fayllarni tartibli saqlash juda muhim. Quyida professional loyihalar uchun tavsiya etilgan papka tuzilmasi keltirilgan:

loyiha-nomi/
├── index.html
├── about.html
├── contact.html
│
├── css/
│   ├── style.css
│   ├── responsive.css
│   └── normalize.css
│
├── js/
│   ├── main.js
│   └── utils.js
│
├── img/
│   ├── logo.svg
│   ├── hero-bg.jpg
│   └── icons/
│       ├── icon1.svg
│       └── icon2.svg
│
├── fonts/
│   ├── custom-font.woff2
│   └── custom-font.woff
│
└── assets/
    └── videos/
        └── intro.mp4
Papkalar tavsifi:
  • css/ - barcha CSS fayllar (style.css, responsive.css, normalize.css)
  • js/ - barcha JavaScript fayllar
  • img/ - barcha rasm fayllar
  • fonts/ - maxsus shrift fayllari
  • assets/ - video, audio va boshqa media fayllar
Maslahat: CSS fayllaringizni vazifasiga qarab ajrating. Masalan: style.css (asosiy stillar), responsive.css (media queries), normalize.css (brauzer default stillarini nollash).

Bilimingizni tekshiring

3-dars testi
1 CSS ni HTML ga ulashning eng tavsiya etilgan usuli qaysi?
2 External CSS ni HTML ga ulash uchun qaysi teg ishlatiladi?
3 Inline CSS qaysi atribut orqali yoziladi?

Amaliy mashq

Vazifa: External CSS usulidan foydalanib, oddiy web sahifa yarating.

  1. Yangi papka yarating va uni mening-loyiham deb nomlang
  2. Ichida index.html va css/style.css fayllarini yarating
  3. index.html da asosiy HTML strukturani yozing
  4. style.css da quyidagi stillarni yozing:
    • h1 - ko'k rang, markazga tekislangan
    • p - 18px o'lcham, qator oralig'i 1.6
    • .box - och kulrang fon, 20px ichki bo'shliq, yumaloq burchaklar

Namunaviy yechim:

index.html:

<!DOCTYPE html>
<html lang="uz">
<head>
    <meta charset="UTF-8">
    <title>Mening Loyiham</title>
    <link rel="stylesheet" href="css/style.css">
</head>
<body>
    <h1>Mening birinchi CSS sahifam</h1>
    <p>Bu paragraf 18px o'lchamda va qator oralig'i 1.6.</p>
    <div class="box">
        Bu och kulrang fonli, ichki bo'shliqli va yumaloq burchakli blok.
    </div>
</body>
</html>

css/style.css:

h1 {
    color: #2563eb;
    text-align: center;
}

p {
    font-size: 18px;
    line-height: 1.6;
}

.box {
    background-color: #f3f4f6;
    padding: 20px;
    border-radius: 8px;
}

Dars xulosasi

  • Inline CSS - style atributi orqali, faqat test uchun
  • Internal CSS - <style> tegi ichida, bitta sahifa uchun
  • External CSS - <link> tegi orqali, professional loyihalar uchun ⭐
  • CSS fayllarni css/ papkasida saqlang
  • Keyingi darsda: Selectors (Tanlab oluvchilar)
