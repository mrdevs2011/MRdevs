Bu darsda nimalarni o'rganasiz?

Bu darsda 1-17 darslarda o'rgangan barcha CSS xossalarini amaliyotda qo'llab, Natour loyihasini yaxshilashni boshlaymiz. Birinchi qismda loyihaning asosiy tuzilmasini yaratamiz va header qismiga stillar beramiz.

Bu darsda nimalarni qilamiz?

  • Loyiha papka tuzilmasini tayyorlash
  • Asosiy CSS faylini yaratish va ulash
  • Global reset va asosiy stillarni yozish
  • Header qismiga stillar berish
  • Logo va navigatsiya menyusini stillash

Bu dars uchun maslahatlar

Global reset va asosiy stillar

CSS faylimizning boshida global reset va loyiha uchun kerakli o'zgaruvchilarni e'lon qilamiz:

CSS (style.css) 
/* ==================== GLOBAL RESET ==================== */
*,
*::before,
*::after {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

/* ==================== CSS O'ZGARUVCHILARI ==================== */
:root {
    --color-primary: #55c57a;
    --color-primary-light: #7ed56f;
    --color-primary-dark: #28b485;
    
    --color-grey-dark: #777;
    --color-grey-dark-2: #999;
    --color-grey-dark-3: #333;
    
    --color-white: #fff;
    --color-black: #000;
    
    --default-font-size: 1.6rem;
}

/* ==================== ASOSIY STILLAR ==================== */
html {
    font-size: 62.5%; /* 1rem = 10px */
}

body {
    font-family: 'Lato', sans-serif;
    font-weight: 400;
    line-height: 1.7;
    color: var(--color-grey-dark);
    padding: 3rem;
}
font-size: 62.5% trick: Brauzer default shrift o'lchami 16px. 62.5% berilsa, 1rem = 10px bo'ladi. Bu hisob-kitoblarni juda osonlashtiradi.

Header qismiga stillar berish

Header qismi to'liq ekran balandligida bo'lishi va chiroyli fon rasmiga ega bo'lishi kerak:

CSS 
/* ==================== HEADER ==================== */
.header {
    height: 95vh;
    background-image: linear-gradient(
        to right bottom,
        rgba(126, 213, 111, 0.8),
        rgba(40, 180, 133, 0.8)
    ),
    url(../img/hero.jpg);
    background-size: cover;
    background-position: top;
    position: relative;
    
    /* Clip path - header ga shakl berish */
    clip-path: polygon(0 0, 100% 0, 100% 75vh, 0 100%);
}

.header__logo-box {
    position: absolute;
    top: 4rem;
    left: 4rem;
}

.header__logo {
    height: 3.5rem;
}

.header__text-box {
    position: absolute;
    top: 40%;
    left: 50%;
    transform: translate(-50%, -50%);
    text-align: center;
}
clip-path: Bu xususiyat elementga turli shakllar berish uchun ishlatiladi. Bizning holatda header pastki qismi qiya kesilgan ko'rinishga ega bo'ladi.

Sarlavha stillari

Asosiy sarlavha ikki qismdan iborat - asosiy matn va ikkinchi darajali matn:

CSS 
/* ==================== SARLAVHALAR ==================== */
.heading-primary {
    color: var(--color-white);
    text-transform: uppercase;
    margin-bottom: 6rem;
}

.heading-primary--main {
    display: block;
    font-size: 6rem;
    font-weight: 400;
    letter-spacing: 3.5rem;
    animation: moveInLeft 1s ease-out;
}

.heading-primary--sub {
    display: block;
    font-size: 2rem;
    font-weight: 700;
    letter-spacing: 1.75rem;
    animation: moveInRight 1s ease-out;
}

/* ==================== ANIMATSIYALAR ==================== */
@keyframes moveInLeft {
    0% {
        opacity: 0;
        transform: translateX(-10rem);
    }
    80% {
        transform: translateX(1rem);
    }
    100% {
        opacity: 1;
        transform: translate(0);
    }
}

@keyframes moveInRight {
    0% {
        opacity: 0;
        transform: translateX(10rem);
    }
    80% {
        transform: translateX(-1rem);
    }
    100% {
        opacity: 1;
        transform: translate(0);
    }
}

Tugma stillari

Call-to-action tugmasi uchun stillar va hover animatsiyasi:

CSS 
/* ==================== TUGMALAR ==================== */
.btn:link,
.btn:visited {
    display: inline-block;
    text-transform: uppercase;
    text-decoration: none;
    padding: 1.5rem 4rem;
    border-radius: 10rem;
    transition: all 0.2s;
    position: relative;
    font-size: 1.6rem;
}

.btn:hover {
    transform: translateY(-3px);
    box-shadow: 0 1rem 2rem rgba(0, 0, 0, 0.2);
}

.btn:active {
    transform: translateY(-1px);
    box-shadow: 0 0.5rem 1rem rgba(0, 0, 0, 0.2);
}

.btn--white {
    background-color: var(--color-white);
    color: var(--color-grey-dark);
}

.btn--animated {
    animation: moveInBottom 0.5s ease-out 0.75s;
    animation-fill-mode: backwards;
}

@keyframes moveInBottom {
    0% {
        opacity: 0;
        transform: translateY(3rem);
    }
    100% {
        opacity: 1;
        transform: translate(0);
    }
}
animation-fill-mode: backwards: Bu xususiyat animatsiya boshlanishidan oldin elementga animatsiyaning birinchi kadridagi stillarni qo'llaydi. Bu sahifa yuklanganda elementlarning birdaniga paydo bo'lib qolishini oldini oladi.

Amaliy mashq

Vazifa: Darsda yaratilgan header qismini o'zingiz takrorlab ko'ring.

  1. Yangi papka yarating va index.html, css/style.css fayllarini yarating
  2. HTML da header, logo va sarlavha elementlarini yarating
  3. CSS da global reset va font-size: 62.5% ni qo'llang
  4. Header ga fon rasmi va gradient qo'shing
  5. Sarlavhaga animatsiya qo'shing (chapdan va o'ngdan kirib kelish)
  6. Tugmaga hover effekti va pastdan kirib kelish animatsiyasi qo'shing

Namunaviy yechim (darsdagi kod):

To'liq kodni dars davomida yozilgan CSS va HTML dan nusxalashingiz mumkin. Yuqoridagi kod bloklarida barcha kerakli stillar keltirilgan.

Dars xulosasi

  • Global reset: * { margin: 0; padding: 0; box-sizing: border-box; }
  • font-size: 62.5%: 1rem = 10px qilish uchun qulay trick
  • CSS o'zgaruvchilari: :root { --color-primary: #55c57a; }
  • clip-path: Elementlarga turli shakllar berish uchun
  • Markazga tekislash: position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%);
