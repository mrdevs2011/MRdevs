Box model (Quti modeli). Margin, Padding, Border

19-yanvar, 2021 CSS Asoslari Boshlang'ich daraja
Bu darsda nimalarni o'rganasiz?

CSS da har bir element to'rtburchak quti sifatida ko'riladi. Box model bu qutining tuzilishini tushuntiradi: content (mazmun), padding (ichki bo'shliq), border (chegara) va margin (tashqi bo'shliq). Bu CSS ning eng fundamental tushunchalaridan biri.

Bu darsda nimalarni o'rganasiz?

  • Box model nima va u qanday tuzilgan?
  • Content - elementning asosiy mazmuni
  • Padding - ichki bo'shliq (content va border orasida)
  • Border - element chegarasi
  • Margin - tashqi bo'shliq (elementlar orasidagi masofa)

Bu dars uchun maslahatlar

Qatlamlarni eslab qoling: Ichkaridan tashqariga: Content → Padding → Border → Margin.yodlash shart
Margin va Padding farqi: Margin - tashqi bo'shliq (elementlar orasida), Padding - ichki bo'shliq (kontent atrofida).muhim
Brauzer DevTools: Elementni tekshirish (Inspect) qilganda, box model rangli qatlamlarda ko'rinadi: ko'k (content), yashil (padding), sariq (border), to'q sariq (margin).amaliyot

Box model nima?

CSS da har bir HTML elementi to'rtburchak quti ichida joylashgan deb qaraladi. Box model bu qutining tuzilishini tavsiflaydi va 4 ta qatlamdan iborat:

Box model qatlamlari
Margin - Tashqi bo'shliq (elementlar orasida)
Border - Chegara chizig'i
Padding - Ichki bo'shliq
Content - Asosiy mazmun (matn, rasm)

Content (Mazmun)

Content - bu elementning asosiy mazmuni: matn, rasm, video yoki boshqa HTML elementlari. Content o'lchamlarini width va height xususiyatlari bilan belgilash mumkin.

CSS
.box {
    width: 300px;
    height: 200px;
    background-color: #93c5fd;
}

Padding (Ichki bo'shliq)

Padding - bu content va border orasidagi bo'shliq. Padding element ichidagi mazmunni chegaradan uzoqlashtiradi. Padding qiymati elementning umumiy o'lchamiga qo'shiladi.

CSS
/* Hamma tomon uchun bir xil */
.box {
    padding: 20px;
}

/* Har bir tomon uchun alohida */
.box {
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 15px;
    padding-left: 25px;
}

/* Qisqa yozuv (shorthand): top right bottom left */
.box {
    padding: 10px 20px 15px 25px;
}
Eslab qoling: Padding qiymatlari soat strelkasi bo'yicha: yuqori (12), o'ng (3), past (6), chap (9).

Border (Chegara)

Border - bu padding va margin orasidagi chegara chizig'i. Border uchta xususiyatdan iborat: qalinlik (border-width), uslub (border-style) va rang (border-color).

CSS
/* Qisqa yozuv */
.box {
    border: 2px solid #3b82f6;
}

/* Alohida yozuv */
.box {
    border-width: 2px;
    border-style: solid; /* solid, dashed, dotted, double */
    border-color: #3b82f6;
}

/* Har bir tomon alohida */
.box {
    border-top: 3px dashed #dc2626;
    border-bottom: 1px solid #e2e8f0;
}

Margin (Tashqi bo'shliq)

Margin - bu elementning tashqi bo'shlig'i, ya'ni border dan tashqaridagi bo'shliq. Margin elementlar orasidagi masofani boshqaradi.

CSS
/* Hamma tomon uchun bir xil */
.box {
    margin: 20px;
}

/* Qisqa yozuv: top right bottom left */
.box {
    margin: 10px 20px 15px 25px;
}

/* Auto - elementni markazlashtirish */
.box {
    width: 300px;
    margin-left: auto;
    margin-right: auto;
}

/* Markazlashtirish uchun qisqa yozuv */
.box {
    width: 300px;
    margin: 0 auto;
}
Block elementlarni markazlashtirish: margin: 0 auto; block elementlarni gorizontal markazlashtirishning eng oson yo'li.

Box model xulosasi

QatlamTavsifiAsosiy xususiyat

PaddingContent va border orasidagi ichki bo'shliqpadding

Amaliy mashq

Vazifa: Box model qatlamlaridan foydalanib, karta komponentini yarating.

  1. 300px kenglikdagi .card elementini yarating
  2. Ichiga rasm, sarlavha va matn qo'shing
  3. padding: 20px bering
  4. border: 1px solid #e2e8f0 va border-radius: 12px bering
  5. margin: 20px bering
  6. Ikkita karta yaratib, ular orasidagi masofani margin orqali boshqaring

Namunaviy yechim:

<div class="card">
    <img src="image.jpg" alt="Rasm" style="width: 100%; border-radius: 8px;">
    <h3>Karta sarlavhasi</h3>
    <p>Karta ichidagi qisqacha matn.</p>
</div>

<style>
.card {
    width: 300px;
    padding: 20px;
    border: 1px solid #e2e8f0;
    border-radius: 12px;
    margin: 20px;
    background: white;
    box-shadow: 0 4px 6px rgba(0,0,0,0.05);
}

.card h3 {
    margin-top: 15px;
    margin-bottom: 10px;
    color: #1e293b;
}

.card p {
    color: #64748b;
    margin: 0;
}
</style>

Dars xulosasi

  • Box model 4 qatlamdan iborat: Content → Padding → Border → Margin
  • Content - asosiy mazmun, width/height bilan boshqariladi
  • Padding - ichki bo'shliq, kontentni chegaradan uzoqlashtiradi
  • Border - chegara chizig'i, border: qalinlik uslub rang
  • Margin - tashqi bo'shliq, elementlar orasidagi masofa
  • Keyingi darsda: Margin collapsing. Shorthands (Qisqartmalar)
