RWD yordamida loyihani yaxshilash (1-qism)

18-fevral, 2021 Amaliy loyiha O'rta daraja
Bu darsda nimalarni o'rganasiz?

Bu darsda 39-41 darslarda o'rgangan RWD, Media queries va Breakpoints mavzularini amaliyotda qo'llab, Natour loyihasini mobil va planshet qurilmalar uchun responsive qilishni boshlaymiz.

Bu darsda nimalarni qilamiz?

  • Mobile-first yondashuvda media querylar yozish
  • Header va navigatsiyani responsive qilish
  • Grid tizimini turli ekranlarga moslash
  • Rasmlarni responsive qilish
  • Shrift o'lchamlarini ekranga qarab o'zgartirish

Bu dars uchun Senior maslahatlar

Mobile-first: Avval mobil uchun default stillar, keyin katta ekranlar uchun min-width media querylar. Senior yodlaydi
Breakpointlar: 600px (planshet), 900px (desktop), 1200px (katta desktop) - kontentga qarab. Professional
em media queryda: @media (min-width: 37.5em) - 600px / 16px = 37.5em. Best practice
Test qilish: Chrome DevTools da turli qurilmalarda testing qilishni unutmang. Muhim

Global reset va breakpointlar

CSS
/* Global reset */
*, *::before, *::after { margin: 0; padding: 0; box-sizing: border-box; }

html { font-size: 62.5%; }  /* 1rem = 10px */

/* Breakpointlar uchun o'zgaruvchilar */
:root {
    --bp-mobile: 37.5em;    /* 600px */
    --bp-tablet: 56.25em;   /* 900px */
    --bp-desktop: 75em;     /* 1200px */
}

body { font-family: "Lato", sans-serif; font-weight: 400; line-height: 1.7; color: #777; padding: 3rem; font-size: 1.6rem; }

CSS
.header { height: 95vh; position: relative; }

}

Grid tizimini responsive qilish

CSS
.row { max-width: 114rem; margin: 0 auto; }
.row:not(:last-child) { margin-bottom: 8rem; }
.row::after { content: ""; display: table; clear: both; }

[class^="col-"] { float: left; }
[class^="col-"]:not(:last-child) { margin-right: 6rem; }

.col-1-of-2 { width: calc((100% - 6rem) / 2); }
.col-1-of-3 { width: calc((100% - 2 * 6rem) / 3); }
.col-1-of-4 { width: calc((100% - 3 * 6rem) / 4); }

/* Planshet - 900px dan kichik */
@media (max-width: 56.25em) {
    .row { max-width: 90%; }
    .row:not(:last-child) { margin-bottom: 6rem; }
    [class^="col-"]:not(:last-child) { margin-right: 3rem; }
    .col-1-of-3 { width: calc((100% - 2 * 3rem) / 3); }
    .col-1-of-4 { width: calc((100% - 3 * 3rem) / 4); }
}

/* Mobil - 600px dan kichik */
@media (max-width: 37.5em) {
    .row { max-width: 95%; }
    [class^="col-"] { width: 100% !important; margin-right: 0 !important; margin-bottom: 4rem; }
    .row:not(:last-child) { margin-bottom: 0; }
}

Amaliy mashq

Vazifa: O'z loyihangizni mobile-first yondashuvda responsive qiling.

  1. Breakpointlar uchun CSS o'zgaruvchilari yarating
  2. Header va navigatsiyani responsive qiling
  3. Grid tizimini turli ekranlarga moslang
  4. Barcha rasmlarga max-width: 100% bering

Namunaviy yechim:

:root { --bp-mobile: 37.5em; --bp-tablet: 56.25em; }
img { max-width: 100%; height: auto; }

/* Mobile-first - default */
.container { width: 100%; padding: 1.5rem; }
.grid { display: block; }
.col { width: 100%; margin-bottom: 2rem; }

/* Planshet */
@media (min-width: 37.5em) {
    .container { width: 90%; padding: 2rem; }
    .grid { display: grid; grid-template-columns: repeat(2, 1fr); gap: 2rem; }
    .col { width: auto; margin-bottom: 0; }
}

/* Desktop */
@media (min-width: 56.25em) {
    .container { width: 85%; max-width: 120rem; }
    .grid { grid-template-columns: repeat(3, 1fr); }
}

  • Grid responsive: mobil - 1 ustun, planshet - 2 ustun, desktop - 3+ ustun
  • Rasmlar: max-width: 100%; height: auto;
