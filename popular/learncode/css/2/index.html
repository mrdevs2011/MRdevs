CSS versiyalari tarixi

CSS vaqt o'tishi bilan rivojlanib bordi. Har bir yangi versiya yangi imkoniyatlar olib keldi. Keling, asosiy versiyalar bilan tanishamiz:

1996

CSS1

Birinchi versiya. Asosiy xususiyatlar: ranglar, shriftlar, margin, padding, border, background. Juda oddiy imkoniyatlar.

Asosiy
1998

CSS2

Positioning (absolute, relative), z-index, media types, pseudo-elementlar qo'shildi. Sahifalarni joylashtirish imkoniyatlari kengaydi.

Kengaytirilgan
2011+

CSS3

Modullarga bo'lindi. Flexbox, Grid, Animatsiyalar, Gradientlar, Media Queries, Transforms va boshqa ko'plab zamonaviy imkoniyatlar.

Zamonaviy
Muhim: CSS3 "modulli" yondashuvni qo'llaydi. Ya'ni, har bir yangi xususiyat (flexbox, grid, animations) alohida modul sifatida ishlab chiqiladi. Bu tezroq yangilanish va brauzerlar tomonidan bosqichma-bosqich qo'llab-quvvatlash imkonini beradi.

Brauzer muvofiqligi va caniuse.com

Har bir brauzer (Chrome, Firefox, Safari, Edge) CSS xususiyatlarini turlicha qo'llab-quvvatlaydi. Yangi CSS xususiyatlari eski brauzerlarda ishlamasligi mumkin. Shuning uchun professional dasturchilar caniuse.com saytidan foydalanadi.

Kurs loyihasi

Bu kurs davomida biz "Natour" (tabiat sayohatlari uchun sayt) loyihasini yaratamiz. Bu zamonaviy, to'liq ishlaydigan web-sayt bo'lib, quyidagi elementlarni o'z ichiga oladi:

Header va navigatsiya menyusi
Hero section (katta rasm va sarlavha)
Kartalar (cards) bilan ma'lumotlar
Formalar (aloqa, ro'yxatdan o'tish)
Animatsiyalar va hover effektlar
Responsive dizayn (mobil, planshet, desktop)

Eslatma: Kurs davomida har bir yangi mavzuni o'rganganimizdan so'ng, loyihamizga yangi stillar qo'shib boramiz. Masalan, Box Model mavzusidan keyin margin va padding larni, Positioning mavzusidan keyin elementlarni joylashtirishni qo'shamiz.

Bilimingizni tekshiring

Amaliy mashq

Vazifa: O'zingizning loyihangiz uchun papka tuzilmasini yarating va caniuse.com saytida 3 ta CSS xususiyatini tekshiring.

  1. Kompyuteringizda "mening-saytim" nomli papka yarating
  2. Ichida index.html, css/style.css va img/ papkasini yarating
  3. index.html fayliga asosiy HTML strukturani yozing
  4. caniuse.com saytiga kirib, quyidagi xususiyatlarni tekshiring: flexbox, grid, border-radius
  5. Har bir xususiyat uchun qaysi brauzer versiyalarida ishlashini yozib oling

Namunaviy natijalar:

1. Papka tuzilmasi:

mening-saytim/
├── index.html
├── css/
│   └── style.css
└── img/

2. index.html namunasi:

<!DOCTYPE html>
<html lang="uz">
<head>
    <meta charset="UTF-8">
    <title>Mening Saytim</title>
    <link rel="stylesheet" href="css/style.css">
</head>
<body>
    <h1>Salom Dunyo!</h1>
</body>
</html>

3. caniuse.com tekshiruvi natijalari:

  • flexbox: Barcha zamonaviy brauzerlarda to'liq ishlaydi (Chrome 29+, Firefox 28+, Safari 9+, Edge 12+)
  • grid: Barcha zamonaviy brauzerlarda to'liq ishlaydi (Chrome 57+, Firefox 52+, Safari 10.1+, Edge 16+)
  • border-radius: Juda eski brauzerlardan tashqari hammasida ishlaydi (Chrome 4+, Firefox 4+, Safari 5+, Edge 12+)
