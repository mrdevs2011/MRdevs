Inheritance (Meros olish)

14-yanvar, 2021 CSS Asoslari O'rta daraja
Bu darsda nimalarni o'rganasiz?

CSS da inheritance (meros olish) - ota elementga berilgan ba'zi xususiyatlarning avtomatik ravishda bola elementlarga o'tish xususiyati. Bu kodni qisqartirish va samarali yozish uchun juda muhim.

Bu darsda nimalarni o'rganasiz?

  • Inheritance nima va u qanday ishlaydi?
  • Qaysi xususiyatlar meros bo'lib o'tadi?
  • Qaysi xususiyatlar meros bo'lib o'tmaydi?
  • inherit, initial, unset kalit so'zlari
  • Inheritance dan samarali foydalanish

Bu dars uchun maslahatlar

Matn xususiyatlari meros bo'ladi: color, font-family, font-size, text-align kabi xususiyatlar avtomatik ravishda bolalarga o'tadi. yodlash shart
Quti modeli meros bo'lmaydi: margin, padding, border, width, height kabi xususiyatlar meros bo'lib o'tmaydi. muhim
body ga shrift bering: Butun sayt bo'ylab bir xil shrift ishlatish uchun body elementiga font-family bering. amaliyot

Inheritance nima?

Inheritance (meros olish) - bu ota elementga berilgan ba'zi CSS xususiyatlarining avtomatik ravishda uning bola (ichki) elementlariga o'tish xususiyati.

CSS 
/* Ota elementga berilgan rang */
.parent {
    color: #2563eb;
    font-family: 'Segoe UI', sans-serif;
}
HTML 
<div class="parent">
    Bu matn ko'k rangda
    <p>Bu paragraf ham ko'k rangda (meros oldi)</p>
    <span>Bu span ham ko'k rangda (meros oldi)</span>
</div>
Natija: parent elementiga berilgan color xususiyati barcha ichki elementlarga avtomatik o'tadi.

Meros bo'ladigan xususiyatlar

Asosan matn bilan bog'liq xususiyatlar meros bo'lib o'tadi:

color

Matn rangi

font-family

Shrift turi

font-size

Shrift o'lchami

font-weight

Shrift qalinligi

text-align

Matnni tekislash

line-height

Qator balandligi

cursor

Kursor ko'rinishi

visibility

Ko'rinuvchanlik

Meros bo'lmaydigan xususiyatlar

Quti modeli va joylashuv bilan bog'liq xususiyatlar meros bo'lib o'tmaydi:

margin

Tashqi bo'shliq

padding

Ichki bo'shliq

border

Chegara

width

Kenglik

height

Balandlik

background

Fon

position

Joylashuv

display

Ko'rinish turi

Eslatma: background xususiyati meros bo'lmaydi, lekin ota elementning foni shaffof bo'lsa, bola elementning foni ko'rinib turadi. Bu inheritance emas, shaffoflik xususiyati.

Inheritance ni boshqarish

CSS da inheritance ni boshqarish uchun maxsus kalit so'zlar mavjud:

Kalit so'z Vazifasi Misol
inherit Ota elementdan majburan meros olish border: inherit;
initial Brauzerning standart qiymatiga qaytarish color: initial;
unset Agar meros bo'lsa inherit, aks holda initial font-size: unset;
revert Brauzer standartiga qaytarish (user styles ni hisobga oladi) display: revert;
CSS 
/* Meros bo'lmaydigan xususiyatni majburan meros qilish */
.child {
    border: inherit;  /* Ota elementning borderini oladi */
}

/* Meros bo'ladigan xususiyatni meros qilmaslik */
.child {
    color: initial;  /* Brauzer standart rangiga qaytadi */
}

Body orqali global stillash

Inheritance dan eng samarali foydalanish usullaridan biri - body elementiga umumiy stillar berish:

CSS 
/* Butun sayt bo'ylab bir xil shrift va rang */
body {
    font-family: 'Inter', sans-serif;
    color: #1e293b;
    line-height: 1.6;
}

/* Kerakli joyda qayta yozish mumkin */
.special-text {
    color: #3b82f6;  /* Faqat shu element uchun */
}
Professional maslahat: Har doim body elementiga asosiy shrift va matn rangini bering. Bu kodni ancha qisqartiradi va sayt bo'ylab bir xillikni ta'minlaydi.

Bilimingizni tekshiring

7-dars testi
1 Qaysi xususiyat meros bo'lib o'tadi?
2 Qaysi kalit so'z ota elementdan majburan meros olish uchun ishlatiladi?
3 Butun sayt bo'ylab bir xil shrift ishlatish uchun qaysi elementga font-family beriladi?
4 Qaysi xususiyat meros bo'lib o'tmaydi?

Amaliy mashq

Vazifa: Inheritance dan foydalanib, quyidagi vazifalarni bajaring.

  1. body elementiga font-family: 'Segoe UI', sans-serif; va color: #1e293b; bering
  2. body ichida div yarating va unga padding: 20px; bering
  3. div ichida h1, p, va span elementlarini yarating
  4. span elementiga color: #3b82f6; bering (merosni bekor qilish)
  5. div elementiga border: 2px solid #e2e8f0; bering va uning ichidagi p elementiga border: inherit; bering

Namunaviy yechim:

<!DOCTYPE html>
<html>
<head>
    <style>
        body {
            font-family: 'Segoe UI', sans-serif;
            color: #1e293b;
        }
        
        div {
            padding: 20px;
            border: 2px solid #e2e8f0;
        }
        
        span {
            color: #3b82f6;
        }
        
        p {
            border: inherit;
        }
    </style>
</head>
<body>
    <div>
        <h1>Sarlavha (meros rangda)</h1>
        <p>Paragraf (border inherit qilgan)</p>
        <span>Span (o'ziga xos ko'k rangda)</span>
    </div>
</body>
</html>

  • Kalit so'zlar: inherit (meros ol), initial (standartga qaytar), unset (avtomatik)
